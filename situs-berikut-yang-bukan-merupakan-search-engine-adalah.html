4. Search engine yang menggunakan spider software dan berfungsi menyusup pada situs-situs tertentu, kemudian mengumpulkan data serta mengelompokkan dengan sedikit bantuan tangan manusia adalah A. Hybrid search engineB. Human organized search engineC. MetacrawlerD. Computer created search engineE. Metasearch​


Search engine yang berfungsi untuk menyusup pada situs-situs tertentu dan mengumpulkan data serta mengelompokkannya dengan menggunakan spider softare dan bantuan tangan manusia adalah (D) Computer created search engine.

Pembahasan

Search engine merupakan sebuah fitur pada perangkat komputer dalam bentuk perangkat lunak yang bertujuan untuk mempermudah user atau pengguna untuk menemukan solusi untuk sebuah kata kunci yang mereka gunakan. Metacrawler merupakan sebuah mesin pencari yang dikembangkan oleh infospace. Sedangkan metasearch merupakan Sebuah mesin pencari meta yang berfungsi sebagai alat pencarian informasi online dengan menggunakan data dari mesin pencari web untuk menghasilkan hasil dengan kriteria sendiri. Hibrid search engine merupakan kombinasi antar jenis crawlerbased search engine dengan penyimpanannya.
