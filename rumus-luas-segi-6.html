Luas = p x l
= (4x + 3)(6x - 4)
= 24x² - 16x + 18x -12
= 24x² + 2x - 12luas segi empat
p × le

p = (4x + 3)
le = (6x - 4)

Luas
= (4x + 3) (6x - 4)
= (4x . 6x) + (4x . (-4)) + (3 . 6x) + (3 . (-4))
= 24x² + (-16x) + 18x - 12
= 24x² + (-16 + 18)x - 12

Luas Persegi Panjang

L = p x l

Keliling Persegi Panjang

K = 2 x p + 2 x l = 2 x ( p + L )

Keterangan : p = panjang, l = lebar

2. Rumus Segitiga

Luas Segitiga

L = ½ x a x t

Keliling segitiga

Luas Jajar Genjang

L = a x t

Keliling Jajar Genjang

K = Sisi + Sisi + Sisi + Sisi = 2 ( p x q )

Keterangan : a = alas, t = tinggi

3. Rumus Trapesium

Luas Trapesium

L = ½ x ( s1 + s2 ) x t

Keliling Trapesium

K = sisi + sisi + sisi + sisi

Keterangan : s1 & s2 = sisi-sisi sejajar, t = tinggi

Rumus Layang-layang

Luas Layang-Layang

L = ½ x d1 x d2

Keliling Layang-Layang

K = sisi + sisi + sisi + sisi

Keterangan : d1 = diagonal 1, d2 = diagonal 2

1. PERSEGI PANJANG /rumus/

Lebar - Pada sisi yang lebih panjang disebut panjang (p), dan pada sisi pendeknya disebut dengan lebar.

= l = L ÷ p

l = L ÷ pl = (K ÷ 2) – p

Luas - Luas persegi panjang adalah panjang dikalikan lebar. Rumus luas persegi panjang adalah L = p × l dengan L = luas persegi panjang, p = panjang persegi panjang, dan l = lebar persegi panjang.

= p x l = panjang kali lebar

2. KELILING SEGITIGA /rumus/

rumus s + s + s merupakan rumus keliling segitiga sama kaki.

= s + s + s = sisi tambah sisi tambah sisi

3. LUAS SEGITIGA /rumus/

Ada dua rumus yang bisa digunakan, yaitu rumus luas (L) dan rumus keliling (kll). Rumus luas segitiga sama kaki adalah L = 1/2 a x t.

= L = 1/2 a x t = Luas Segitiga 1/2 Alas kali Tinggi

4. KELILING PERSEGI PANJANG /rumus/

Keliling persegi panjang adalah jumlah semua sisi persegi panjang. Rumus keliling persegi panjang adalah K = 2 (p + l) dengan K = keliling persegi panjang, p = panjang persegi panjang, dan l = lebar.

= K = 2 p + l = keliling persegi panjang p - panjang l - lebar.

5. KELILING PERSEGI /rumus/

Rumus kelilingnya adalah: K = s + s + s + s.

= 4 X dihitung sisi + sisi + sisi + sisi.

6. LUAS PERSEGI /rumus/

Rumus luas persegi adalah L = s x s. L adalah luas dan s merupakan sisi.

= L = sisi dikali sisi l - luas

29. 2. Berdasarkan rumus luas segitiga, temukanlah rumus luas segi 6 beraturan!


Rumus luas segi enam beraturan berdasarkan rumus luas segitiga adalah [tex]\frac{3}{2}a^{2}\sqrt{3}[/tex]

Pembahasan

Seperti kita ketahui segienam beraturan itu terdiri dari 6 buah segitiga sama sisi. Lebih jelasnya, bisa memperhatikan gambar ilustrasi terlampir. Berdasarkan gambar ilustrasi, rumus luas segitiga sama sisi, apabila menggunakan aturan sinus menjadi:

L = [tex]\frac{1}{2}[/tex] x a x a x sin 60° ⇒ Mengapa 60°? karena segitiga sama sisi pasti setiap sudutnya besarnya sama.

L = [tex]\frac{1}{2}[/tex] x a x a x [tex]\frac{1}{2} \sqrt{3}[/tex]

L = [tex]\frac{1}{4}a^{2}\sqrt{3}[/tex]

Pembuktian lain apabila kita menggunakan rumus luas segitiga :

L = [tex]\sqrt{s(s-a)(s-b)(s-c)}[/tex]

dimana s = 1/2 K = 1/2 x (a + a + a) = 1/2 x 3a = 3/2 a

L = [tex]\sqrt{\frac{3}{2}a(\frac{3}{2}a-a)(\frac{3}{2}a-a)(\frac{3}{2}a-a)}[/tex]

L = [tex]\sqrt{\frac{3}{2}a(\frac{1}{2}a)(\frac{1}{2}a)(\frac{1}{2}a)}[/tex]

L = [tex]\sqrt{\frac{3}{16}a^4}[/tex]

L = [tex]\frac{1}{4}a^2\sqrt{3}[/tex]

Seperti dijelaskan sebelumnya, berarti luas segienam beraturan adalah 6 x luas segitiga sama sisi, maka rumusnya menjadi

L = 6 x [tex]\frac{1}{4}a^{2}\sqrt{3}[/tex]

L = [tex]\frac{3}{2}a^{2}\sqrt{3}[/tex]

Keterangan :

a = sisi segitiga
