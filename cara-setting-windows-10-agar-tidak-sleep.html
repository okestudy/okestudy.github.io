1. Cara merubah settingan windows shortcut pada desktop di Wd 10​


Windows Key + Arrow kiri – untuk meletakkan windows di kiri layar

Windows Key + Arrow kanan – untuk meletakkan windows di kanan layar.

Windows Key + Arrow atas – untuk meletakkan windows di bagian atas layar.

Windows Key + Arrow bawah – untuk meletakkan windows di bawah layar.

Hal ini sebenarnya juga bisa Anda lakukan dengan mouse, Anda tinggal menggeser window yang ingin Anda letakkan ke salah satu sisi, maka window akan menyesuaikan ukuran sesuai dengan Anda meletakkanya. Namun, dengan menggunakan shortcut, hal itu akan terasa lebih mudah.

7. Cara mendapatkan ip address static di windows kita harus melakukan setting secara


Jawaban:

Buka Control Panel.Navigasi ke Network and Internet > Network and Sharing Center.Pada panel kiri, klik opsi Change adapter settings.Pada network adapter klik kanan lalu pilih Properties.Klik 2x pada opsi Internet Protocol Version 4 (TCP/IPv4).Klik opsi Use the following IP address dan masukan IP address sesuai kasus yang kamu hadapi (misalnya 192.168.1.7)Subnet mask (biasanya untuk pengguna koneksi lokal adalah 255.255.255.0.Default gateway yang biasanya merupakan IP address yang digunakan pada router. (misalnya 192.168.1.1)Pada menu DNS server, klik opsi Use the following DNS server addresses dan masukan alamat Preferred DNS server, dimana biasanya merupakan IP address yang sama dengan Default gateway.Alternate DNS server dapat kamu isi sebagai opsi lain jika kamu tidak dapat terkoneksi dengan DNS server utama, contohnya bisa kamu masukan DNS google (8.8.8.8). Namun hal ini bersifat opsional (yakni bisa kamu isi, bisa juga tidak).Klik OK untuk menyimpan perubahan.

14. apa yang dimaksud dengan setting peripheral pada sistem operasi windows dan bagian caranya​


Jawaban:

Pengaturan(setting) periferal adalah pengaturan suatu hardware ke dalam komputer. Setting Peripheral pada OS Peripheral adalah perangkat keras tambahan yang mendukung kerja komputer dan berada diluar.

Penjelasan:

1. Mengatur Setting Monitor

Caranya :

a. Klik Display pada Control Panel

b. Selanjutnya akan ditampilkan kotak Dialog Display Properties

Lakukan Pengaturan :

1) Themes : Mengatur tema windows yang akan digunakan

2) Desktop : Memberi background gambar pada layar monitor/ desktop

3) Screen Saver : Mengatur Screen Saver

4) Appearance : Pengaturan jendela dan tombol (Windows and Button), Skema warna (Color Schemes), dan Ukuran huruf (Font Size)

5) Setting : Mengatur resolusi monitor yang akan digunakan

2. Mengatur Setting Printer

Caranya :

a. Buka Control Panel

b. Klik 2 kali pada Icon printer and faxes

c. klik kanan pada printer yang anak diatur - klik printing preferences

d. Selanjutnya akan ditampilkan dialog printing preferences

Keterangan :

a. Layout : mengatur orientasi kertas (Paper Orientation)

b. Paper: menentukan jumlah cetakan tiap halaman (Copies), jenis kertas (Paper option-size), dan skala cetakan (Scaling Printing)

c. Graphic : Mengatur Resolusi (Resolution) dan gelap/terang pencetakan (Darkness)

3. Mengatur Setting Mouse

Caranya :

a. Buka control panel

b. klik 2 kali pada Icon Mouse

c. selanjutnya akan tampil jendela mouse properties

Keterangan :

a. Buttons : Mengatur konfigurasi tombol mouse (Button configuration), Double-click speed : mengatur kecepatan double click mouse

b. Pointer : Mengatur pola pointer dan jenis pointer

c. Pointer option : Mengatur kecepatan gerakan pointer mouse setelah memilih instruksi pada kotak dialog

d. Wheel : Mengatur gulungan mouse

e. Hardware : Mengatur mouse yang digunakan

4. Mengatur Efek Suara

Caranya :

a. Buka control panel

b. Klik 2 kali pada Icon Sound and Audio Device

c. Selanjutnya akan ditampilkan kotak dialog sound and audio properties

d. Lakukan pengaturan :

1) Volume : Mengatur volume (klik Advance untuk mengatur master volume), mute (Menyembunyikan suara), Place Volume icon in the taskbar (menampilkan icon volume pada taskbar)

2) Sound : mengatur skema suara

3) Audio : Mengatur sumber suara

4) Voice : Mengatur Sumber suara

5) Hardware : Memilih jenis Soundcard yang digunakan
