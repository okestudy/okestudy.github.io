Soal matematika di atas merupakan materi dari turunan/diferensial dari persamaan kuadrat. Turunan atau diferensial adalah perhitungan dengan menurunkan pangkat suatu variabel pada fungsi bervariabel tertentu.

Pembahasan

Perhitungan secara diferensial/turunan adalah dengan menurunkan pangkat suatu variabel sebanyak 1 atau x^(n-1) kemudian dikalikan dengan nilai pangkat sebelum diturunkan sehingga menjadi nx^(n-1).  Fungsi diferensialnya dinotasikan f(x)' atau df(x)/dx. Sebagai contoh terdapat fungsi persamaan kuadrat yaitu f(x) = ax^2+bx+c.

[tex]f(x) = ax^2+bx+c[/tex]

[tex]f'(x) = 2ax^{1}+1bx^{0}\\f'(x) = 2ax+b\\{d[f(x)]}/{dx} = 2ax+b[/tex]

Perhitungan secara turunan berlaku sifat komutatif dan sifat asosiatif.

h(x) = f(x) + g(x) atau h(x) = g(x) + h(x)

h'(x) = f'(x) + g'(x) atau h'(x) = g'(x) + f'(x)

d[h(x)]/dx = d[f(x)]/dx + d[g(x)]/dx atau d[h(x)]/dx = d[g(x)]/dx + d[f(x)]/dx

Penyelesaian soal-soal turunan/diferensial persamaan kuadrat.

1. Diketahui:

Fungsi produksi: Q = 150 + 10Qx² − 2 /3 x²

Q adalah output dan X adalah input, maka Q=f(X) atau fungsi dari X.

Ditanya:

Nilai X jika nilai fungsi Q bernilai maksimum?Nilai X jika nilai fungsi Q sebesar dua kali nilai X (Q=2X).

Jawab:

a. Fungsi produksi belum sempurna sehingga harus disederhanakan.

Q = 150 + 10Qx² − 2 /3 x²

Q - 10Qx² = 150 − 2 /3 x²

Q(1 - 10x²) = 150 − 2 /3 x²

Q = (150 − 2 /3 x²)/(1 - 10x²)

Misalnya u=150 - 2 /3 x² dan v=1 - 10x², maka nilai turunannya masing-masing adalah u'=-4/3x dan v'=-20x. Turunan untuk Q dalam bentuk pecahan yaitu dengan rumus seperti berikut.

[tex]Q'=\frac{U'V-V'U}{V^2}\\Q' = \frac{(-4/3x)(1 - 10x^2)-(-20x)(150 -\frac{2}{3}x^2)}{(1 - 10x^2)^2}[/tex]

Nilai Q maksimum jika Q'=0

[tex]0 = \frac{(-\frac{4}{3}x)(1 - 10x^2)-(-20x)(150 -\frac{2}{3}x^2)}{(1 - 10x^2)^2}\\0 = (-\frac{4}{3}x)(1 - 10x^2)-(-20x)(150 -\frac{2}{3}x^2)\\0=-\frac{4}{3}x+\frac{40}{3}x^2 + 3000x -\frac{40}{3}x^3\\0=-\frac{4}{3}x^2-3000x\\0=(x)(-\frac{4}{3}x-3000)\\\frac{4}{3}x=3000\\x=2250[/tex]

Maka nilai X jika nilai Q maksimum adalah 2250 atau x=2250.

Q = 150 + 10Qx² − 2 /3 x²

2x = 150 + 10(2x)(x²) − 2 /3 x²

0 = 150 − 2x − 2 /3 x²+ 20x³

Nilai Q maksimum jika Q'=0, maka dicari persamaan untuk turunan pertama

0 = 150 − 2x − 2 /3 x²+ 20x³

0 = −2 − 4 /3 x+ 60x²

0 = −6−4  x+180x²

Nilai Q' maksimum jika Q''=0, maka dicari persamaan untuk turunan kedua

0 = −6−4  x+180x²

0 = −4+360x

0 = −4+360x

4 = 360x

x = 1/90

Soal matematika di atas merupakan materi dari turunan/diferensial dari persamaan kuadrat. Turunan atau diferensial adalah perhitungan dengan menurunkan pangkat suatu variabel pada fungsi bervariabel tertentu.

Pembahasan

Perhitungan secara diferensial/turunan adalah dengan menurunkan pangkat suatu variabel sebanyak 1 atau x^(n-1) kemudian dikalikan dengan nilai pangkat sebelum diturunkan sehingga menjadi nx^(n-1).  Fungsi diferensialnya dinotasikan f(x)' atau df(x)/dx. Sebagai contoh terdapat fungsi persamaan kuadrat yaitu f(x) = ax^2+bx+c.

[tex]f(x) = ax^2+bx+c[/tex]

[tex]f'(x) = 2ax^{1}+1bx^{0}\\f'(x) = 2ax+b\\{d[f(x)]}/{dx} = 2ax+b[/tex]

Perhitungan secara turunan berlaku sifat komutatif dan sifat asosiatif.

[tex]h(x) = f(x) + g(x) atau h(x) = g(x) + h(x) [/tex]

[tex]h'(x) = f'(x) + g'(x) atau h'(x) = g'(x) + f'(x) [/tex]

[tex]d[h(x)]/dx = d[f(x)]/dx + d[g(x)]/dx atau d[h(x)]/dx = d[g(x)]/dx + d[f(x)]/dx [/tex]
