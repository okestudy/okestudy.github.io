1. apa rumus kerangka prisma segitiga??


BANGUN RUANG PRISMA SEGITIGA

Seperti kita ketahui bahwa prisma segitiga memiliki 9 rusuk yaitu :

3 rusuk pada bidang alas

3 rusuk pada bidang tegak

3 rusuk pada bidang atas


Maka rumus untuk mencari panjang kerangka prisma segitiga adalah

= keliling alas prisma + keliling atas prisma + 3 x tinggi prisma segitiga

= 2 x (keliling alas prisma) + 3 x tinggi prisma segitiga

= 2 x (jumlah sisi sisi segitiga pada bidang alas) + 3 x tinggi prisma segitiga


2. rumus kerangka prisma segitiga

3. rumus kerangka prisma segitiga

n = prisma segi-n
t = tinggi prisma
