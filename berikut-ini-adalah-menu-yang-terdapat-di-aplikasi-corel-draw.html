3. jelaskan fungsi menu yang terdapat dalam aplikasi corel draw X5


Pick tool. Tool ini berfungsi untuk merubah ukuran sesuai yang kita inginkan dan juga untuk menyeleksi gambar pada Corel draw.

Shape tool. Mempunyai fungsi yaitu digunakan untuk mengubah suatu bentuk objek yang terdapat di halaman kerja Corel draw. Selain itu, shape tool juga mempunya beberapa sup tool lagi yang berjumlah 4 tool. Yang mana 4 sub tool ini saling berhubangan satu dengan yang lainnya. 4 tool itu adalah shape, Smudge Brush, Free transform dan Roughen Brush. Adapun jalan pintasnya atau sering di sebut dengan tombol shortcut, kita cukup menekan tombol F10, mudah kan.

Crop tool merupakan tool yang dapat kita gunakan untuk memotong objek yang tidak kita butuhkan. Tool ini juga sama seperti shape tool, sama – sama mempunyai sub tool berjumlah 4 yaitu Crop, Eraser, Knife dan Virtual Segment Delete. Yang mana sub tool ini saling berhubungan.

Zoom tool. Tool ini berfungsi untuk memperbesar dan memperkecil objek atau halaman kerja Corel draw. Tool i9ni juga mempunyai 2 sub tool yaitu Zoom atau dengan menekan Z dan Pan cukup menekan H.

Freehand tool merupakan tool yang berfungsi untuk membuat berbagai macam garis. Di dalam Freehand tool juga terdapat berbagai tool antara lain : Freehand, Bezier, 2-point line, Pen, Artistic Media, Polyne, B-spline dan 3-point curve. Dapat juga kita menekan F5 dan untuk Artistic Media kita menekan tombol I.

Smart Fill Tool yaitu tool yang dapat kita gunakan untuk membuat dan mewarnai objek. Mempunyai 2 sub tool : Smart Fill dan Smart Drawing. Cukup menekan Shif + S.

Rectangle tool, untuk membuat berbagai bentuk persegi. Terdapat 2 sub tool yaitu Rectangle dan 3-point Rectangle, cukup menekan F6.

Ellipse tool, dengan tool ini kita bisa membuat berbagai lingkaran, juga mempunyai 2 tool yang tersembunyi antara lain : Ellipse dan 3-point Ellipse, agar mudah cukup tekan tombol F7.

Polygon tool, tool ini dapat membantu kita agar kita bisa membuat macam – macam bentuk objek pada halaman kerja Corel draw. Di dalam tool ini mempunyai tool- tool yang tidak terlihat, yaitu : Polygon, Star, Graph Paper, complex Star dan spiral. Juga mempunyai shortcut tombol Y untuk polygon, tombol D untuk Graph Paper dan tombol A untuk spiral.

Basic shapes tool, yaitu tool yang berguna untuk membuat objek dasar pada Corel draw. Di sini tersembunyi 5 macam tool yang saling berhubungan yaitu : Basic Shapes, Flowchart Shapes, Arrow Shapes, Callout Shapes dan Banner shapes. Tool – tool ini tidak mempunyai shortcut.

Text tool, dengan tool ini kita dapat menyelipkan dek yang kita inginkan, agar mudah gunakan tombol F8.

Table tool, dengan adanya tool ini kita bisa menambahkan tabel di halaman kerja Corel draw kita.

Parallel Dimension Tool, di sini kita dapat membuat objek dimensi. Mempunyai sup tool 5 : Parallel Dimension, Angular Dimension, 3-point Callot, vertcal / horizontal Dimension dan Segment Dimension.

Straight-Line Connector Tool, berfungsi untuk menghubungkan 2 objek dengan menciptakan garis. 4 macam tool yang terdapat dalam tool Straight-Line Connector Tool : Rigt-Angle Connector, Edit Anchor, Straight-Line Connector dan Right-Angle Rounded Connector.

Blend Tool, di sini kita bisa mengkolaborasikan 2 atau banyak objek. Terdapat juga sub tool yang lainnya : Blend, Contour, Drop Shadow, Extrude, Distort, Envelope dan Transparency.

Eyedropper Tool, adanya tool Eyedropper kita bisa memberi warna pada objek. Dan juga terselip 2 tool : Attributes Eyedropper dan Color Eyedropper.

Outline Tool, dengan ini kita bisa menciptakan garis tepi pada Corel draw, banyak sekali macam outline : Outline Pen, No Outline, Outline Color dan Hairline Outline. Outline juga mempunyai banyak ukuran dan warna, ukurannya antara lain : 8 pt, 4 pt, 3 pt, 2 pt, 1.5 pt, 1 pt, 0.75 pt, dan 0.5 pt. F12 untuk Outline Pen, Shift + F12 untuk Outline Colo.

Fill tool, berbagai metode cara memberi warna terdapat dalam film tool ini. 6 sub tool yang tersembunyi pada film tool yaitu : Uniform Fill, Pattern Fill, No Fill, Fountain Fill, PostScriot Fill serta Color. Untuk Uniform Fill dapat menekan tombol Shift + F11, sedangkan F11 untuk Fountain Fill.

Inteactive Fill tool, tool yang ini mempunyai fungsi untuk mengkaloborasikan 2 warna atau banyak warna terhadap objek. Inteactive Fill tool juga tidak mau kalah dengan yang lainnya, tool ini juga mempunyai 2 tool yang terselip di dalamnya : Mesh FIll (tekan tombol M) dan Interactive Fill (tekan tombol G)

15. Derskripsikam menu-menu yang terdapat pada tampilan welcome screen corel draw x3


CorelDraw adalah sebuah editor grafis vektor yang dikembangkan dan dipasarkan oleh Corel Corporation. CorelDraw di desain untuk melakukan pengeditan pada gambar dua dimensi seperti logo dan poster.  

Menu-menu yang terdapat pada tampilan Welcome Screen CorelDraw X3 adalah sebagai berikut

- New. Digunakan untuk membuat lembar kerja atau dokumen baru yang akan dikerjakan

- Recently Used. Digunakan untuk membuka kembali lembar kerja yang baru dikerjakan. Dengan menggunakan menu ini, user dapat langsung masuk ke lembar kerja yang diinginkan

- Open. Berfungsi untuk membuka direktori lembar kerja sehingga user dapat memilih lembar kerja yang telah dibuat sebelumnya

- New From Template. Digunakan untuk membuat lembar kerja berdasarkan template bawaan dari CorelDraw

- Corel Tutor. Digunakan sebagai panduan pemakaian CorelDraw

- What's New. Digunakan untuk mengetahui fitur-fitur baru yang terdapat pada CorelDraw

fungsi menu teks

Text Properties berfungsi untuk menampkan text atau menghiden text.

Tabs digunankan untuk memberikan tab pada text yang kita buat.

Columns berfungsi untuk memberikan kolom di lembar kerja CorelDraw.

Bullet digunakan sebagai pemberi petunjuk atau tanda seperti lingkaran kecil berwarna black.

Drop Cap berfungsi untuk merubah huruf awal dari huruf kecil menjadi huruf kapital.

Font Palyground berfungsi untuk menyeleksi suatu font yang akan digunakan dengan cara menyatukan atau mengumpulkan banyak font, kemudian memilih font mana yang akan digunakan.

Edit Text berfungsi untuk merubah tulisan pada text.

Insert Caracter berfungsi untuk memasukan karakter pada suatu text.

Insert Formating Code berfungsi untuk mamasukan code pada text.

Convert dapat digunakan untuk mengganti text menjadi paragraf dan mengganti text menjadi artistik.

Show Non-Printing Characters berfungsi untuk mengemukakan karakter menjadi non print.

Paragraph Taxt Frame digunakan untuk memilih kolom pada sebuah fram text.

Fit Text To Path berfungsi untuk mengatur text dalam bidang lurus dirubah menjadi melengkung dengan menggunakan bangun lingkarang atau dengan bangun lain sebagainya.

Straighten Text fungsi yang digunakan untuk mengatur text.

Align to Baseline berfungsi untuk mengatur ulang dari font menjadi bentuk aslinya.

Align to Baseline Grid berfungsi untuk mengatur ulang font pada grid di lembar kerja CoralDraw.

Use Hyphenation berfungsi untuk mengatur panjang suatu kata pada kalimat dan jarak  pada line pinggir.

Hyphenation Settings berfungsi untuk menentukan panjang suatu kata dan jarak antar garis pinggir secara tidak otomatis.

Writing Tools berfungsi untuk mengecek text yang telah dibuat pada lembar kerja CorelDraw.

Change Case untuk mengatur kata menjadi huruf kapital , kecil, huruf besar kecil, dan huruf besar pada  huruf vokal.

Make Taxt Web-compatible berfungsi untuk membuat text mejadi dapat bersaing  pada Web.

Encode Berfungsi untuk membuat kata sandi pada objek.

What the Font berfungsi untuk mencari font yang langsung terhubung dengan internet pada web CorelDraw.

Text Statistics berfungsi untuk melihat text yang dipilih.

Font List Option berfungsi sebagai penyusun urutan font secara  tidak otomatis.

fungsi menu help:

Poduct Help digunakan sebagai petunjuk penggunaan Aplikasi CorelDraw.

Welcome Screen berfungsi untuk menampilkan awal lembar kerja sebelum membuka lembar kerja yang baru, di sana terdapat beberapa daftar nama file yang pernah kita buat.

Video Tutorials berfungi untuk menampilkan Video Tutorial sebagai media pemblajaran ,bagimana cara penggunakan tool serta teknik mendesign.

Hints berfungsi untuk memunculkan fungsi dan kegunakan dari masing-masing tool.

Quick Start Guide berfungsi untuk menampilkan langkah penggunaan secara cepat.

Insights Form the Eperts berfungsi sebagai media pembelajaran berformat PDF.

Whats' new berfungsi sebagai pengirim pemberitahuan yang berkaitan dengan terupdate pada CorelDraw.

Highlight Whats New fungsinya hampir sama dengan fungsi pada what's new, serta dapat memberi informasi pada versi yang lama.  

Updates berfungsi sebagai media untuk mengupdate ke versi yang lebih terbaru dari versi yang sebelumnya.

CorelDraw.com berfungsi untuk bertemu dengan  komunitas CorelDraw lainya yang disediakan pada link yang ada pada CorelDraw.

Corel Support diguankan untuk memberikan informasi menyeluruh tentang CorelDraw.

About CorelDraw MemberShip berfungsi langkah menjadi member secara gratis atau berbayar.  

Account Settings berfungsi sebagai tempat untuk masuk ke account CorelDraw atau membuat Account pada CorelDraw.

Sing In fungsinya sama dengan account setting.

About CorelDraw digunakan untuk mengetahui tentang seputar CorelDraw.
