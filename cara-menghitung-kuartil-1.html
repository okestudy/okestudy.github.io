1. cara menghitung data kuartil


Kuartil bawah adalah data pada posisi 1/4 dari kumpulan data yang telah diurutkan. Kuartil bawah disimbolkan dengan Q1.
Kuartil tengah adalah data pada posisi 2/4 dari kumpulan data yang telah diurutkan. Kuartil tengah sama dengan median. Kuartil tengah disimbolkan dengan Q2.
Kuartil atas adalah data pada posisi 3/4 dari kumpulan data yang telah diurutkan. Kuartil atas disimbolkan dengan Q3.
Posisi ketiga kuartil ditentukan dari rumus berikut.

Posisi Qi = i(n+1)/4

i = indeks kuartil yaitu 1, 2, 3 dan n = banyaknya data

Misalkan kita akan menentukan kuartil bawah, tengah, dan kuartil atas dari 15 data berikut: 11, 24, 12, 15, 12, 18, 22, 25, 26, 27, 17, 22, 24, 19, 12.

Urutan data dari yang terkecil:
11, 12, 12, 12, 15, 17, 18, 19, 22, 22, 24, 24, 25, 26, 27
Posisi ketiga kuartil adalah sebagai berikut
Posisi Q1 = 1.(15+1)/4 = (16)/4 = 4 (data urutan ke 4)
Posisi Q2 = 2. (15+1)/4 = 2(16)/4 = 8 (data urutan ke 4)
Posisi Q3 = 3. (15+1)/4 = 3(16)/4 = 12 (data urutan ke 4)
Berdasarkan posisi kuartil pada urutan data maka dapat ditentukan ketiga kuartil
11, 12, 12, 12, 15, 17, 18, 19, 22, 22, 24, 24, 25, 26, 27
Jadi
kuartil bawah adalah 12
Kuartil tengah = median = 19
Kuartil atas = 24

Jangkauan Antar Kuartil ( Hamparan)
Yang dimaksud jangkauan antar kuartil atau hamparan adalah selisih antara kuartil atas dengan kuartil bawah. Jangkauan antar kuartil diberi simbol H.

H = Q3 – Q1

Jangkauan Semi Antar Kuartil (Simpangan Kuartil)
Yang dimaksud semi antar kuartil atau simpangan kuartil adalah setengah dari selisih antara kuartil atas dengan kuartil bawah. Jangkauan semi antar kuartil diberi simbol Qd.

Qd = ½ ( Q3 – Q1)

7. cara mencari kuartil 1 gimana ?


Cara Mencari kuartil bawah atau Q₁ , yaitu :
Pertama yg anda harus mengerti ialah rumus Umum Untuk kuartil data tunggal :

Untuk n-ganjil :

[tex]Q_{i}= \frac{i(n+1)}{4} [/tex]

Dimana : Qi = Kuartil ke i
              n   = banyak data

Untuk n-genap :
[tex]Q_{1} = X_{(n+2)/4}[/tex]

Rumus Untuk kuartil bawah data berkelompok :
[tex]Q_{1} = T_{b} + ( \frac{ \frac{1}{4}n - F}{f} ) \cdot I[/tex]

Dimana : Q₁ = kuartil ke 1
              Tb = Tepi bawah kelas kuartil ke 1
              n   = banyaknya data
              F   = Frekuensi kumulatif kelas sebelum kelas kuartil
              f    = Frekuensi kelas kuartil
              I    = Lebar kelas

18. Diketahui upah 8 karyawan adalah sebagai berikut ini, 20000,80000,75000,60000,50000,85000,45000,90000. Hitung mean, median, kuartil 1, kuartil 2 dan kuartil 3 data tersebut.


Nilai meannya adalah 63.125.Nilai mediannya adalah 67.500.Nilai kuartil bawah (Q1) adalah 50.000.Nilai kuartil tengah (Q2) adalah 75.000.Nilai kuartil atas (Q3) adalah 85.000.

Pembahasan

Ukuran Pemusatan Data

1. Mean/Rata-rata

[tex] \boxed{ \rm{x \: = \: \dfrac{jumlah \: semua \: nilai}{banyaknya \: nilai} }}[/tex]

2. Modus (Mo)

Nilai yang sering muncul.

Nilai yang memiliki frekuensi terbesar.

3. Kuartil (Qi)

Kuartil terdapat 3 bagian, yaitu kuartil bawah (Q1), kuartil tengah (Q2), dan kuartil atas (Q3).

•> Kuartil bawah (Q1)

Jika jumlah data genap : [tex] \boxed{ \rm{Q_1 \: = \: x_{ \frac{n \: + \: 3}{4} }}}[/tex]Jika jumlah data ganjil : [tex] \boxed{ \rm{Q_2 \: = \: x_{ \frac{1}{4} (n \: + \:1) }}}[/tex]

•> Kuartil tengah (Q2) = Median

Jika jumlah data genap : [tex] \boxed{ \rm{Q_2 \: = \: \dfrac{1}{2} \{x_{ \frac{n}{2} } \: + \: x_{ \frac{n}{2} \: + \: 1} \}}}[/tex]Jika jumlah data ganjil : [tex] \boxed{ \rm{Q_2 \: = \: x_{ \frac{1}{2} (n \: + \:1) }}}[/tex]

•> Kuartil atas (Q3)

Jika jumlah data genap : [tex] \boxed{ \rm{Q_3 \: = \: x_{ \frac{3n \: + \: 2}{4} }}}[/tex]Jika jumlah data ganjil : [tex] \boxed {\rm{Q_3 \: = \: x_{ \frac{3}{4} (n \: + \: 1)}}}[/tex]

Keterangan:

x = mean/rata-rata

xn = data ke-n

Diketahui:

Upah 8 karyawan = jumlah data (genap)

20.000, 80.000, 75.000, 60.000, 50.000, 85.000, 45.000, 90.000

Jika diurutkan ↓↓↓

20.000, 45.000, 50.000, 60.000, 75.000, 80.000, 85.000, 90.000

Ditanyakan:

MeanMedianKuartil bawah (Q1)Kuartil tengah (Q2)Kuartil atas (Q3)

Penyelesaian:

a. Menentukan mean.

[tex] \rm{x \: = \: \dfrac{jumlah \: data}{banyak \: data} }[/tex]

[tex] \rm { x \: = \: \dfrac{20.000 \: + \: 45.000 \: + \:50. 000 \: + \:60.000 \: + \: 75.000 \: + \: 80.000 \: + \: 85.000 \: + \: 90.000 }{8} }[/tex]

[tex] \rm{x \: = \: \dfrac{505.000}{8} }[/tex]

[tex] \boxed{ \rm{x \: = \: 63.125}}[/tex]

Jadi, mean data tersebut adalah 63.125.

b. Menentukan median.

[tex] \rm{Me \: = \: \dfrac{1}{2} \: \{x_{ \frac{n}{2} } \: + \: x_{ \frac{n}{2} + 1} \}}[/tex]

[tex] \rm{Me \: = \: \dfrac{1}{2} \: \{x_{ \frac{8}{2} } \: + \: x_{ \frac{8}{2} + 1} \}}[/tex]

[tex] \rm{Me \: = \: \dfrac{1}{2} \: \{x_{4} \: + \: x_{5} \}}[/tex]

[tex] \rm{Me \: = \: \dfrac{1}{2} \: \times \: x_9}[/tex]

[tex] \rm{Me \: = \: x_{4,5}}[/tex]

Dapat kita ketahui, bahwa data ke-4,5 antara 60.000 dan 75.000. Maka:

[tex] \rm{Me \: = \: \dfrac{60.000 \: + \: 75.000}{2} }[/tex]

[tex] \rm{Me \: = \: \dfrac{135.000}{2} }[/tex]

[tex] \boxed{ \rm{Me \: = \: 67.500}}[/tex]

Jadi, median data tersebut adalah 67.500.

c. Menentukan kuartil bawah (Q1).

[tex] \rm{Q_1 \: = \: x_{ \frac{n \: + \: 3}{4}} }[/tex]

[tex] \rm{Q_1 \: = \: x_{ \frac{8 \: + \:3 }{4} }}[/tex]

[tex] \rm{Q_1 \: = \: x_{ \frac{11}{4} }}[/tex]

[tex] \rm{Q_1 \: = \: x_{2,75}}[/tex]

[tex] \rm{Q_1 \: = \: x_3 \: (dibulatkan)}[/tex]

[tex] \boxed{\rm{Q_1 \: = \: 50.000}}[/tex]

Jadi, nilai kuartil bawah pada data tersebut adalah 50.000.

d. Menentukan kuartil tengah (Q2).

[tex] \rm{Q_2 \: = \: Me}[/tex]

[tex] \rm{Q_2 \: = \: x_{4,5}}[/tex]

[tex] \rm{Q_2 \: = \: x_5 \: ( dibulatkan)}[/tex]

[tex] \boxed {\rm{Q_2 \: = \: 75.000}}[/tex]

Jadi, nilai kuartil tengah pada data tersebut adalah 75.000.

e. Menentukan kuartil atas (Q3).

[tex] \rm{Q_3 \: = \: x_{ \frac{3n \: + \:2 }{4} }}[/tex]

[tex] \rm{Q_3 \: = \: x_{ \frac{3(8) \: + \: 2}{4} }}[/tex]

[tex] \rm{Q_3 \: = \: x_{ \frac{24 \: + \: 2}{4} }}[/tex]

[tex] \rm{Q_3 \: = \: x_{ \frac{26}{4} }}[/tex]

[tex] \rm{Q_3 \: = \: x_{6,5}}[/tex]

[tex] \rm{Q_3 \: = \: x_7 \: (dibulatkan)}[/tex]

[tex] \boxed{ \rm{Q_3 \: = \: 85.000}}[/tex]

Jadi, nilai kuartil atas data tersebut adalah 85.000.
