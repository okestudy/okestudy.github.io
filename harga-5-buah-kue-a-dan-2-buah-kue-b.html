Ayo berlatih! Harga 5 buah kue A dan 2 buah kue B Rp4.000,00. Sedangkan harga 2 buah kue A dan harga 3 buah kue B Rp2.700,00. Jadi, harga sebuah kue A dan dua buah kue B adalah...Pembahasan :

Diketahui :

Harga 5 Kue A + 2 Kue B = Rp. 4.000.-

Harga 2 Kue A + 3 Kue B = Rp. 2.700.-


Ditanyakan :

Harga 1 Kue A + 2 Kue B = ...?

Penyelesaian :

Persoalan diata merupakan salah satu bentuk penerapan dari materi persamaan linear dua variabel.

Untuk menyelesaikannya, lakukan proses eliminasi salah satu nilai :

5A + 2B = Rp. 4.000 → Kali 3

2A + 3B = Rp. 2.700 → Kali 2

-----------------------------------------

15A + 6B = Rp. 12.000

4A + 6B = Rp. 5.400

------------------------------- -

        11A = Rp. 6.600.-

           A = Rp. 6.600 : 11

           A = Rp. 600.- → Harga 1 Buah Kue A

Selanjutnya menentukan harga Kue B :

      15A + 6B = Rp. 12.000 → subtitusi harga Kue A

15(600) + 6B = Rp. 12.000

 9.000 + 6B = Rp. 12.000

               6B = Rp. 12.000 - 9.000

               6B = Rp. 3.000.-

                 B = Rp. 3.000 : 6

                 B = Rp. 500.- → Harga 1 Buah Kue B

Sehingga, Harga sebuah kue A dan dua buah kue B adalah :

=  A + 2B

= 600 + 2(500)

= 600 + 1.000

= Rp. 1.600.-

Harga 5 buah kue A dan 2 buah kue B Rp4.000,00. Sedangkan harga 2 buah kue A dan harga 3 buah kue B Rp2.700,00. Jadi, harga sebuah kue A dan dua buah kue B adalah ....


A. Rp1.200,00


B. Rp1.600,00


C. Rp1.800,00


D. Rp2.400,00


PEMBAHASAN:

Diketahui:


5x + 2y = 4.000


2x + 3y = 2.700


Ditanya:


x + 2y = ?


Penyelesaian:


Pertama, kita eliminasi x terlebih dahulu


 


5x + 2y = 4.000 x 2 10x + 4y = 8.000


2x + 3y = 2.700 x5 10x + 15y = 13.500


------------------------------------------------_


-11y = -5.500


y = -5.500/-11


y = 500


Kedua, substitusikan nilai y ke salah satu persamaan


5x + 2.500 = 4.000


5x + 1.000 = 4.000


5x = 4.000 – 1.000


5x = 3.000


x = 3.000/5


x = 600


HP = {600, 500}


x + 2y = 600 + 2.500


= 600 + 1.000


= Rp1.600,00
