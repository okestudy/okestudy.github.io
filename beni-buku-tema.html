5. sebanyak 48 buku dibagikan kepada andi, beni, dan coki. banyak buku yang diterima andi 6 buku. banyak buku yang diterima beni dua kali dari banyak buku yang di terima coki. banyak buku yang di terima coki adalah ... buku


Misal banyak buku yang didapat Andi, Beni, dan Coki : A, B, C

Maka,
A + B + C = 48                    ...(1)
A = 6
Subs. nilai A = 6 ke pers. (1),
6 + B + C = 48
B + C = 42                          ...(2)
B = 2C                                ...(3)
Subs. pers. (3) ke pers. (2),
2C + C = 42
3C = 42
C = 14

Jadi, banyak buku yang diterima Coki adalah 14 buku.

23. jumlah buku andi 8 lebih banyak dari buku beni.buku kristin 3×buku andi. jika beni mempunyai x buku,jumlah buku andi,beni,dan kristin adalah​


Jumlah buku andi 8 lebih banyak dari buku beni.

→ A = B + 8

buku kristin 3×buku andi.

→ K = 3 × A

jika beni mempunyai x buku,

→ B = x

jumlah buku andi,beni,dan kristin adalah

= A + B + K

= (B + 8) + x + 3A

= (x + 8) + x + 3(B + 8)

= x + x + 8 + 3(x + 8)

= 2x + 8 + 3x + 24

= 5x + 32

Demikian, moga ga kliru:)
