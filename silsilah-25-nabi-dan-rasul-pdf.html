4. Buatlah silsilah 25 rasul, mulai dari nabi adam as sampai nabi muhammad swt. !


Jawaban:

Silsilah adalah asal-usul keluarga atau garis keturunan berupa bagan. Berikut penjelasan detail tentang garis keturunan Nabi Adam AS. hingga Nabi Muhammad SAW.

1. Nabi Adam AS.

Adam adalah manusia pertama yang diciptakan oleh Allah SWT. Allah SWT kemudian menciptakan Hawa untuk menemaninya, kemudian melahirkan anak-anak, dan generasi keturunannya terus berkembang hingga sampai saat ini.

2. Nabi Idris AS.

Garis keturunan: Adam AS → Syith → Anusy → Qinan → Mihlail → Yarid → Idris AS.

3. Nabi Nuh AS.

Garis keturunan: Idris AS → Matusyalih → Lamak → Nuh AS.

4. Nabi Hud AS.

Garis keturunan: Nuh AS. → Sam → Iram (Aram) → 'Aush → 'Ad → al-Khulud → Rabah → Abdullah → Hud AS.

5. Nabi Shalih AS.

Garis keturunan : Nuh AS. → Sam → Iram (Aram) → Amir → Tsamud → Hadzir → Ubaid → Masih → Asif → Ubaid → Shalih AS.

6. Nabi Ibrahim AS.

Garis keturunan: Nuh AS. → Sam → Arfakhsyad → Syalih → Abir → Falij → Ra'u → Saruj → Nahur → Tarakh → Ibrahim AS.

7. Nabi Luth AS.

Garis keturunan: Nuh AS. → Sam → Arfakhsyad → Syalih → Abir → Falij → Ra'u → Saruj → Nahur → Tarakh → Haran → Luth AS.

8. Nabi Ismail AS.

Garis keturunan: Ibrahim AS → Ismail AS.

9. Nabi Ishaq as.

Garis keturunan: Ibrahim AS → Ishaq AS.

10. Nabi Ya'qub AS.

Garis keturunan: Ishaq AS. → Ya'qub AS.

11. Nabi Yusuf AS.

Garis keturunan: Ya'qub AS. → Yusuf AS.

12. Nabi Ayyub AS.

Garis keturunan: Ishaq AS. → al-‘Aish → Rum → Tawakh → Amush → Ayyub AS.

13. Nabi Syu'aib AS.

Garis keturunann: Ibrahim AS. → Madyan → Yasyjur → Mikail → Syu’aib AS.

14. Nabi Musa AS.

Garis keturunan: Ya'qub AS → Lawi → Azar → Qahats → Imran → Musa AS.

15. Nabi Harun AS.

Garis keturunan: Ya’qub AS. → Lawi → Azar → Qahats → Imran → Harun AS.

16. Nabi Dzulkifri AS.

Garis keturunan: Ayyub AS. → Dzulkifri AS.

17. Nabi Daud AS.

Garis keturunan: Ishaq AS. → Yahudza → Farish → Hashrun → Aram → Aminadab → Hasyun → Salmun → Bu’az → Uwaibid → Isya → Daud AS.

18. Nabi Sulaiman AS.

Garis keturunan: Daud AS. → Sulaiman AS.

19. Nabi Ilyas AS.

Garis keturunanL: Harun AS. → Alzar → Fanhash → Yasin → Ilyas AS.

20. Nabi Ilyasa AS.

Garis keturunan: Yusuf AS. → Ifrayim → Syutlim → Akhthub → Ilyasa’ AS.

21. Nabi Yunus AS.

Garis keturunan: Bunyamin → Abumatta → Matta → Yunus AS.

22. Nabi Zakaria AS.

Garis keturunan: Sulaiman AS. → Rahab’am → Aynaman → Yahfayath → Syalum → Nahur → Bal’athah → Barkhiya → Shiddiqah → Muslim → Sulaiman → Daud → Hasyban → Shaduq → Muslim → Dan → Zakariya AS.

23. Nabi Yahya AS.

Garis keturunan: Zakariya AS. → Yahya AS.

24. Nabi Isa AS.

Garis keturunan: Sulaiman AS. → Rahab’am → Radim → Yahusafat → Barid → Nausa → Nawas → Amsaya → Izazaya → Au’am → Ahrif → Hizkil → Misyam → Amur → Sahim → Imran → Maryam → Isa AS.

25. Nabi Muhammad SAW.

Garis keturunan: Ismail AS. → Nabit → Yasyjub → Ya’rub → Tairah → Nahur → Muqawwim → Udad → Adnan → Ma’ad → Nizar → Mudhar → Ilyas → Mudrikah → Khuzaimah → Kinanah → an-Nadhar → Malik → Quraisy (Fihr) → Ghalib → Lu’ay → Ka’ab → Murrah → Kilab → Qushay → Zuhrah → Abdu Manaf → Hasyim → Abdul Muthalib → Abdullah → Muhammad SAW.

14. Sebutkan silsilah dan rasul 25. Saya belum paham silsilah itu


Jawaban:

1. Nabi Adam As.

Nabi Adam As merupakan manusia serta khalifah pertama yang diciptakan Allah SWT dari segumpal tanah.

2. Nabi Idris As.

Nabi Idris As merupakan keturunan keenam Nabi Adam As, dan beliau merupakan nabi pertama yang diangkat Allah SWT dari garis keturunan Nabi Adam.

3. Nabi Nuh As.

Nabi Nuh As diberikan beberapa mukjizat. Satu di antaranya membuat bahtera besar untuk menyelamatkan kaumnya dan kehidupan binatang dari banjir bandang.

4. Nabi Hud As.

Nabi Hud As diutus untuk mengajak kaum Aad berhenti menuhankan atau menyembah berhala. Nabi Hud mengajak kaum Aad untuk menyembah kepada Allah.

5. Nabi Saleh As.

Mukjizat yang diberikan Allah oleh Nabi Sholeh As, yakni seekor unta betina yang lahir dari celah-celah batu.

6. Nabi Ibrahim As.

Nabi Ibrahim atau "Bapak para nabi" memiliki satu di antara mukjizat dari Allah SWT untuk membangun Ka'bah.

7. Nabi Luth As.

Nabi Luth As merupakan keponakan Nabi Ibrahim As. Beliau diutus Allah SWT bagi kaum Sodom dan Gomorrah yang memiliki perilaku seks yang menyimpang.

8. Nabi Ismail As.

Nabi Ismail merupakan putra dari Nabi Ibrahim dan Siti Hajar. Beliau bersama-sama membangun Ka’bah dengan ayahnya yang saat ini menjadi pusat peribadatan kaum muslim di seluruh dunia.

9. Nabi Ishaq As.

Nabi Ishaq merupakan putra kedua dari Nabi Ibrahim As dan beliau merupakan adik Nabi Ismail As.

10. Nabi Yaqub As.

Nabi Yakub As merupakan anak Nabi Ishaq As. Dalam beberapa riwayat, Nabi Yaqub As digambarkan sebagai sosok yang memiliki karakter yang kuat dan keimanan yang luar biasa.

11. Nabi Yusuf As.

Nabi Yusuf As merupakan anak ke-12 Nabi Yaqub bin Ishaq. Mukjizat yang dimiliki Nabi Yusuf As adalah memiliki kelebihan yang mampu menafsirkan mimpi.

12. Nabi Ayyub As.

Nabi Ayub As bisa dikatakan sebagai satu di antara nabi tersabar yang pernah ada. Kehidupannya penuh kebaikan dan kaya raya.

Allah SWT mengujinya melalui penderitaan panjang dengan penyakit kulit hingga hartanya habis.

13. Nabi Syu’aib As.

Nabi Syuaib As diutus untuk kaum Madyan, namun kaum ini tetap ingkar dan menolak ajakan Nabi Syuaib As untuk menyembah kepada Allah SWT.

14. Nabi Musa As.

Nabi Musa As merupakan saudara Nabi Harun As, yang dilahirkan di Mesir ketika masa pemerintahan Raja Fir’aun. Kisah Beliau paling terkenal adalah ketika tongkat Nabi Musa As mampu membelah lautan ketika dikejar oleh Raja Fir’aun.

15. Nabi Harun As.

Nabi Harun As adalah saudara Nabi Musa As yang membantunya berdakwah untuk mengingatkan Fir’aun. Beliau dikaruniai kemampuan berbahasa yang luar biasa.

16. Nabi Zulkifli As.

Nabi Zulkifli As adalah satu di antara utusan Allah SWT yang terkenal dengan kesabaran serta sifat dermawannya. Nabi Zulkifli As adalah anak satu-satunya dari Nabi Ayyub As yang selamat dari reruntuhan rumah.

17. Nabi Daud As.

Nabi Daud memiliki mukjizat kitab Zabur. Selain itu, beliau juga memiliki kemampuan dapat berkomunikasi dengan burung, memiliki suara merdu dan bisa melunakkan besi dengan tangan kosong.

18. Nabi Sulaiman As.

Satu di antara mukjizat Nabi Sulaiman As adalah bisa bicara dengan binatang.

19. Nabi Ilyas As.

Nabi Ilyas As merupakan utusan Allah SWT bagi kaun Bani Israil yang gemar menyembah patung yang diberi nama Ba’al.

20. Nabi Ilyasa As.

Nabi Ilyas As diutus oleh Allah SWT pada kaum Bani Israil dan Arami supaya mereka mau kembali ke jalan yang benar, yakni di jalan Allah SWT.

21. Nabi Yunus As.

Mukjizat Nabi Yunus As yang terkenal adalah tetap selamat ketika ditelan ikan Paus.

22. Nabi Zakaria As.

Nabi Dzakaria As merupakan keturunan Nabi Daud As dan Nabi Sulaiman As.

23. Nabi Yahya As.

Nabi Yahya As adalah anak dari Nabi Zakaria As yang lahir ketika usianya sudah sangat tua. Nabi Yahya As adalah seorang manusia yang berprinsip, integritasnya tinggi serta benar-benar menegakkan kebenaran.

24. Nabi Isa As.

Nabi Isa As terlahir dengan mukjizat Allah SWT pada seorang perempuan suci bernama Maryam. Beliau mendapatkan mukjizat kitab Injil dan menjadi nabi dari umat Nasrani.

25. Nabi Muhammad SAW.

Nabi Muhammad SAW merupakan nabi terakhir sekaligus penutup. Mukjizat Nabi Muhammad SAW adalah kitab suci Al-Qur'an.

Kitab suci Al-Qur'an sebagai kitab suci yang terakhir sebagai penyempurna kitab-kitab sebelumnya yang diturunkan oleh Allah SWT.

Pembahasan

Silsilah adalah suatu bagan yang menampilkan hubungan keluarga (silsilah) dalam suatu struktur pohon.

Untuk silsilah para Nabi, tidak mungkin dicantumkan secara keseluruhan, seperti putra-putri Nabi Adam ada 40 orang, tidak mungkin kita masukkan bagan seluruhnya.

Secara garis besar, silsilah para Nabi dapat dilihat dalam lampiran.

Maaf, itu hanya secara garis besar, tidak dapat kita buat secara detail.

Pembahasan

Silsilah adalah asal-usul keluarga atau garis keturunan berupa bagan. Berikut penjelasan detail tentang garis keturunan Nabi Adam AS. hingga Nabi Muhammad SAW.

1. Nabi Adam AS.

Adam adalah manusia pertama yang diciptakan oleh Allah SWT. Allah SWT kemudian menciptakan Hawa untuk menemaninya, kemudian melahirkan anak-anak, dan generasi keturunannya terus berkembang hingga sampai saat ini.

2. Nabi Idris AS.

Garis keturunan: Adam AS → Syith → Anusy → Qinan → Mihlail → Yarid → Idris AS.

3. Nabi Nuh AS.

Garis keturunan: Idris AS → Matusyalih → Lamak → Nuh AS.

4. Nabi Hud AS.

Garis keturunan: Nuh AS. → Sam → Iram (Aram) → 'Aush → 'Ad → al-Khulud → Rabah → Abdullah → Hud AS.

5. Nabi Shalih AS.

Garis keturunan : Nuh AS. → Sam → Iram (Aram) → Amir → Tsamud → Hadzir → Ubaid → Masih → Asif → Ubaid → Shalih AS.

6. Nabi Ibrahim AS.

Garis keturunan: Nuh AS. → Sam → Arfakhsyad → Syalih → Abir → Falij → Ra'u → Saruj → Nahur → Tarakh → Ibrahim AS.

7. Nabi Luth AS.

Garis keturunan: Nuh AS. → Sam → Arfakhsyad → Syalih → Abir → Falij → Ra'u → Saruj → Nahur → Tarakh → Haran → Luth AS.

8. Nabi Ismail AS.

Garis keturunan: Ibrahim AS → Ismail AS.

9. Nabi Ishaq as.

Garis keturunan: Ibrahim AS → Ishaq AS.

10. Nabi Ya'qub AS.

Garis keturunan: Ishaq AS. → Ya'qub AS.

11. Nabi Yusuf AS.

Garis keturunan: Ya'qub AS. → Yusuf AS.

12. Nabi Ayyub AS.

Garis keturunan: Ishaq AS. → al-‘Aish → Rum → Tawakh → Amush → Ayyub AS.

13. Nabi Syu'aib AS.

Garis keturunann: Ibrahim AS. → Madyan → Yasyjur → Mikail → Syu’aib AS.

14. Nabi Musa AS.

Garis keturunan: Ya'qub AS → Lawi → Azar → Qahats → Imran → Musa AS.

15. Nabi Harun AS.

Garis keturunan: Ya’qub AS. → Lawi → Azar → Qahats → Imran → Harun AS.

16. Nabi Dzulkifri AS.

Garis keturunan: Ayyub AS. → Dzulkifri AS.

17. Nabi Daud AS.

Garis keturunan: Ishaq AS. → Yahudza → Farish → Hashrun → Aram → Aminadab → Hasyun → Salmun → Bu’az → Uwaibid → Isya → Daud AS.

Garis keturunanL: Harun AS. → Alzar → Fanhash → Yasin → Ilyas AS.

20. Nabi Ilyasa AS.

Garis keturunan: Yusuf AS. → Ifrayim → Syutlim → Akhthub → Ilyasa’ AS.

21. Nabi Yunus AS.

Garis keturunan: Bunyamin → Abumatta → Matta → Yunus AS.

Garis keturunan: Zakariya AS. → Yahya AS.
