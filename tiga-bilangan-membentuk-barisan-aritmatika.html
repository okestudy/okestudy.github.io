4. tiga buah bilangan membentuk barisan aritmatika. jumlah ketiga bilangan tersebut adalah 9. jika bilangan terakhir ditambah 4,maka ketiga bilangan itu membentuk barisan geometri. selisih bilangan terkecil dan terbesar dari barisan geometri itu ​


Jawab:

Selisih bilangan terkecil dan terbesar dari barisan geometri itu ​ adalah 8.

Penjelasan dengan langkah-langkah:

U₁, U₂, U₃ membentuk barisan aritmatika :

a, a + b , a + 2b

Jumlah ke-3 suku tersebut adalah 9 :

a + a + b + a + 2b = 9

3a + 3b = 9

a + b = 3

Suku ke-3 ditambah 4 menjadi barisan geometri :

a, a + b, a + 2b + 4

a + b = 3 ..... (U₂)

a = 3 - b ...... (U₁)

a + 2b + 4 = a + b + b + 4 = 3 + b + 4 = b + 7 ..... (U₃)

Dalam Barisan geometri berlaku :

U₂ / U₁ = U₃ / U₂

3 / (3 - b) = (b + 7) / 3

(3 - b) (b + 7) = 3 . 3

3b - b² + 21 - 7b = 9

-b² - 4b + 21 - 9 = 0

-b² - 4b + 12 = 0

b² + 4b - 12 = 0

(b + 6) (b - 2) = 0

b = -6 atau b = 2

Untuk b = -6, Barisan geometri yang terbentuk :

U₁ = 3 - b = 3 - (-6) = 9

U₂ = 3

U₃ = b + 7 = -6 + 7 = 1

U₁ - U₃ = 9 - 1 = 8

untuk b = 2, Barisan geometri yang terbentuk :

U₁ = 3 - b = 3 - 2 = 1

U₂ = 3

U₃ = b + 7 = 2 + 7 = 9

U₃ - U₁ = 9 - 1 = 8

Jadi  selisih bilangan terkecil dan terbesar dari barisan geometri itu ​ adalah 8

8. tiga buah bilangan (x + 1) , (2x - 1) dan (2x+2) membentuk barisan aritmatika , tentukan bilangan-bilangan yang menyusun barisan aritmatika tersebut.


Tiga buah bilangan (x + 1), (2x – 1) dan (2x + 2) membentuk barisan aritmatika, bilangan-bilangan yang menyusun barisan aritmatika tersebut adalah 6, 9, 12. Barisan aritmatika adalah barisan yang antar dua suku berdekatannya memiliki selisih yang tetap.  

Rumus suku ke n pada barisan aritmatika

Un = a + (n – 1)b

Rumus jumlah n suku pertama

Sn =  [tex]\frac{n}{2}[/tex] (2a + (n – 1)b) Sn =  [tex]\frac{n}{2}[/tex] (a + Un)

Keterangan

a = suku pertama b = beda ⇒ b = U₂ – U₁ = U₃ – U₂ = ....

Pembahasan

Diketahui

(x + 1), (2x – 1) dan (2x + 2) membentuk barisan aritmatika

Ditanyakan

Bilangan-bilangan yang dimaksud = … ?

Jawab

Dengan menggunakan rumus beda, maka diperoleh

U₂ – U₁ = U₃ – U₂

(2x – 1) – (x + 1) = (2x + 2) – (2x – 1)

2x – 1 – x – 1 = 2x + 2 – 2x + 1

x – 2 = 3

x = 3 + 2

x = 5

Jadi bilangan-bilangan tersebut adalah

= (x + 1), (2x – 1), (2x + 2)

= (5 + 1), (2(5) – 1), (2(5) + 2)

= (6), (10 – 1), (10 + 2)

= 6, 9, 12

Materi : Barisan dan Deret

Misalkan suku sukunya a, b, dan c, maka :

a, b, c (aritmatika)

karena a, b, c aritmatika, maka :

a = a

b = a + b

c = a + 2b

Jika b + 3 dan c - 21, maka :

a, b + 3, c - 21 (geometri) atau

a, a + b + 3, a + 2b - 21 (geometri)

Bila c + 9, maka c + 9 (aritmatika) = 3(b + 3) (geometri) sehingga :

a + 2b + 9 = 3a + 3b + 9

2a + b = 0

b = -2a...(1)

[tex]\frac{u_2}{u_1}=\frac{u_3}{u_2}[/tex]

[tex]\frac{a+b+3}{a}=\frac{a+2b-21}{a+b+3}\\{(a+b+3)}^2=a(a+2b-21)\\{(a-2a+3)}^2=a(a-4a-21)\\{(3-a)}^2=-3a^2-21a\\9-6a+a^2=-3a^2-21a\\4a^2+15a+9=0\,(bagi\,2)\\(4a+3)(a+3)=0\\a=-\frac{3}{4}\,dan\,a=-3[/tex]

ambil a bulat, diperoleh :

b = -2(-3) = 6
