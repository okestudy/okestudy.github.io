12. Jika jumlah kromosom diploid suatu organisme adalah 48 maka jumlah kromosom haploid nya adalah


Jika jumlah kromosom diploid suatu organisme adalah 48 maka jumlah kromosom haploidnya adalah?

Jawaban:

jumlah kromosom haploidnya adalah 24, atau separuh dari 48.

Pendahuluan:

Diploid artinya sel atau organisme tersebut memiliki 2 set kromosom lengkap di inti sel.

Sementara haploid berarti sel atau organisme tersebut hanya memiliki separuh dari jumlah set kromosom, pada organisme diploid ini berarti memiliki 1 set kromosom lengkap di inti sel.

Jumlah kromosom diploid adalah 2 kali jumlah kromosom haploid, atau jumlah kromosom haploid adalah ½ jumlah kromosom diploid.

Pembahasan:

Sel dan makhkluk hidup digolongkan sesuai dengan jumlah set kromosom yang ada di dalam inti (level ploidy): monoploid (1 set kromosom lengjao), diploid (2 set), triploid (3 set), tetraploid (4 set), dan seterusnya. Istilah "poliploid" digunakan untuk menggambarkan sel dengan tiga atau lebih set kromosom (triploid atau ploidy yang lebih tinggi).

Haploid adalah separuh dari jumlah set normal kromosom, sehingga bila jumlah normal adalah dua set (2 set) maka haploidnha memiliki 1 set.

Manusia adalah organisme diploid, artinya manusia memiliki dua set kromosom lengkap: satu set dari 23 kromosom dari ayah mereka dan satu set dari 23 kromosom dari ibu mereka. Kombinasi dua set menjadi set lengkap yang terdiri atas 46 kromosom.  

Jumlah haploid untuk manusia (setengah dari 46) adalah satu set lengkap atau 23 kromosom. Kondisi haploid ini ditemukan pada sel sperma dan sel telur, yang masing-masing hanya memiliki satu set lengkap atau 23 kromosom.  

Ketika terjadi pembuahan kedua set kromosom bergabung saat sel sperma dan sel telur membentuk zigot. Zigot ini bersifat diploid (memiliki dua set kromosom).
