8. Tentukan titik stasioner, nilai stasioner, serta jenisnya untuk fungsi trigonometri f(x)=sin(x + phi/6) pada interval 0≤x≤2phi.​


Titik stasioner dan jenisnya untuk [tex]f(x)=sin\left ( x+\frac{\pi}{6} \right ),~0\leq x\leq 2\pi[/tex] adalah titik maksimum di [tex]\boldsymbol{\left ( \frac{\pi}{3},1 \right )}[/tex] dan titik minimum di [tex]\boldsymbol{\left ( \frac{4\pi}{3},-1 \right )}[/tex].

PEMBAHASAN

Turunan atau Diferensial merupakan pengukuran terhadap bagaimana fungsi berubah seiring perubahan nilai input. Salah satu fungsi dari turunan adalah menentukan nilai minimum/maksimum dari suatu fungsi. Dimana suatu fungsi akan memiliki nilai minimum/maksimum pada saat :

[tex]f'(x)=0[/tex]

dengan :

f'(x) = turunan pertama fungsi.

Dari [tex]f'(x)=0[/tex] kita akan memperoleh titik titik stasioner, misal x = a. Untuk menentukan apakah titik tersebut menyebabkan fungsi bernilai minimum atau maksimum dapat kita gunakan uji turunan kedua.

1. Jika f''(a) > 0 maka x = a menyebabkan fungsi bernilai minimum.

2. Jika f''(a) < 0 maka x = a menyebabkan fungsi bernilai maksimum.

3. Jika f''(a) = 0 maka x = a merupakan titik belok fungsi.

.

DIKETAHUI

[tex]f(x)=sin\left ( x+\frac{\pi}{6} \right ),~0\leq x\leq 2\pi[/tex]

.

DITANYA

Tentukan titik stasioner, nilai stasioner dan jenisnya.

.

PENYELESAIAN

> Cari titik stasioner menggunakan turunan pertama.

[tex]f(x)=sin\left ( x+\frac{\pi}{6} \right )[/tex]

[tex]f'(x)=cos\left ( x+\frac{\pi}{6} \right )[/tex]

[tex]f''(x)=-sin\left ( x+\frac{\pi}{6} \right )[/tex]

.

Titik stasioner diperoleh pada saat [tex]f'(x)=0[/tex].

Gunakan rumus :

[tex]cosx=cosA,~maka~[/tex] [tex]x=A+K\times2\pi~~atau~~x=-A+K\times2\pi[/tex]

.

[tex]f'(x)=0[/tex]

[tex]cos\left ( x+\frac{\pi}{6} \right )=0[/tex]

[tex]cos\left ( x+\frac{\pi}{6} \right )=cos\frac{\pi}{2}[/tex]

[tex]x+\frac{\pi}{6}=\frac{\pi}{2}+K\times2\pi[/tex]

[tex]x=\frac{\pi}{3}+K\times2\pi[/tex]

[tex]K=0~\to~x=\frac{\pi}{3}+0\times2\pi=\frac{\pi}{3}[/tex]

[tex]K=1~\to~x=\frac{\pi}{3}+1\times2\pi=\frac{7\pi}{3}~(\boldsymbol{tidak~memenuhi})[/tex]

.

Atau

[tex]x+\frac{\pi}{6}=-\frac{\pi}{2}+K\times2\pi[/tex]

[tex]x=-\frac{2\pi}{3}+K\times2\pi[/tex]

[tex]K=0~\to~x=-\frac{2\pi}{3}+0\times2\pi=-\frac{2\pi}{3}~(\boldsymbol{tidak~memenuhi})[/tex]

[tex]K=1~\to~x=-\frac{2\pi}{3}+1\times2\pi=\frac{4\pi}{3}[/tex]

[tex]K=2~\to~x=-\frac{2\pi}{3}+2\times2\pi=\frac{10\pi}{3}~(\boldsymbol{tidak~memenuhi})[/tex]

.

Diperoleh titik stasioner : [tex]x=\frac{\pi}{3}~dan~x=\frac{4\pi}{3}[/tex]

.

> Cek uji turunan kedua.

Kita substitusi titik stasione ke fungsi turunan kedua.

[tex]x=\frac{\pi}{3}:[/tex]

[tex]f''\left ( \frac{\pi}{3} \right )=-sin\left ( \frac{\pi}{3}+\frac{\pi}{6} \right )[/tex]

[tex]f''\left ( \frac{\pi}{3} \right )=-sin\left ( \frac{\pi}{2} \right )[/tex]

[tex]f''\left ( \frac{\pi}{3} \right )=-1~(< 0)[/tex]

Karena hasilnya < 0 maka untuk [tex]x=\frac{\pi}{3}[/tex] fungsi akan bernilai maksimum.

.

[tex]x=\frac{4\pi}{3}:[/tex]

[tex]f''\left ( \frac{4\pi}{3} \right )=-sin\left ( \frac{4\pi}{3}+\frac{\pi}{6} \right )[/tex]

[tex]f''\left ( \frac{\pi}{3} \right )=-sin\left ( \frac{3\pi}{2} \right )[/tex]

[tex]f''\left ( \frac{\pi}{3} \right )=1~(> 0)[/tex]

Karena hasilnya > 0 maka untuk [tex]x=\frac{4\pi}{3}[/tex] fungsi akan bernilai minimum.

.

> Mencari nilai stasioner.

Kita substitusi titik stasione ke fungsi f(x).

[tex]x=\frac{\pi}{3}:[/tex]

[tex]f\left ( \frac{\pi}{3} \right )=sin\left ( \frac{\pi}{3}+\frac{\pi}{6} \right )[/tex]

[tex]f\left ( \frac{\pi}{3} \right )=sin\left ( \frac{\pi}{2} \right )[/tex]

[tex]f\left ( \frac{\pi}{3} \right )=1[/tex]

.

[tex]x=\frac{4\pi}{3}:[/tex]

[tex]f\left ( \frac{4\pi}{3} \right )=sin\left ( \frac{4\pi}{3}+\frac{\pi}{6} \right )[/tex]

[tex]f\left ( \frac{4\pi}{3} \right )=sin\left ( \frac{3\pi}{2} \right )[/tex]

[tex]f\left ( \frac{4\pi}{3} \right )=-1[/tex]

.

Kita peroleh titik titik stasioner sebagai berikut :

[tex]\left ( \frac{\pi}{3},1 \right )~\to~titik~maksimum[/tex]

[tex]\left ( \frac{4\pi}{3},-1 \right )~\to~titik~minimum[/tex]

.

KESIMPULAN

Titik stasioner dan jenisnya untuk [tex]f(x)=sin\left ( x+\frac{\pi}{6} \right ),~0\leq x\leq 2\pi[/tex] adalah titik maksimum di [tex]\boldsymbol{\left ( \frac{\pi}{3},1 \right )}[/tex] dan titik minimum di [tex]\boldsymbol{\left ( \frac{4\pi}{3},-1 \right )}[/tex].

.

11. tentukan nilai titik stasioner (min,max/ belok) dari fungsi trigonometri x³ + 3x² + 3x + 1​


Penjelasan dengan langkah-langkah:

Turunan pertama dan kedua dari fungsi

y = x³ + 6x² + 9x + 7

y'' = 6x + 12

Titik belok adalah apabila turunan kedua fungsi sama dengan nol

y'' = 6x + 12

0 = 6x + 12

6x = -12

x = -2

Kemudian untuk mencari nilai y, masukkan nilai x = -2 ke dalam fungsi awal

y = x³ + 6x² + 9x + 7

y = -8 + 24 - 18 + 7

y = 5

Sehingga titik beloknya adalah (-2, 5)

19. Diketahui fingsi trigonometri f(x) = sin 2x. Tentukan titik stasioner fungsi f(x) pada interval 0 ≤ x ≤ 2π dan jenis stasionernya!


Diketahui fungsi trigonometri f(x) = sin 2x. Tentukan titik stasioner fungsi f(x) pada interval 0 ≤ x ≤ 2π dan jenis stasionernya!

Pembahasan :

Titik stasioner diperoleh saat f'(x) = 0

Ada 3 jenis titik stasioner

1) titik balik maksimum
=> jika garis bilangan f'(x) = 0 nya adalah +++|---
atau
=> f"(a) < 0

2) titik balik minimum
=> jika garis bilangan f'(x) = 0 nya adalah ---|+++
atau
=> f"(a) > 0

3) titik belok
=> jika garis bilangan f'(x) = 0 nya adalah +++|+++ atau ---|---
atau
=> f"(a) = 0

Turunan trigonometri :
y = sin u => y' = u' . cos u
y = cos u => y' = -u' . sin u
dengan u adalah fungsi dalam x


KITA KEMBALI KE SOAL

f(x) = sin 2x
f'(x) = 2 cos 2x

Titik stasioner diperoleh jika f'(x) = 0
2 cos 2x = 0
cos 2x = 0
cos 2x = cos 90°
2x = 90° + k . 360° atau 2x = -90° + k . 360°
x = 45° + k . 180° atau x = -45 + k . 180°
x = 45° .......................... x = -45° (TM) => untuk k = 0
x = 225° ........................ x = 135° => untuk k = 1
x = 405° (TM) .............. x = 315° => untuk k = 2
HP = {45°, 135°, 225°, 315°}

Keterangan : TM = tidak memenuhi

Sekarang kita cari nilai stasionernya dengan disubstitusikan ke :
f(x) = sin 2x
f(45°) = sin 2(45°) = sin 90° = 1 => (45°, 1)
f(135°) = sin 2(135°) = sin 270° = -1 => (135°, -1)
f(225°) = sin 2(225°) = sin 450° = 1 => (225°, 1)
f(315°) = sin 2(315°) = sin 630° = -1 => (315°, -1)

tanpa menguji turunan kedua, sudah terlihat bahwa nilai maksimumnya 1 dan nilai minimumnya -1

Karena syaratnya dalah ukuran radian : 0 ≤ x ≤ 2π maka kita ubah x nya menjadi π radian
45° = (45°/180°)π = (1/4)π = π/4
135° = 3 × 45° = 3 × π/4 = 3π/4
225° = 5 × 45° = 5 × π/4 = 5π/4
315° = 7 × 45° = 7 × π/4 = 7π/4

Jadi titik stasionernya adalah :
1) titik balik maksimum = (π/4, 1) dan (5π/4, 1)
2) titik balik minimum = (3π/4, -1) dan (7π/4, -1)
