4. DIketahui : laska : 172 Km waktu sekunder : 08.30'.25'' Ditanya : Waktu sekunder ?

JAWABAN:

Mungkin maksudnya mencari waktu gelombang primer.

DIketahui :
laska : 172 Km
waktu sekunder : 08.30'.25''
Ditanya :
Pukul gelombang primer?
Jawab:
P = {S - (E/1000) + 1') }
P = {08.30'.25'' - (172/1000) + 1') }
P = {08.30'.25'' - (0,172 + 1')}
P = {08.30'.25'' - (1' 172) }
P = {08.30'.25'' - (1' 172/1000'') }
P = {08.30'.25'' - (1' (0,172 x 60"))}
P = {08.30'.25'' - 1' 10" }
P = 08.29'15"

Jadi gelombang primer datang pukul 08.29'15"

PEMBAHASAN LEBIH LANJUT:

Episentrum gempa adalah gelombang gempa yang berada di permukaan bumi dan tegak lurus dengan hiposentrum kemudian menyebar ke berbagai arah. Untuk menghitung jarak episentrum gempa dapat menggunakan metode episentral berikut:

Metode Episentral
Episentral ialah jarak episentrum atau pusat gempa di stasiun pencatat gempa. Untuk menentukan episentrum dengan menggunakan metode episentral diperlukan minimal tiga stasiun pengamat yang mencatat kejadian gempa, sehingga dapat dihitung jarak episentral masing-masing stasiun. Untuk menghitung jarak episentral digunakan rumus LASKA, sebagai berikut:

Delta = ((S-P) – 1’) X 1.000 Kilometer)
P = {S - (E/1000) + 1') }
S = {(E/1000) + 1') + P}

Ket:
Delta = jarak episentral dari stasiun pengamat dalam satuan kilometer
S = gelombang sekunder (dalam menit)
P = gelombang primer (dalam menit)
1’ = 1 menit = 60 detik (AS)

Jawaban:

Rumus Laska adalah rumus yang digunakan untuk mencari jarak episentrum gempa Bumi. Berikut ini merupakan rumus laska:

Rumus: Δ = {(S-P) - 1'} x 1.000 Km

Keterangan:

Δ = Episentrum gempa (Km)

S = Gelombang sekunder

P = Gelombang primer

1.000 = konstanta

→ Contoh soal:

Pada seismograf tercatat gelombang primer terjadi pada pukul 07.22'45" WIB dan gelombang sekunder terjadi pada pukul 07.24'.00" WIB. Berapakah jarak episentrum (Δ) ?

Diketahui:

Gelombang Primer (P) = 07.22'.45"

Gelombang Sekunder (S) = 07.24'.00"

Jawab:

$$\begin{lgathered}Episentrum = {(07.24'.00" - 07.22'.45") - 1'}\ x\ 1.000\ Km\\= {1'.15" - 1'} x 1.000\ Km\\= 0.15" x 1.000\ Km\\= \frac{15}{60}\ x\ 1.000\ Km\\= 250 Km\end{lgathered}$$

Berdasarkan perhitungan di atas, diperoleh jarak episentrum gempa tersebut adalah 250 Km.

16. contoh soal menggunakan rumus laska​


Jawaban:

Contoh soal menggunakan rumus LASKA :

Stasiun pencatat gempa A, mencatat gempa yang terjadi di wilayah tersebut. Gelombang primer tercatat terjadi pada pukul 02.28'.25''. Sedangkan gelombang sekunder tercatat terjadi pada pukul  02.30'.40". Hitunglah jarak Episentrum dari statiun pencatat gempa A!

Jawab :

Ep = {(s - p) - 1'} x 1.000 km

Ep = {(02.30'.40" - 02.28'.25") - 1'} x 1.000 km

Ep = {2'15'' - 1'} x 1.000 km

Ep = 1,25' x 1.000 km

Ep = 1.250 km

Jadi jarak episentrum dari stasiun pencatat A adalah sejauh 1.250 km

Penjelasan:

Episentrum adalah pusat gempa yang terletak di permukaan bumi. Sedangkan pusat bumi yang terletak di dalam bumi disebut dengan Hiposentrum. Untuk menghitung jarak episentru dapat digunakan rumus LASKA.

Ep = {(s - p) - 1'} x 1.000 km

Ep = jarak episentrum

s = waktu terjadi nya gelombang sekunder

p = waktu terjadi nya gelombang primer

17. 4. Gempa yang terjadi di wilayah A tercatat gelombang primer pada pukul 07.21'25" WIB sedangkan gelombang sekunder tercatat pada pukul 07.23' 10" WIB. Hitunglah jarak episentrum gempa dengan menggunakan rumus Laska!​


Jawaban:

Rumus Laska digunakan untuk menghitung jarak episentrum gempa berdasarkan selisih waktu kedatangan gelombang primer dan gelombang sekunder serta kecepatan rambat gelombang seismik di dalam bumi. Rumus Laska adalah:

Jarak (d) = (Selisih waktu / 2) x Kecepatan rambat gelombang

Kita perlu mengetahui kecepatan rambat gelombang di dalam bumi. Kecepatan ini bervariasi tergantung pada jenis batuan yang dilalui oleh gelombang. Untuk keperluan ini, kita asumsikan kecepatan rambat gelombang seismik di wilayah A sebesar 6 km/detik.

Selisih waktu kedatangan gelombang sekunder dan gelombang primer adalah:

07.23'10" - 07.21'25" = 1 menit 45 detik = 105 detik

Jarak episentrum gempa di wilayah A adalah:

d = (105 detik / 2) x 6 km/detik = 315 km

Jadi, jarak episentrum gempa di wilayah A adalah sekitar 315 km.

Rumus Laska untuk menghitung jarak episentrum gempa adalah:

Jarak (dalam km) = (kecepatan gelombang seismik x selisih waktu dalam detik) / 2

Namun, untuk keperluan perhitungan sederhana, kita dapat menggunakan kecepatan rata-rata gelombang seismik di bumi yang sekitar 8 km/s.

Selisih waktu antara gelombang primer dan gelombang sekunder adalah:

selisih waktu = (07.23'10" - 07.21'25") x 60 detik/menit + 45 detik

selisih waktu = 105 detik

Maka, jarak episentrum gempa di wilayah A dapat dihitung sebagai berikut:

jarak = (8 km/s x 105 detik) / 2

jarak = 420 km

Jadi, jarak episentrum gempa di wilayah A adalah sekitar 420 km.

Pada seismograf tercatat gelombang primer terjadi pada pukul 07.22'45" WIB dan gelombang sekunder terjadi pada pukul 07.24'.00" WIB. Berapakah jarak episentrum (Δ) ?

Diketahui:

Gelombang Primer (P) = 07.22'.45"

Gelombang Sekunder (S) = 07.24'.00"

Jawab:

[tex]Episentrum = {(07.24'.00" - 07.22'.45") - 1'}\ x\ 1.000\ Km\\= {1'.15" - 1'} x 1.000\ Km\\= 0.15" x 1.000\ Km\\= \frac{15}{60}\ x\ 1.000\ Km\\= 250 Km[/tex]

Berdasarkan perhitungan di atas, diperoleh jarak episentrum gempa tersebut adalah 250 Km.

24. Pada stasiun gempa dikota h mencatat gelombang gempa primer pada pukul 10.20’.15"dan sekunder pukul10.23’.30".berdasarkan rumus laska, jarak episentrum gempa dari stasiun pencatat gempa adalah.... *


■ Jawaban :

2.250 km

■ Penjelasan :

● Diketahui :

→ Gelombang Primer ( P ) = 10.20’.15"

→ Gelombang Sekunder ( S ) = 10.23’.30"

● Ditanya :

→ Jarak episentrum gempa ( Δ ) = ??

● Penyelesaian :

Δ = ( ( S - P ) - 1' ) × 1.000 km )

= ( ( 10.23’.30" - 10.20’.15"  ) - 1' ) × 1.000 km )

= ( 3'15" - 1' ) × 1.000 km )

= 2'15" × 1.000 km

= ( 2 × 1.000 km ) + ( 15/60 × 1.000 km )

= 2.000 km + 250 km

= 2.250 km

Jadi, jarak episentrum gempa dari stasiun pencatat gempa 2.250 km
