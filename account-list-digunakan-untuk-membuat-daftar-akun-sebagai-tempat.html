1. Jelaskan cara membuat daftar/list di html?

cara untuk membuat daftar/list di html kamu bisa mengunakan <ol>, <ul> dan <dl> untuk menentukan tipe listnya dan list mengunakan <li>(untuk <ol> dan <ul>) atau <dt> dan <dd>(untuk <dl>) untuk menyatakan bahwa teks didalamnya adalah bagian dari daftar atau listnya.

<ol> adalah list yang mempunyai urutannya contohnya

1. daftar 1

2. daftar 2

3. daftar 3

sedangkan <ol> adalah list yang tidak mempunyai urutannya contohnya

- daftar 1

- daftar 2

- daftar 3

sedangkan <dl> adalah list yang mempunyai deskripsinya, <dt> adalah sesuatu yang akan di deskripsikan sedangkan <dd> adalah deskripsinya contohnya

Kopi

    -minuman panas berwarna hitam

Susu

    -minuman dingin berwarna putih

<!DOCTYPE html>

<html>

     <head>

     </head>

     <body>

      <h1>List mempunyai urutan</h1>

       <ol>

           <li>urutan 1</li>

           <li>urutan 2</li>

           <li>urutan 3</li>

       </ol>

      <h1>List tanpa urutan</h1>

       <ul>

           <li>tanpa urutan 1</li>

           <li>tanpa urutan 2</li>

           <li>tanpa urutan 3</li>

       </ul>

      <h1>List deskripsi</h1>

       <dl>

           <dt>kopi</dt>

           <dd>Minuman panas berwarna hitam</dd>

           <dt>susu</dt>

           <dd>Minuman dingin berwarna putih</dd>

       </dl>

     </body>

</html>

15. Daftar berurut (Ordered List), daftar tidak berurut (Unordered List), daftar istilah (Definition List), dan daftar bersarang mempunyai tag dan atribut masing-masing. Tag yang digunakan untuk membuat list adalah ​


Jawaban:

1. Ordered List menggunakan tag <ol></ol>

2. Unordered List menggunakan tag <ul></ul>

3. Definition List menggunakan tag <dl></dl>

Penjelasan:

Untuk membuat sebuah list dalam HTML, struktur code biasanya seperti ini

1. Unordered List

<ul>

 <li>Coffee</li>

 <li>Tea</li>

 <li>Milk</li>

</ul>

2. Ordered list

<ol>

 <li>Coffee</li>

 <li>Tea</li>

 <li>Milk</li>

</ol>

3. Definition List

<dl>

 <dt>Coffee</dt>

 <dd>- black hot drink</dd>

 <dt>Milk</dt>

 <dd>- white cold drink</dd>

</dl>

tag <dt> bertujuan untuk menjelaskan term dari item list

sedangkan <dd> untuk deskripsi dari item

Itu bentuk sederhana dari list dalam HTML
