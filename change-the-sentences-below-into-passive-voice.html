Passive Voice:

1. Passive Voice [Simple Present Tense]

-Pattern = Subject + to be (is/am/are) + Verb3

2. Passive Voice [Present Continuous Tense]

-Pattern = Subject + to be (is/am/are) + being + Verb3

3. Passive Voice [Present Perfect Tense]

-Pattern = Subject + have/has + been + Verb3

4. Passive Voice [Simple Past Tense]

-Pattern = Subject + to be (was – were) + Verb 3

5. Passive Voice [Past Continuous Tense]

-Pattern = Subject + to be (was – were) + Being + Verb3

6. Passive Voice [Past Perfect Tense]

-Pattern = Subject + had + been + Verb3

7. Passive Voice [Simple Future Tense]

-Pattern = Subject + will + be + Verb3

8. Passive Voice [Future Continuous Tense]

-Pattern = Subject + will be + being + Verb3

9. Passive Voice [Future Perfect Tense]

-Pattern = Subject + will have + been + Verb3

5. change the following sentences into passive voice​


Jawaban:

Change the following sentences into passive voice.

1. Putri Permata usually telephones me on Saturdays.

Passive: I am usually telephoned by Putri Permata on Saturdays.

2. My sister cleans the room everyday.

Passive: The room is cleaned by my sister everyday.

3. My mother is washing the plates now.

Passive: The plates is being washed by my mother now.

4. The woman has helped the children.

Passive: The children have been helped by the woman.

5. The students have been doing their homework.

Passive: The students' homework has been being done (by them).

Penjelasan:

Pada soal nomor 1 dan 2: object + am/is/are + verb 3 + by subject (simple present tense)

Pada soal nomor 3: object + am/is/are + being + verb 3 + by subject (present continuous tense)

Pada soal nomor 4: object + has/have + been + verb 3 + by subject.

Pada soal nomor 5, yang di dalam kurung (by them), boleh ditulis tetapi lebih baik tidak ditulis karena pada object telah terdapat kata ganti subject-nya.

17. Change the sentences below into passive voice?? 1. John did not write the note


Jawaban:

1. The note was written by John.

ACTIVE-PASSIVE VOICE

The note was not written by John

Explanation

Passive voice, merefleksikan tindakan dan object (recipient) sebagai pemberitaan utama atau si pelaku aksi mendapatkan perlakuan dari object.

Pada umumnya, object pada kalimat aktif diposisikan sebagai subject pada kalimat pasif.

Sedangkan, subject (si pelaku aksi) pada kalimat aktif diposisikan sebagai Agent dengan diawali preposisi "by" pada kalimat aktif (obyek preposisi) dan terkadang dihilangkan.

Struktur utama Passive voice yaitu Be+V ³.

ELABORATION

1. Simple Past

A ➡ S+did+not+Verb¹+O

John did not write the note

[ John tidak menulis catatan ]

P ➡ S+was/were+not+Verb3+by agent.....

The note was not written by John

[ Catatan tidak ditulis oleh John ]

Conclusion :

The note ➡ Singular noun

John ➡ Agent [ the doer of action ]

Ketika kalimat aktif nya berbentuk negatif, kalimat pasif akan berbentuk negatif juga dengan tidak mengubah informasi yang terkandung dalam kalimat aktif.
