5. tentukan luas permukaan dan volume dari bangun kerucut berikut​


Diketahui :

d = 10 cm → r = 5 cm

s = 13 cm

Luas permukaan kerucut

Lpk = [tex] {\sf {\pi}} [/tex] × r × (r + s)

Lpk = 3,14 × 5 cm × (5 + 13) cm

Lpk = 3,14 × 5 cm × 18 cm

Lpk = 15,7 cm × 18 cm

Lpk = 282,6 cm²

Jadi, luas permukaan kerucut adalah 282,6 cm².

-----

Volume kerucut

Cari tinggi kerucut terlebih dahulu

t = [tex] {\sf{ \sqrt{{s}^{2} - {r}^{2}}}} [/tex]

t = [tex] {\sf{ \sqrt{{13}^{2} - {5}^{2}}}} [/tex]

t = [tex] {\sf{ \sqrt{169 - 25}}} [/tex]

t = 144

t = [tex] {\sf {\sqrt{144}}} [/tex]

t = 12 cm

Setelah itu, tinggal cari volume kerucut

V[tex] {\large {\sf {_{kerucut}}}} [/tex] = [tex] {\sf{ \dfrac{1}{3}}} [/tex] × [tex] {\sf{\pi}} [/tex] × r² × t

V[tex] {\large {\sf {_{kerucut}}}} [/tex] = [tex] {\sf {\dfrac{1}{3}}} [/tex] × 3,14 × 5² cm² × 12 cm

V[tex] {\large {\sf {_{kerucut}}}} [/tex] = [tex] {\sf {\dfrac{1}{3}}} [/tex] × 3,14 × 5 cm × 5 cm × 12 cm

V[tex] {\large {\sf {_{kerucut}}}} [/tex] = 3,14 × 25 cm² × 4 cm

V[tex] {\large {\sf {_{kerucut}}}} [/tex] = 78,5 cm² × 4 cm

V[tex] {\large {\sf {_{kerucut}}}} [/tex] = 314 cm³

Jadi, volume kerucut adalah 314 cm³.

16. tentukan luas permukaan dan volume dari bangun kerucut berikut


Jawab: Luas permukaan dan volume untuk kerucut pada gambar di atas adalah:

a. Luas permukaan kerucut = 204.1 cm²; Volume = 200.96 cm³

b. Luas permukaan kerucut = 301.44 cm²; Volume = 301.44 cm²

c. Luas permukaan kerucut = 332.71 cm²; Volume = 376.8 cm³

d. Luas permukaan kerucut = 704 cm²; Volume = 1232 cm²

e. Luas permukaan kerucut = 55.33 cm²; Volume = 21.98 cm³

f. Luas permukaan kerucut = 282.6 cm²; Volume = 314 cm²

Penjelasan dengan langkah-langkah:

Untuk menghitung luas permukaan sebuah kerucut digunakan rumus:

Luas permukaan kerucut = π x r (r + s)

Volume kerucut dihitung dengan rumus:

volume kerucut = ¹/₃ x π r² x t

Untuk mencari tinggi atau garis pelukis kerucut digunakan rumus pythagoras:

sisi miring² =  alas² + tinggi²

dengan sisi miring sama dengan panjang garis pelukis kerucut

a. Garis pelukis² = alas² + tinggi²

Garis pelukis² =  ( 4 cm )² + ( 12 cm )²

Garis pelukis² = 16 cm² + 144 cm²

Garis pelukis = √ ( 160 cm² )

Garis pelukis = 12.65 cm

Luas permukaan kerucut = π x r (r + s)

Luas permukaan kerucut = 3.14 x 4 cm (4 cm + 12.65 cm)

Luas permukaan kerucut = 3.14 x 4 cm ( 16.25 cm )

Luas permukaan kerucut = 204.1 cm²

Volume = ¹/₃ x π r² x t

Volume = ¹/₃ x 3.14 x ( 4 cm )² x 12 cm

Volume = ¹/₃ x 3.14 x 16 cm² x 12 cm

Volume = 200.96 cm³

b. Garis pelukis² = alas² + tinggi²

( 10 cm )² =  ( 6 cm )² + tinggi ²

100 cm² = 36 cm² + tinggi²

tinggi² = 100 cm² - 36 cm²

tinggi = √ ( 64 cm² )

tinggi = 8 cm

Luas permukaan kerucut = π x r (r + s)

Luas permukaan kerucut = 3.14 x 6 cm (6 cm + 10 cm)

Luas permukaan kerucut = 3.14 x 6 cm ( 16 cm )

Luas permukaan kerucut = 301.44 cm²

Volume = ¹/₃ x π r² x t

Volume = ¹/₃ x 3.14 x ( 6 cm )² x 8 cm

Volume = ¹/₃ x 3.14 x 36 cm² x 8 cm

Volume = 301.44 cm²

c. Garis pelukis² = alas² + tinggi²

Garis pelukis² =  ( 6 cm )² + ( 10 cm )²

Garis pelukis² = 36 cm² + 100 cm²

Garis pelukis = √ ( 136 cm² )

Garis pelukis = 11.66 cm

Luas permukaan kerucut = π x r (r + s)

Luas permukaan kerucut = 3.14 x 6 cm (6 cm + 11.66 cm)

Luas permukaan kerucut = 3.14 x 6 cm ( 17.66 cm )

Luas permukaan kerucut = 332.71 cm²

Volume = ¹/₃ x π r² x t

Volume = ¹/₃ x 3.14 x ( 6 cm )² x 10 cm

Volume = ¹/₃ x 3.14 x 36 cm² x 10 cm

Volume = 376.8 cm³

d. Garis pelukis² = alas² + tinggi²

( 25 cm )² =  ( 7 cm )² + tinggi ²

625 cm² = 49 cm² + tinggi²

tinggi² = 625 cm² - 49 cm²

tinggi = √ ( 576 cm² )

tinggi = 24 cm

Luas permukaan kerucut = π x r (r + s)

Luas permukaan kerucut = ²²/₇ x 7 cm (7 cm + 25 cm)

Luas permukaan kerucut = ²²/₇ x 7 cm ( 32 cm )

Luas permukaan kerucut = 704 cm²

Volume = ¹/₃ x π r² x t

Volume = ¹/₃ x ²²/₇ x ( 7 cm )² x 24 cm

Volume = ¹/₃ x ²²/₇ x 49 cm² x 24 cm

Volume = 1232 cm²

e. Garis pelukis² = alas² + tinggi²

( 4 cm )² =  alas² + ( 3 cm )²

16 cm² = alas² + 9 cm²

alas² = 16 cm² - 9 cm²

alas = √ ( 7 cm² )

alas = 2.65 cm

Luas permukaan kerucut = π x r (r + s)

Luas permukaan kerucut = 3.14 x 2.65 cm (2.65 cm + 4 cm)

Luas permukaan kerucut = 3.14 x 2.65 cm ( 6.65 cm )

Luas permukaan kerucut = 55.33 cm²

Volume = ¹/₃ x π r² x t

Volume = ¹/₃ x 3.14 x ( 2.65 cm )² x 3 cm

Volume = ¹/₃ x 3.14 x 7 cm² x 3 cm

Volume = 21.98 cm³

f. Garis pelukis² = alas² + tinggi²

( 13 cm )² =  ( 5 cm )² + tinggi ²

169 cm² = 25 cm² + tinggi²

tinggi² = 169 cm² - 25 cm²

tinggi = √ ( 144 cm² )

tinggi = 12 cm

Luas permukaan kerucut = π x r (r + s)

Luas permukaan kerucut = 3.14 x 5 cm (5 cm + 13 cm)

Luas permukaan kerucut = 3.14 x 5 cm ( 18 cm )

Luas permukaan kerucut = 282.6 cm²

Volume = ¹/₃ x π r² x t

Volume = ¹/₃ x 3.14 x ( 5 cm )² x 12 cm

Volume = ¹/₃ x 3.14 x 25 cm² x 12 cm

Volume = 314 cm²

PembahasanBangun Ruang Sisi Lengkung - Kerucut

Bangun Ruang Sisi Lengkung adalah bangun tiga dimensi yang memiliki bagian berbentuk lengkungan. Adapun tiga bangun ruang sisi lengkung adalah tabung, kerucut, dan bola.

Kerucut memiliki bagian alas dan selimut saja. Contoh benda berbentuk kerucut adalah topi ulang tahun, es krim, dan traffic cone di jalan raya.  

Luas permukaan sebuah kerucut dapat dihitung dengan menggunakan rumus:

L kerucut = L alas + L selimutL kerucut = π r² + π r s

dengan:

π = 22/7 atau 3,14

r = jari-jari alas kerucut

t = tinggi kerucut

s = garis pelukis atau s = √(r² + t²)

Diketahui

Gambar kerucut dan ukurannya di gambar

Ditanya

Luas permukaan kerucut

Penyelesaian

a) r = 5 cm

t = 12 cm

s = 13 cm

Luas permukaan kerucut

= π r² + π r s

= (3,14) (5)² + (3,14) (5) (13)

= 3,14 (25) + 3,14 (65)

= 78,5 + 204,1

= 282,6 cm²

b) d = 12 cm

t = 8 cm

s = 10 cm

Maka r = 1/2 d = 1/2 (12) = 6 cm

Luas permukaan kerucut

= π r² + π r s

= (3,14) (6)² + (3,14) (6) (10)

= 3,14 (36) + 3,14 (60)

= 113,04 + 188,4

= 301,44 cm²

c) r = 8 cm

t = 15 cm

Maka s = √(8²+15²) = √(64+225) = √(289) = 17 cm

Luas permukaan kerucut

= π r² + π r s

= (3,14) (8)² + (3,14) (8) (17)

= 3,14 (64) + 3,14 (136)

= 200,96 + 427,04

= 628 cm²

d) r = 3 cm

s = 5 cm

Luas permukaan kerucut

= π r² + π r s

= (3,14) (3)² + (3,14) (3) (5)

= 3,14 (9) + 3,14 (15)

= 28,26 + 47,1

= 75,36 cm²

e) t = 24 cm

s = 26 cm

Maka r = √(26²-24²) = √(676-576) = √(100) = 10 cm

Luas permukaan kerucut

= π r² + π r s

= (3,14) (10)² + (3,14) (10) (26)

= 3,14 (100) + 3,14 (260)

= 314 + 816,4

= 1130,4 cm²

f) d = 6 cm

t = 4 cm

Maka:

r = 1/2 d = 1/2 (6) = 3 cm

Panjang garis pelukis= 4 cm

Jari-jari²= 16-9=7

Jari-jari= √7

π×r² + π×r×s

π×7 + π×√7×4 = 55.2 cm ²

Volume kerucut=

1/3×π×r²×t

1/3×π×7×3=7π=22 cm ³
