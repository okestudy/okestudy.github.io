Jawaban:

intinya semakin banyak core dan Thread , dan semakin tinggi speednya maka processor akan lebih kencang & mampu menjalankan banyak program

Penjelasan:

Core itu adalah proses menjalankan program , semakin banyak core semakin ringan jika membuka banyak program

Thread itu adalah jalur dari perintah core ke komponen komputer yang lain

Speed adalah kecepatan processornya semakin tinggi speed nya maka performa processor akan maksimal

16. spesifikasi pc server dan operating system(op) server!


pc server harus dg processor yg lebih cepat, karena server di tuntut untuk memproses data yg sangat banyak dg cepat, processor intel xeon E-series.
pc server harus dg memory yg lebih besar, semakin besar semakin cepat dalam memproses data. gunakan ddr4
pc server harus dg penyimpanan yg cepat dan besar.. gunakan ssd untuk penyimpanan data.
pc server harus dg jaringan yg cepat, gunakan lan card bertype gigabit ethernet.
untuk merk dan harga bisa cek di google.

untuk sistem operasi, gunakan yg sudah support virtual machine, biar server dapat bekerja yg maksimal.. bisa pake windows server 2016, linux redhat, linux suse, vmware esxi, proxmox, dan xen. selain dari ini tidak disarankan.
