3. soal soal mtk bentuk aljabar


Contoh soal aljabar :

>> Penjumlahan
1. 3x-3y dan -2y+z
Jawab : 3x + (-3y - 2y) + z
            = 3x - 5y + z

>> Pengurangan
1. -4x³ - x² + 8x dari 2x² - x + 2
Jawab : (2x² - x + 2) - (-4x³ - x² + 8x)
            = 4x³ + (2x² + x²) + (8x - x) + 2
            = 4x³ + 3x² + 7x + 2

>> Perkalian
1. (x² + y)(x + y²)
jawab : (x² × x) + (x² × y²) + (y × x) + (y × y²)
           = x³ + x²y² + xy + y³
           = x³ + y³ + x²y² + xy

>> Pembagian
1. (x² + 3x + 2) ÷ (x + 2)   ------- faktorkan
Jawab : (x + 2) (x + 1) ÷ (x + 2)
           = (x + 1)

5. buat soal tentang :menyederhanakan bentuk aljabar​


Jawaban:

Contoh soal :

Sederhanakanlah bentuk aljabar berikut :

1.( 5x² + 3xy + 2 ) - (3x² - xy - 1)

2. hasil penjumlahan dari (4x - 3x² + 1) dan (x + x² - 3)

3. Hasil pengurangan (7x² + 3x²y - 5) dari (9x² - 4 + 2x²y)

Jawab :

1. Hasil dari :

= (5x² + 3xy + 2) - (3x² - xy - 1)

= 5x² + 3xy + 2 - 3x² + xy + 1

= 5x² - 3x² + 3xy + xy + 2 + 1

= 2x² + 4xy + 3 ✔️

2. Hasil dari :

= (4x - 3x² + 1) + (x + x² - 3)

= 4x + x - 3x² + x² - 3

= 5x - 2x² - 3

atau

= -2x² + 5x - 3

3. Hasil dari :

= (9x² - 4 + 2x²y) - (7x² + 3x²y - 5)

= 9x² - 4 + 2x²y - 7x² - 3x²y + 5

= 9x² - 7x² - 4 + 5 + 2x²y - 3x²y

= 2x² + 1 - x²y

atau

= 2x² - x²y + 1

11. soal dan pembahasan tentang faktorisasi bentuk aljabar


contoh soal:
2x²+12x+10

pembahasan:
2x²+12x+10
-misal a=2, b=12, c=10
-pertama yang harus kita lakukan adalah a x c = 2 x 10 = 20
-nah sekarang kita cari bilangan yang bila dikali menjadi 20, dan bila ditambah menjadi 12(b). kita dapati bilangan 10 dan 2.
-selanjutnya kita ubah 2x²+12x+10 menjadi 2x²+10x+2x+10,
- lalu kita gunakan sifat komutatif, menjadi (2x²+10x)+(2x+10),
-nah sekarang kita sederhanakan menjadi 2x(x+5) + 2(x+5)
-tinggal kita faktorkan menjadi (2x+2)(x+5)
