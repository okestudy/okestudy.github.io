12. diketahui energi ikatan rata - rata untuk :


Diketahui energi ikatan rata :

C - C : 348 kJ/mol

C = C : 641 kJ/mol

C - H : 413 kJ/mol

C - Cl : 328 kJ/mol

H - Cl : 431 kJ/mol

Maka perubahan entalpi reaksi : CH₂ = CH₂ + HCl ⇒ CH₃ - CH₂ - Cl adalah -17 kJ/mol.

Penyelesaian Soal :

Maka untuk memudahkan dalam menghitung, bukalah rantai rantai pada reaksi menjadi :

H    H                                H  H

Ι      Ι                                  Ι    Ι

C = C   +   H - Cl   ⇒  H - C - C - Cl

Ι      Ι                                  Ι     Ι

H    H                                H   H

ΔH reaksi = ΣEI zat pereaksi – ΣEI zat hasil reaksi

                = {(4 × C–H) + C=C + H-Cl} – {(5 × C–H) + C-C + C-Cl}

                = {(4 × 413) + 641 + 431} – {(5 × 413) + 348 + 328}

                = {1652 + 641 + 431} – {2065 + 348 + 328}

                = 2724 kJ/mol – 2741 kJ/mol

                = -17 kJ/mol

∴ Kesimpulan perubahan entalpi reaksi tersebut adalah -17 kJ/mol.

14. diketahui: C2H4(g) -> 2C(g) + 4H(g) deltaH =+2.266 kj dan energi ikatan rata rata C-H = 413 kj ! hitunglah energi ikatan rata rata C=C.


Diketahui : C₂H₄ (g)  ⇒  2C (g) + 4H (g)

ΔH reaksi sebesar +2266 kJ/mol dan energi ikatan rata - rata C - H = 413 kJ/mol. Maka nilai energi ikatan rata - rata C = C adalah sebesar 614 Kj/mol.

Penyelesaian Soal :

Diketahui : reaksi : C₂H₄ (g)  ⇒  2C (g) + 4H (g)

                  ΔH = +2.266 kJ/mol

                  energi C - H = 413 kJ/mol

Ditanya : energi C = C ?

Jawab :

Hitung  energi rata rata C = C menggunakan rumus sebagai berikut :

ΔH reaksi = ΣEI zat pereaksi – ΣEI zat hasil reaksi

+2.266 kJ/mol = {C = C + (4 × C - H)} - 0

+2.266 kJ/mol = {C = C + (4 × 413 kJ/mol)}

+ 2.266 kj/mol = {C = C + 1652 kJ/mol}

C = C = +2266 kJ/mol - 1652 kJ/mol

          = 614 kJ/mol

∴ Kesimpulan nilai energi ikatan rata C = C adalah sebesar 614 Kj/mol.

Penjelasan:

Reaksi : [tex]C_{2}H_{5}[/tex] - C ≡ N + 2[tex]H_{2}[/tex] → [tex]C_{2}H_{5}[/tex] - [tex]CH_{2}[/tex] - [tex]NH_{2}[/tex]

Struktur dari reaksi tersebut adalah:

     H   H                                       H   H    H   H

      Ι     Ι                                         Ι     Ι     Ι     Ι

H - C - C - C ≡ N + 2(H - H) → H - C - C - C - N

      Ι     Ι                                         Ι     Ι     Ι     Ι

     H   H                                        H   H   H   H

ΔH = ∑EI reaktan - ∑EI produk (EI = energi ikatan)

∑EI reaktan = ((2 x EI C-C) + (5 x EI C-H) + (EI C≡N) + (2 x EI H-H))

                  = (160 + 665 + 70 + 188)

                  = 1083 kkal/mol

ΔH = ∑EI reaktan - ∑EI produk

      = 1053 kkal/mol - 1083 kkal/mol

18. Jika energi ikat diketahui rata-rata C-H = 413 kJ/mol dan C2H6 --> 2C + 6H dengan deltaH = -2826 kJ. Tentukan energi ikat rata-rata C-C adalah....kJ/mol


Energi ikatan rata-rata C-C adalah 348 kJ/mol. Energi ikatan rata-rata ini dihitung berdasarkan hukum Hess dari ikatan antaratom. Perubahan entalpi adalah total energi pemutusan dikurangi total energi pembentukan ikatan.

Pembahasan

Diketahui

Energi ikatan rata-rata C-H = 413 kJ/mol

Perubahan entalpi dari pemutusan senyawa C₂H₆ adalah 2.826 kJ (bertanda positif sesuai sumber soal aslinya)

Ditanya

Energi ikatan rata-rata C-C (kita misalkan sebagai x kJ/mol)

Penyelesaian

C₂H₆ termasuk ke dalam jenis alkana dengan deret homolog [tex]\boxed{~C_nH_{2n+2}~}[/tex], berikatan tunggal dan merupakan senyawa hidrokarbon jenuh.

C₂H₆(g) → 2C(g) + 6H(g)     ΔH = 2.826 kJ

Struktur ikatan dapat dilihat pada gambar terlampir. Terdapat 1 ikatan C-C dan 6 ikatan C-H di ruas kiri, sedangkan di ruas kanan tidak terdapat ikatan antaratom sebab atom-atom C dan H sudah terurai.

Hukum Hess terkait energi ikatan rata-rata antaratom adalah sebagai berikut:

[tex]\boxed{~\Delta H=\Sigma energi~pemutusan-\Sigma energi~pembentukan~}[/tex]

ΔH = 6(C-H) + (C-C) - 0

2.826 = 6(413) + x

x = 2.826 - 2.478

x = 348

Diperoleh energi ikatan rata-rata C-C sebesar 348 kJ/mol.

20. Jika diketahui perubahan entalpi reaksi dari :2NH3 --> n2 + 3h-h deltaH = +97 kJ.Energi ikatan rata-rata N=N = 941 kJ; H-H = 436 kJ.energi ikatan rata-rata untuk N-H adalah..​


Diketahui energi ikatan rata-rata : N ≡ N = 941 kJ; H – H = 436 kJ, maka energi ikatan rata-rata untuk N – H jika diketahui perubahan entalpi reaksi dari : 2 NH₃ → N₂ + 3 H₂   ∆H = +97 kJ adalah –136 kJ.

Pembahasan :  

Diketahui : N ≡ N = 941 kJ

                  H – H = 436 kJ

                  Reaksi : 2 NH₃ → N₂ + 3 H₂      ∆H = +97 kJ

Ditanya : ∆H ?  

Penyelesaian soal :  

Reaksi : 2 NH₃ → N₂ + 3 H₂     ∆H = +97 kJ

Jika reaksi dibuka strukturnya, maka menjadi :

  H

   |

2 N − H   +   N ≡ N   →   H – H        ∆H = +97 kJ

   |

  H

ΔH reaksi = Σ energi ikatan pereaksi - Σ energi ikatan hasil reaksi

+97 = {(3 × N–H) + (N≡N)} – {(H–H)}

+97 = {(3 × N–H) + (941)} – {(436)}

+97 + 436 = (3 × N–H) + (941)

533 = (3 × N–H) + (941)

533 – 941 = (3 × N–H)

–408 = (3 × N–H)

(3 × N–H) = –408

N–H = [tex]\frac{-408}{3}[/tex]

N–H = –136 kJ

⇒ Kesimpulan, jiketahui energi ikatan rata-rata : N ≡ N = 941 kJ; H – H = 436 kJ, maka energi ikatan rata-rata untuk N – H jika diketahui perubahan entalpi reaksi dari : 2 NH₃ → N₂ + 3 H₂   ∆H = +97 kJ adalah –136 kJ.

Jawaban :

Energi ikatan rata-rata untuk N-H adalah 391 kJ.

Pembahasan :

Suatu reaksi dapat berjalan dengan adanya pemutusan ikatan dari reaktan dan pembentukan ikatan baru pada produk. Oleh karena itu, nilai perubahan entalpi reaksi dapat dihitung dengan menjumlahkan seluruh nilai energi ikat rata-rata pemutusan ikatan dan nilai energi ikat rata-rata pembentukan ikatan. Namun perlu diketahui bahwa pemutusan ikatan selalu membutuhkan energi (bertanda +) dan pembentukan ikatan melepaskan energi (bertanda -). Pada soal ini, ikatan N-H akan putus dan ikatan N-N serta H-H akan terbentuk. Maka untuk menghitung nilai energi ikat rata-rata untuk N-H dapat menggunakan persamaan berikut.
