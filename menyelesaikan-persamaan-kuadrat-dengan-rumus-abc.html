18. selesaikan persamaan kuadrat berikut dengan rumus abc​

[tex]8 {x}^{2} + 2x - 3 = 0 \\ rumus \: abc \: = \frac{ - b + - \sqrt{b {}^{2} - 4ac } }{2a} \\ = \frac{ - 2 + - \sqrt{2 {}^{2} - 4 \times 8 \times - 3} }{2 \times 8} \\ = \frac{ - 2 + - \sqrt{100} }{16} \\ = \frac{ - 2 + - 10}{16} [/tex]

[tex] \frac{ - 2 + 10}{16} = \frac{8}{16} = \frac{1}{2} [/tex]

[tex] \frac{ - 2 - 10}{16} = \frac{ - 12}{16} = - \frac{3}{4} [/tex]

Jadi, akar-akar persamaan kuadrat tersebut adalah 1/2 dan -3/4.

e. 8x² + 2x - 3 = 0

selesaikan persamaan kuadrat berikut dengan rumus ABC

Rumus ABC adalah salah satu metode untuk menentukan nilai x dari suatu persamaan kuadrat. Perlu diketahui! untuk menentukan nilai x dengan rumus ABC harus dibuat ke bentuk ax² + bx + c = 0

Rumus ABC sebagai berikut :

[tex]x = \frac{ - b \: ± \: \sqrt{b {}^{2} - 4ac } }{2a} [/tex]

Pembahasan :

Diketahui suatu persamaan kuadrat 8x² + 2x - 3 = 0. Persamaan tersebut sudah berbentuk ax² + bx + c = 0,maka dari itu kita tidak perlu lagi mengubahnya. Sekarang, kita tentukan nilai a, b, dan c terlebih dahulu.

8x² + 2x - 3 = 0

a = 8, b = 2 , c = -3

[a angka yang menempati ax² ,b angka yang menempati bx, dan c sebagai konstanta ]

Sekarang kita substitusikan ke rumus kuartik ABC

[tex]x = \frac{ - b \:± \: \sqrt{b {}^{2} - 4ac } }{2a} [/tex]

[tex]x = \frac{ - 2 \: ± \: \sqrt{2 {}^{2} - 4(8)( - 3) } }{2(8)} [/tex]

[tex]x = \frac{ - 2 \: ± \: \sqrt{4 + 96} }{16} [/tex]

[tex]x = \frac{ - 2 \:± \: \sqrt{100} }{16} = \frac{ - 2 \: ± \: 10}{16} [/tex]

[tex]x1 = \frac{ - 2 + 10}{16} atau \: x2 = \frac{ - 2 - 10}{16} [/tex]

[tex]x1 = \frac{1}{2} atau. \: x2 = - \frac{3}{4} [/tex]

Jadi, penyelesaian persamaan tersebut adalah x = 1/2 atau x = -3/4

25. selesaikan persamaan kuadrat 3x² =x dengan menggunakan rumus abc


3x² = x

3x² - x = 0

a = 3; b = -1; c = 0


D = b² - 4 . a . c

D = (-1)² - 4 . 3 . 0

D = 1


x12 = (-b +- √D)/(2 . a)

x12 = (- (-1) +- √1) / (2 . 3)

x12 = (1 +- 1)/6


x1 = (1 + 1)/6

x1 = 2/6

x1 = 1/3


x2 = (1 - 1)/6

x2 = 0/6

x2 = 0


Jadi, x = 0 atau x = 1/3
