23. tentukan nilai logaritma atau X yang memenuhi persamaan atau pertidaksamaan Log(X²+1)² = 2


Nilai x yang memenuhi persamaan log (x² + 1)² = 2 adalah ....
JawabanPendahuluan

Logaritma sering disebut sebagai kebalikan (invers) dari eksponen. Dalam matematika, persamaan logaritma biasanya dinyatakan dalam bentuk [tex]\[{}^a\log b=c\][/tex], dimana a > 0, a ≠ 1, b > 0, a disebut bilangan pokok atau basis, b disebut bilangan numerus (bilangan yang akan dicari nilai logaritmanya), dan c disebut sebagai nilai logaritma. Lebih lanjut, jika bilangan pokok tidak dituliskan, maka bilangan pokok tersebut adalah 10.

Pembahasan

Berdasarkan soal di atas, kita ketahui bahwa a = 10, b = (x² + 1)², dan c = 2.

Dengan demikian, jika kita gunakan rumus [tex]\[{}^a\log b = c \Leftrightarrow b = {a^c}\][/tex], maka kita peroleh hasil sebagai berikut:

log(x² + 1)² = 2

⇔ (x² + 1)² = 10²

⇔ (x² + 1) = 10

⇔ x² + 1 - 10 = 0

⇔ x² - 9 = 0

⇔ (x - 3)(x + 3) = 0

⇔ x = 3 atau x = -3

Kesimpulan

Berdasarkan uraian pada bagian pembahasan di atas, dapat disimpulkan bahwa nilai x yang memenuhi persamaan log (x² + 1)² = 2 adalah x = 3 atau x = -3.
