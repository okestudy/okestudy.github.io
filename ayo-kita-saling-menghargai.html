21. Ayo berlatih! Harga 3 celana dan 2 baju adalah Rp280.000,00. Sedangkan harga 1 celana dan 3 baju di tempat dan model yang sama adalah Rp210.000,00. Harga sebuah celana adalah


Ayo berlatih!

Harga 3 celana dan 2 baju adalah Rp. 280.000,00. Sedangkan harga 1 celana dan 3 baju di tempat dan model yang sama adalah Rp. 210.000,00. Harga sebuah celana adalah....

Pembahasan

Persoalan diatas merupakan salah bentuk penerapan dari materi sistem persamaan linear dua variabel.

Diketahui :

3 Celana + 2 Baju = Rp. 280.000.-

 1 Celana + 3 Baju = Rp. 210.000.-


Ditanyakan Harga Sebuah Celana = ...?

Penyelesaian :

agar lebih mudah menyelesaikan persoalan diatas, perlu kita ubah dalam bentuk persamaan linear dua variabel.

Misalkan :

Celana = x

    Baju = y

Sehingga,

3 Celana + 2 Baju = Rp. 280.000.-

 1 Celana + 3 Baju = Rp. 210.000.-

Menjadi :

 3x + 2y = Rp. 280.000

   x + 3y = Rp. 210.000

Untuk menentukan berapa harga sebuah celana, perlu di - eliminasi harga dari baju yang telah dimisalkan menjadi variabel y

3x + 2y = Rp. 280.000 → Kali 3

   x + 3y = Rp. 210.000 → Kali 2

---------------------------------------------

9x + 6y = Rp. 840.000

2x + 2y = Rp. 420.000

---------------------------------- - (Kurangkan untuk eliminasi nilai y)

        7x = Rp. 420.000

          x = Rp. 420.000 : 7

          x = Rp. 60.000.- [tex]\boxed{\bold{Harga \ sebuah \ Celana}}[/tex]


Jadi, Harga sebuah celana dari persoalan tersebut Rp. 60.000.-
