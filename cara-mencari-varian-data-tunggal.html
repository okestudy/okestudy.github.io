1. cara menghitung deviasi dan varian data tunggal


Dalam statistik, standar deviasi ( simpangan buku ) dan varian adalah ukuran-ukuran variasi data yang sering digunakan. Standar deviasi adalah akar dari varian. Adapun cara menghitung standar deviasi data tunggal adalah sebagai berikut :

Menghitung nilai rata-rata dari data yang ada, kemudian nilai rata-rata = jumlah nilai data dibagi dengan banyak data tersebut.Setelah itu menghitung penyimpangan setiap data dari rata-rata data tersebut, dengan cara mengurangi nilai dari nilai rata-rata.

Cara menghitung varian data tunggal adalah sebagai berikut :

Menghitung nilai rata-rata dari data yang ada, kemudian nilai rata-rata = jumlah nilai data dibagi dengan banyak data tersebut.Setelah itu menghitung penyimpangan setiap data dari rata-rata data tersebut, dengan cara mengurangi nilai dari nilai rata-rata.Terakhir, simpangan setiap nilai data dikuadratkan kemudian bagi dengan nilai rata-rata data. Nilai yang dihasilkan disebut varian.

Pembahasan :

Dalam ilmu statistik terdapat istilah standar deviasi maupun varians dalam mengolah suatu data, baik data tunggal maupun data kelompok. Data didapat karena hasil dari penelitian. Varians dan standar deviasi adalah ukuran kegaraman dalam analisi deskripstif yang biasa digunakan untuk mengetahui bagaimana penyebaran data.

Varians adalah ukuran seberapa jauh penyebaran data dari nilai rata-rata. Sedangkan standar deviasi adalah akar dari varians.

Adapun rumus dari varians data tunggal adalah :

[tex]S^{2} =[/tex] ∑[tex](x_{I} -mean)^2/n[/tex]

rumus dari standar deviasi data tunggal adalah akar dari varians.

[tex]s=\sqrt{s^{2} }[/tex].

Dalam masalah statistika dan probabilitas, varian atau ragam sebuah peubah acak atau variansi merupakan ukuran seberapa jauh suatu kumpulan bilangan atau angka tersebar. Jika terdapat variansi nol, maka hal tersebut menyatakan bahwa semua nilai sama.

Pengertian simpangan baku

Dalam masalah statistika dan probabilitas, standar deviasi atau simpangan baku merupakan ukuran sebaran statistik yang sering digunakan. Atau dengan kata lain, standar deviasi atau simpangan baku mengukur bagaimana nilai-nilai dari suatu data tersebar.  

Rumus mencari variansi

S^2 =  (sigma(x-xbar)^2/(n-1)

Rumus mencari simpangan baku

S =  √(sigma(x-xbar)^2/(n-1)

Rumus x bar (rata-rata)

X bar = data 1 + data 2 + data 3 + …. / banyaknya data  

Keterangan:

S = standar deviasi atau simpangan baku

n = banyaknya data

sigma = jumlah total

15. varian (ragam) dari data 4,5,8,11,12 adalah pake cara jangan jawab asal ​


Varians dari data 4, 5, 8, 11, 12 adalah 10.

Varians merupakan salah satu dari ukuran keragaman (variasi) data statistik yang paling sering digunakan di mana dasar penghitungannya adalah keinginan untuk mengetahui keragaman suatu kelompok data dengan mengurangi setiap nilai data dengan rata - rata kelompok data tersebut dan dikuadratkan, selanjutnya semua hasilnya dijumlahkan dan hasil penjumlahan kuadrat (sum of squares) tersebut akan selalu bernilai positif. Nilai varian diperoleh dari pembagian hasil penjumlahan kuadrat (sum of squares) dengan jumlah data (n) yang dituliskan dengan S² = [tex] \frac{1}{n} [/tex]Σ(xi - x̄)².

Agar lebih jelasnya, simak pembahasan soal berikut.

PEMBAHASAN :

Diketahui beberapa data 4, 5, 8, 11, 12. Dari data - data tersebut, diketahui bahwa ada 5 data sehingga ditulis n = 5.

Sebelum kita menghitung variannya, kita akan hitung rata - rata data tersebut.

x̄ = [tex] \frac{jumlah \: data}{frekuensi \: data} [/tex]

x̄ = [tex] \frac{4 + 5 + 8 + 11 + 12}{5} [/tex]

x̄ = [tex] \frac{40}{5} [/tex]

x̄ = rata - rata data = 8

Kemudian kita hitung varians-nya.

S² = [tex] \frac{1}{n} [/tex]Σ(xi - x̄)²

S² = [tex] \frac{1}{5} [/tex][(4 - 8)² + (5 - 8)² + (8 - 8)² + (11 - 8)² + (12 - 8)²]

S² = [tex] \frac{1}{5} [/tex][16 + 9 + 0 + 9 + 16]

S² = [tex] \frac{1}{5} [/tex] × 50

S² = varians data = 10

            i = 1

dimana x merupakan mean atau rata-rata data, n merupakan banyaknya data, dan xi merupakan datum ke-i.

Varian data dari 5, 7, 5, 8, 5, 5, 4, 7, 6, 5, 7 adalah...

Jawab:
Diketahui data 4, 5, 5, 5, 5, 5, 6, 7, 7, 7, 8.

Banyaknya data (n) = 11.

Mean atau rata-rata adalah

x = [tex] \frac{4\ +\ 5\ +\ 5\ +\ 5\ +\ 5\ +\ 5\ +\ 6\ +\ 7\ +\ 7\ +\ 7\ +\ 8}{11} [/tex]
⇔ x = [tex] \frac{64}{11} [/tex]
⇔ x = 5,82
⇔ x = 6 (pembulatan ke atas).

Varians atau ragam adalah

s² = [tex] \frac{1}{11} [/tex] x [(4 - 6)² + (5 - 6)² + (5 - 6)² + (5 - 6)² + (5 - 6)² + (5 - 6)² + (6 - 6)² + (7 - 6)² + (7 - 6)² + (7 - 6)² + (8 - 6)²]
⇔ s² = [tex] \frac{1}{11} [/tex] x [(-2)² + (-1)² + (-1)² + (-1)² + (-1)² + (-1)² + 0² + 1² + 1² + 2²]
⇔ s² = [tex] \frac{1}{11} [/tex] x [4 + 1 + 1 + 1 + 1 + 1 + 0 + 1 + 1 + 4]
⇔ s² = [tex] \frac{1}{11} [/tex] x [15]
⇔ s² = 1,36
⇔ s² = 1,4 (pembulatan ke atas).

Jadi, varians atau ragam dari data 5, 7, 5, 8, 5, 5, 4, 7, 6, 5, 7 adalah 1,4.

28. varian dari data 3,5,6,7,9,9,10,11,12 adalah


Varian dari data 3, 5, 6, 7, 9, 9, 10, 11, 12 adalah [tex]\frac{70}{9} = 7\frac{7}{9}[/tex] = 7,78. Varians adalah salah satu ukuran sebaran dalam statistika. Nama lain dari varians adalah ragam. Rata-rata adalah perbandingan antara jumlah data dengan banyaknya data.  

Rumus rata-rata

[tex]\bar{x} = \frac{jumlah \: data}{banyaknya \: data} = \frac{\sum x_{i}}{n}[/tex]

Rumus Ragam (Varians)

R = [tex] \frac{\sum (x_{i} - \bar{x})^{2}}{n}[/tex]

Pembahasan

Rata-rata dari 3, 5, 6, 7, 9, 9, 10, 11, 12 adalah

[tex]\bar{x} = \frac{3 + 5 + 6 + 7 + 9 + 9 + 10 + 11 + 12}{9} [/tex]

[tex]\bar{x} = \frac{72}{9} [/tex]

[tex]\bar{x}[/tex] = 8

Sehingga nilai varians dari 3, 5, 6, 7, 9, 9, 10, 11, 12 adalah

R = [tex]\frac{(3 - 8)^{2} + (5 - 8)^{2} + (6 - 8)^{2} + (7 - 8)^{2} + (9 - 8)^{2} + (9 - 8)^{2} + (10 - 8)^{2} + (11 - 8)^{2} + (12 - 8)^{2}}{9} [/tex]

R = [tex]\frac{(-5)^{2} + (-3)^{2} + (-2)^{2} + (-1)^{2} + 1^{2} + 1^{2} + 2^{2} + 3^{2} + 4^{2}}{9} [/tex]

R = [tex]\frac{25 + 9 + 4 + 1 + 1 + 1 + 4 + 9 + 16}{9} [/tex]

R = [tex]\frac{70}{9} [/tex]

R = [tex]7\frac{7}{9} [/tex]

R = 7,78

29. varian dari data 2,4,4,5,6,6,7,8,9,9

Pendahuluan

Varian sangat erat kaitannya dengan standar deviasi (simpangan baku). Varian sendiri diartikan sebagai ukuran keragaman atau variasi data statistik yang paling sering digunakan.

Perhitungan varian biasanya didasari oleh keinginan untuk mengetahui keberagaman suatu kelompok data. Salah satu cara untuk mengetahui keragaman dari suatu kelompok data adalah dengan mengurangi setiap nilai data dengan rata - rata dari kelompok tersebut, setelah itu semua hasil dijumlahkan. Akan tetapi cara seperti ini tidak selalu bisa digunakan karena hasilnya akan menjadi 0. Oleh karena itu, solusi agar nilai varian tidak menjadi 0 adalah dengan mengkuadratkan setiap pengurangan nilai data dan rata - rata kelompok data tersebut, dilanjutkan dengan penjumlahan. Hasil penjumlahan kuadrat tersebut akan selalu bersifat positif.
