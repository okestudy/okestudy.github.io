1. 1. Nilai dari tan 15° tan 60° tan 75° adalah ...A. 1B. akar 2C. akar 3D. 2E. 3​


Jawaban:

Tan 15° x tan 60° x tan 75°

= Tan ( 90 - 75° ) x √3 x tan 75°

Cot 75° x √3 x tan 75°

Cot (x) = 1 / tan x

1 / tan 75° x √3 x tan 75°

= 1 x √3

= √3 (C)

10. Jika tan x = 1/2 akar 3 maka tan ( x - 30° )


Materi : Trigonometri

Pembahasan :
Tan (x - 30°)
= (Tan x - Tan 30°)/(1 + Tan x . Tan 30°)
= (1/2 √3 - 1/3 √3)/(1 + 1/2 √3 . 1/3 √3)
= (3/6 √3 - 2/6 √3)/(1 + 1/2)
= (1/6 √3)/(3/2)
= (1/6 √3) . 2/3
= 1/9 √3

28. 4. Tentukan Himpunan penyelesaian persamaan 2 tan (x-phi/3)/1-tan^2(x-phi/3)=akar 3


persamaan trigonometri

2 tan (x - π/3) / (1 - tan² (x - π/3)) = √3

tan 2(x - π/3) = √3

2(x - π/3) = π/3 + k.π

2x = 2π/3 + π/3 + k.π

2x = π + k.π

x = π/2 + k.π/2

k bilangan bulat

0 ≤ x ≤ π

k = -1 → x = 0

k = 0 → x = π/2

k = 1 → x = π

dan seterusnya

HP = {0 , π/2 , π}
