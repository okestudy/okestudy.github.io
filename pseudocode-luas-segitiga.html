1. Buat lah pseudocode untuk menghitung luas segitiga


jawab

berikut pseudocode utk ngitung luas segitiga:

1. masukkan nilai alas segitiga (alas)

2. masukkan nilai tinggi segitiga (tinggi)

3. hitung luas segitiga dengan rumus: luas = 0.5 * alas * tinggi

4. tampilkan nilai luas segitiga

nt : Pseudocode yaitu bentuk abstraksi dari sebuah algoritma yg disajikan dalam bahasa yg lebih mudah dipahami oleh manusia.

29. TUGAS MENULISKAN NOTASI ALGORITMA DALAM BENTUK PSEUDOCODE1. Menghitung luas segitiga2. Menghitung keliling persegi panjang jika diketahui panjang dan luas​


Jawaban:

program menghitung_luas_segitiga

deklarasi

var alas : integer

var tinggi : integer

var diketahui : array { alas, tinggi }

algoritma:

INPUT diketahui

IF (diketahui INCLUDES alas AND diketahui INCLUDES tinggi) then

output Math: ½ * alas * tinggi

ELSE then

output "Tolong masukkan alas dan tinggi segitiga!"

program menghitung_keliling_persegi

deklarasi

var panjang : integer

var lebar : integer

var diketahui : array { panjang, lebar }

algoritma:

INPUT diketahui

IF (diketahui INCLUDES panjang AND diketahui INCLUDES lebar) then

output Math: 2 * (panjang + lebar)

ELSE then

output "Tolong masukkan panjang dan lebar persegi panjang!"

Pseudocode merupakan kode yang mengilustrasikan suatu program dengan Bahasa yang mudah dipahami, jadi ini hanya sebagai contoh cara kerja kodenya nanti dan apa yang dilakukan kodenya. Karena biasanya Algoritma berupa kalimat deskriptif lebih rumit dipahami maka pseudocode lebih banyak dipakai dalam membuat algoritma yang memudahkan programmer untuk merealisasikan algoritmanya menjadi kode.

Jadi, kode diatas ini bukan kode yang work atau bekerja di suatu engine atau machine yaa, ini hanya ilustrasi suatu algoritma kode.
