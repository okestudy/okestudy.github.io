1. suatu persegi panjang diketahui lebarnya adalah dua kali panjangnya. jika diketahui luas persegi panjang 128cm2 maka keliling persegi panjang tersebut adalah


Pembahasan.

Diketahui

l = 2p
L = 128 cm²

Masukan ke rumus luas

p . l = L
2l . l = 128
2l² = 128
l² = 64
l = ± √ 64
l = ± 8

Didapat
l.1 = -8
l.2 = 8

Ambil yang positif yaitu 8

l = 8
l = 2p
8 = 2p
p = 4


Masukan dalam rumus keliling

2p + 2l = K
2(4) + 2(8) = K
8 + 16 = K
K = 24 cm

Jadi kelilingnya sebesar 24 cm

8. diketahui keliling persegi panjang 72cm agar luas daerah persegi panjang tidak lebih dari 315cm² panjang persegi panjang adalah


K = 72 cm
K = 2( p + l )
72/2 = p + l
36 = p + l
maka P = l - 36

L = p x l
315 = ( l - 36 ) x l
315 = l^2 - 36 l
l^2 - 36 l - 315 = 0

akar kuadrat persamaan
l^2 - 36 l - 315 = 0
( x - 21 ) ( x - 15 ) = 0
x1 = 21 dan x2 = 15

jadi dpt ditentukan bahwa :
p = 21 cm
l = 15 cm

uji :

K = 2 ( p + l )
K = 2 ( 21 + 15 )
K = 2 x 36
K = 72 cm

L = p x l
L = 21 x 15
L = 315 cm^2
