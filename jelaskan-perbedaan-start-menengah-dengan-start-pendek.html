3. jelaskan perbedaan start menengah dengan start pendek!


START PENDEK (BUNCH START)

Start pendek adalah lutut kaki belakang (kaki kiri) diletakkan didepan ujung kaki depan (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.


START MENENGAH (MEDIUM START)

Start menengah adalah lutut kaki belakang (kaki kiri) diletakkan sejajar dengan ujung kaki depan (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.

Perbedaannya hanya terletak pada posisi lutut dengan telapak kaki.
Apabila start pendek maka posisi lutut sejajar dengan jari-jari kaki
Apabila start menengah maka posisi lutut sejajar dengan tengah telapak kaki

4. Tugas jelaskan cara melakukan start pendek ,(bunch start) start menengah (medium start ) dan start panjang (long start ) ​


Berikut adalah cara pelaksanaan dari short start, medium start dan long start untuk cabang lari jarak pendek.

Short Start :

Kaki kanan di depan, kaki kiri di belakang. Jari-jari kaki belakang segaris dengan tumit kaki depan

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Medium Start :

Kaki kanan di depan, kaki kiri di belakang. Lutut segaris dengan ujung jari

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Long Start :

Kaki kanan di depan, kaki kiri di belakang. Lutut segaris dengan tumit

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

18. jelaskan pengertian start pendek, menengah dan start panjang! ​


Start Pendek (Bunch Start)

Start pendek ini dilakukan dengan cara berjongkok, dengan kaki kiri di depan dan lutut kaki kanan diletakkan sejajar di sebelah kaki kiri, beri jarak sekitar satu kepal. Jari-jari tangan rapat dan ibu jari terpisah, keduanya diletakkan di belakang garis start.

Start menengah (medium start)

Lutut kaki kanan menempel di tanah. Kaki kiri berada di samping lutut kaki kanan dengan jarak sekitar satu kepal. Kedua tangan menempel di atas garis start dengan membentuk huruf "V" terbalik. Pandangan rileks ke depan, konsentrasi pada aba-aba start berikutnya

Start panjang (long start)

Lutut kaki kanan menempel di tanah. Kaki kiri berada di depan dengan posisi jinjit. Letak ujung kaki belakang segaris dengan tumit kaki depan adalah sikap start jongkok panjang. Kedua tangan menempel di atas garis start dengan membentuk huruf "V" terbalik

30. perbedaan start jongkok pendek menengah panjang


Jawaban:

Start Pendek/Short Start/Bunch Start

Start pendek memiliki posisi jongkok yang bagian lutut belakangnya berada di samping tumit atau depan telapak kaki.

Start Medium/menengah

Start medium di awali dengan posisi badan berjongkok. Kemudian lutut kaki belakang letaknya di samping lekukan telapak kaki bagian depan. Kemudian setelah posisi badan berdiri, ujung kaki belakang ditempatkan kebelakang tumit kaki bagian depan sedikit.

Start Panjang

Start panjang diawali dengan lutut kaki bagian belakang diposisikan 30 cm dari kaki depan maupun menyesuaikan. Yang terpenting dalam jenis start ini ialah letak lutut kaki bagian belakang harus berada di belakang kaki depan.
