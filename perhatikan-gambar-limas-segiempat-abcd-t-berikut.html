tinggi = 4 cm

rusuk alas = 8 cm

OC = 1/2 . AC

OC = 1/2 . r√2

OC = 1/2 . 8√2 cm

OC = 4√2 cm

Jarak titik T ke titik C

TC = √(TO² + OC²)

TC = √(4² + (4√2)²)

TC = √(16 + 32)

TC = √48

TC = 4√3 cm

21. perhatikan gambar segiempat ABCD berikut panjang CD adalah


Jawab:

Cari dulu panjang BD dengan aturan cosinus

sisi:

DB = a

AD = b

AB = d

a² = b² + d² - 2bd cos A

a² = (6)² + (6√2)² - 2 (6) (6√2) cos 45°

a² = 36 + 72 - 72√2 ([tex]\frac{1}{2}[/tex]√2)

a² = 98 - 72

a² = 36 cm

a = √36 cm

DB = 6 cm

Cari panjang CD dengan aturan sinus

[tex]\frac{sinB}{sinC} =\frac{CD}{DB}[/tex]

[tex]\frac{sin90}{sin60} =\frac{CD}{6cm}[/tex]

[tex]\frac{1}{\frac{1}{2} \sqrt{3} } =\frac{CD}{6cm}[/tex]

[tex]\frac{1}{2} \sqrt{3}[/tex] CD = 6 cm

CD = [tex]\frac{6}{\frac{\sqrt{3} }{2} }[/tex] cm

CD = 6 x [tex]\frac{2}{\sqrt{3} }[/tex] cm (dikali [tex]\frac{\sqrt{3} }{\sqrt{3} }[/tex] )

CD = [tex]\frac{12\sqrt{3} }{3}[/tex] cm

CD = 4√3 cm

Jadi, panjang CD adalah 4√3 cm
