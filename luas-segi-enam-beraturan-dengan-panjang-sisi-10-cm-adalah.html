1. Luas segi enam beraturan dengan panjang sisi 10 cm adalah ?


Jawab:

Luas segi enam beraturan jika panjang sisinya 10 cm adalah 150√3 cm².

PEMBAHASAN

Segi enam dibagi enam maka akan terbentuk 6 segitiga sama sisi yang ketiga sisinya sama panjang yaitu 10 cm.

Terlebih dahulu kita mencari tinggi dari dari segitiga tersebut dengan rumus Pythagoras :

t² = 10² - 5²

t² = 100 - 25

t² = 75

t = √75

t = √25√3

t = 5√3

Maka luas segienam tersebut :

L = 6 x luas segitiga

L = 6 x a x t/2

L = 6 x 10 x 5√3/2

L = 6 x 25√3

L = 150√3 cm².

23. Sebuah prisma mempunyai alas berbentuk segi enam beraturan dengan panjang sisi 8 cm dan tinggi 10 cm.hitunglah volmue dan luas permukaan prisma tersebut.


Bangun prisma adalah bangun ruang yang memiliki alas berbentuk segi-n. Rumus volume nya adalah :

Volume prisma = luas alas x tinggi

Luas alas prisma tergantung dengan bentuk alas nya.

=> Alas prisma jika berbentuk belah ketupat :

    Luas alas = [tex] \frac {1} {2}[/tex] x diagonal 1 x diagonal 2

=> Alas prisma jika berbentuk segitiga :

    Luas alas = [tex] \frac {1} {2}[/tex] x alas x tinggi

=> Alas prisma jika berbentuk segienam :

    Luas alas = [tex] \frac {1} {2}[/tex] x 3√3 x s², dimana s = sisi segienam

Sedangkan untuk rumus luas permukaan prisma adalah :

Luas permukaan prisma = 2 x luas alas + jumlah luas sisi-sisi tegak prisma.

Pembahasan

Luas alas prisma segienam adalah :

= [tex] \frac {1} {2}[/tex] x 3√3 x s²

= ( [tex] \frac {1} {2}[/tex] x 3√3 x 8² ) cm²

= ( [tex] \frac {1} {2}[/tex] x 3√3 x 64 ) cm²

= ( [tex] \frac {1} {2 : 2}[/tex] x 3√3 x ( 64 : 2 ) ) cm²

= ( [tex] \frac {1} {1}[/tex] x 3√3 x 32 ) cm²

= 96√3 cm²

Luas sisi tegak prisma seluruhnya adalah :

= 6 x ( 8 cm x 10 cm )

= 6 x 80 cm²

= 480 cm²

Luas prisma seluruhnya adalah :

= ( 2 x luas alas ) + luas seluruh sisi tegak  

= ( 2 x 96√3 cm² ) + 480 cm²

= 192√3 cm² + 480 cm²

= ( 192√3 + 480 ) cm²

Volume prisma adalah :

= luas alas x tinggi

= 96√3 cm²  x 10 cm

= 960√3 cm³

Sehingga diperoleh bahwa luas alas limas dan volume limas adalah ( 192√3 + 480 ) cm² dan 960√3 cm³
