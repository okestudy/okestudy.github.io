1. cara menentukan kuartil data kelompok


Qi = bi + l [((i/4)N - F)/f]

Qi : kuartil ke-i
bi : tepi bawah kelas kuartil ke i
N : banyaknya data
F : frekuensi kumulatif kelas sebelum kelas kuartil
l : lebar kelas
f : frekuensi kelas kuartil

9. Cara menentukan kuartil


Kuartil suatu data diperoleh dengan membagi suatu data terurut menjadi empat bagian sama besar. Kuartil terdiri atas tiga macam, yaitu kuartil pertama atau kuartil bawah (Q1), kuartil kedua atau kuartil tengah atau median (Q2), dan kuartil ketiga atau kuartil atas (Q3). 
Langkah pertama yang Anda harus lakukan untuk menentukan nilai-nilai kuartil dari suatu data yaitumengurutkan data tersebut dari datum terkecil ke datum terbesar. Setelah data diurutkan, maka tentukan median dari data tersebut. Dimana median tersebut merupakan kuartil tengah (Q2).

13. Cari median, kuartil atas, dan kuartil bawah dari data kelompok berikut!


Nilai median, kuartil atas dan kuartil bawah berturut-turut adalah 23.3, 38.42 dan 28.9375. Soal ini berkaitan dengan materi statistika data berkelompok.

Penjelasan dengan langkah-langkah

Diketahui :

Terlampir pada gambar soal

Ditanyakan :

Terlampir pada gambar soal

Jawab :

Mencari median

Rumus median data berkelompok

Me = Tb + [(n/2 - fkk)/fi].C

Dengan

Tb = batas bawah kelas median

n = frekuensi total

fkk = frekuensi kumulatif sebelum kelas median

fi = frekuensi kelas median

C = Panjang kelas

Kita ketahui bahwa n = 30, maka median terletak di data ke 15 dan 16

Perhatikan bahwa

xii = 20.5

fkii = 10

fi = 9

C = 5

Maka mediannya adalah

Me = 20.5 + [(30/2 - 10)/9].5

     = 20.5 + [5/9].5

     = 20.5 + 2.8

     = 23.3

Mencari kuartil atas

Rumus kuartil atas data berkelompok

Q3 = Tb + [((3/4)n – fkk)/fQ3].C

Dengan

Tb = batas bawah kelas

fkk = frekuensi kumulatif sebelum kelas kuartil

C = panjang kelas

Letak kuartil atas yaitu di antara (3/4).30 = 22.5 dan (3/4).30 + 1 = 23.5 yaitu di kelas interval 36-40,

Dan kita ketahui bahwa

Tb = 35.5

fkk = 19

fQ3 = 6

C = 5

maka nilai Q3

Q3 = 35.5 + [((3/4)n – fkk)/fQ3].C

     = 35.5 + [((3/4)30 – 19)/6].5

     = 35.5 + [(22.5 – 19)/6].5

     = 35.5 + [3.5/6].5

     = 35.5 + 2.92

     = 38.42

Mencari kuartil bawah

Rumus kuartil bawah data berkelompok

Q1 = Tb + [((1/4)n – fkk)/fQ1].C

Dengan

Tb = batas bawah kelas

fkk = frekuensi kumulatif sebelum kelas kuartil

C = panjang kelas

Letak kuartil bawah yaitu di antara (1/4).30 = 7.5 dan (1/4).30 + 1 = 8.5 yaitu di kelas interval 26-30,

Dan kita ketahui bahwa

Tb = 25.5

fkk = 2

fQ1 = 8

C = 5

maka nilai Q1

Q1 = 25.5 + [((1/4)n – fkk)/fQ1].C

     = 25.5 + [((1/4)30 – 2)/8].5

     = 25.5 + [(7.5 – 2)/8].5

     = 25.5 + [5.5/8].5

     = 25.5 + 3.4375

     = 28.9375

contoh: jumlah suku data=24
25%=100*i/(24+1)

tinggal dicari kuartilnya data ke 6 & 7
25 percentile= data ke-6 + (data ke-7 dikurangi data ke-6)*0.25
