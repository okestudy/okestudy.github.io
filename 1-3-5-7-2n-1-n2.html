7. Buktikan bahwa : 1+3+5+...+ (2n-1) =n2


Pernyataan 1 + 3 + 5 + (2n - 1) = n² adalah terbukti benar. Hal ini dibuktikan bahwa pernyataan bernilai benar untuk n = 1 dan pernyataan terbukti benar untuk n = k + 1 jika pernyataan benar untuk n = k.

Pembahasan

PEMBUKTIAN DENGAN PRINSIP INDUKSI MATEMATIKA

Prinsip induksi matematika atau PIM dapat digunakan untuk membuktikan pernyataan matematis.

Prinsip ini memiliki tiga tahapan:

Membuktikan pernyataan benar untuk suku pertama. Suku pertama tidak harus bernilai satu. kita harus menyesuaikan dengan syarat pada pernyataan. Misalnya syarat n ≥ 3, maka suku pertamanya 3. Mengasumsi bahwa pernyataan bernilai benar untuk suku ke-k. Menguji kebenaran pernyataan untuk suku ke-(k+1) menggunakan pernyataan pada langkah kedua.

Diketahui:

1 + 3 + 5 + ... + (2n - 1) = n²

Ditanyakan:

Pembuktian pernyataan?

Penjelasan:

Langkah ke-1

Karena pada pernyataan tidak disertai syarat, maka n = 1.

Untuk n = 1

2n - 1 = (2 × 1) - 1 = 2 - 1 = 1n² = 1² = 12n - 1 = n²

Pernyataan terbukti benar untuk n = 1.

Langkah ke-2

Asumsikan pernyataan benar untuk n = k, maka

1 + 3 + 5 + ... + (2k - 1) = k²

Langkah ke-3

Untuk n = k + 1

Akan dibuktikan bahwa

1 + 3 + 5 + (2k - 1) + (2(k + 1) - 1) = (k + 1)²

Hitung dari ruas kiri.

1 + 3 + 5 + (2k - 1) + (2(k + 1) - 1)

Subtitusi dari langkah kedua.

= k² + 2 (k + 1) - 1

= k² + 2k + 2 - 1

= k² + 2k + 1

= (k² + k) + (k + 1)

= k (k + 1) + 1 (k + 1)

= (k + 1) (k + 1)

= (k + 1)²

Terbukti 1 + 3 + 5 + (2k - 1) + (2(k + 1) - 1) = (k + 1)²

Dari ketiga langkah maka dapat dibuktikan bahwa pernyataan 1 + 3 + 5 + (2n - 1) = n² adalah terbukti benar.

8. dengan induksi matematika, buktikan bahwa 1+3+5+7+...+(2n-1)=n2berlaku untuk setiap n bilangan asli


Buktikan bahwa 1 + 3 + 5 + 7 + ... + (2n - 1 )= n²

Pendahuluan

Pembuktian rumus atau sifat P (n) menggunakan induksi matematika dengan langkah :

Buktikan P (n) benar untuk n = 1Asumsikan P (n) benar untuk n = k, lalu buktikan P (n) benar untuk n = k + 1. Jika P (k) benar, maka P (k + 1) benar untuk k ≥ n

Jadi, dari (1) dan (2), pernyataan tersebut benar untuk semua bilangan asli n.

Pelajari lebih lanjut : Buktikan bahwa 1 + 2 + 3 + ... + n = ⁿ/₂ (n + 1) → https://brainly.co.id/tugas/25892828

Pembahasan

Buktikan 1 + 3 + 5 + 7 + ...+ (2n - 1) = n²

Misalkan P (n) adalah 1 + 3 + 5 + 7 + ...+ (2n - 1) = n²

Langkah I

Akan dibuktikan P (n) benar untuk n = 1. Dengan mensubtitusikan n = 1 ke dua ruas diperoleh :

P (n) = n²   ⇔ 2n - 1 = n²

untuk n = 1 ⇒ 2(1) - 1 = 1²

                 ⇔         1 = 1

                 ⇔ ruas kiri = ruas kanan

Oleh karena ruas kiri = ruas kanan, maka p (n) benar untuk n = 1

Langkah II

Andaikan P (n) benar untuk n = k, yaitu

1 + 3 + 5 + 7 + ...+ (2k - 1) = k² bernilai benar

Akan dibuktikan P (n) juga benar untuk n = k + 1, yaitu

Ruas kiri

1 + 3 + 5 + 7 + ... + (2k - 1) + (2 (k + 1) - 1))

= k² + (2 (k + 1) - 1)

= k² + 2k + 2 - 1

= k² + 2k + 1

Ruas kanan

1 + 3 + 5 + 7 + ...+ (2 (k + 1) - 1)

= n²

= (k + 1)²

= k² + 2k + 1

Oleh karena ruas kiri = ruas kanan, maka TERBUKTI bernilai benar untuk n bilangan asli.

23. Tunjukkan bahwa pernyataan berikut bernilai benar 1 + 3+5+7+.....+(2n-1)=n2 Tolong bantu


Jawaban:

Buktikan dengan induksi matematika bahwa 1 + 3 + 5 + 7 + ... + (2n – 1) = n² berlaku untuk setiap n bilangan asli. Untuk pembuktian suatu rumus tersebut benar (berlaku), bisa kita gunakan induksi matematika, yang terdiri dari dua langkay yaitu:

Buktikan untuk n = 1 benar

Misal untuk n = k benar, akan dibuktikan untuk n = (k + 1) juga benar

Pembahasan

1 + 3 + 5 + 7 + ... + (2n – 1) = n²

Langkah pertama  

Akan dibuktikan untuk n = 1 Benar

(2n – 1) = n²

2(1) – 1 = 1²

2 – 1 = 1

1 = 1 (benar)

Langkah kedua

Misal untuk n = k benar

1 + 3 + 5 + 7 + ... + (2k – 1) = k²

Akan dibuktikan untuk n = (k + 1) juga benar

1 + 3 + 5 + 7 + .... + (2k – 1) + (2(k + 1) – 1) = (k + 1)²

|__________________|  

                     k²                  + (2(k + 1) – 1) = (k + 1)²

                                         k² + 2k + 2 – 1 = (k + 1)²

                                               k² + 2k + 1 = (k + 1)²

                                                     (k + 1)² = (k + 1)²

                                                          (Benar)

Jadi TERBUKTI bahwa 1 + 3 + 5 + 7 + ... + (2n – 1) = n² berlaku untuk setiap n bilangan asli

Kata Kunci : Buktikan dengan induksi matematika bahwa 1 + 3 + 5 + 7 + ... + (2n – 1) = n² berlaku untuk setiap n bilangan asli
