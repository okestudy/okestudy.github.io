22. Arti dari on sunday morning,i usually go to the park with my family


Arti dari "on sunday morning,i usually go to the park with my family" adalah "Pada minggu pagi, Saya biasanya pergi ke taman bersama keluargaku"

PEMBAHASAN

Perintah soal adalah mengartikan kalimat “on sunday morning, I usually go to the park with my family”

Artinya: Pada minggu pagi, Saya biasanya pergi ke taman bersama keluargakuKeterangan:On => Pada (digunakan untuk menyatakan hari)Sunday => minggu Morning => pagiI => biasanyaUsually => Biasanya Go => pergiTo => kePark => tamanWith => bersamaMy family => keluargaku

30. apa artinya usually. berikan contohnya ​


Biasanya

I usually go to the school at 7 am.

Usually = Biasanya

Catatan:

Usually adalah Keterangan yg digunakan dalam Present Tense untuk menunjukkan suatu kegiatan Habitual atau rutinitas.

Contoh:

(+) S + V1 (es/s) + O/C

= I usually go to school early

(Saya biasanya pergi ke sekolah lebih awal)
