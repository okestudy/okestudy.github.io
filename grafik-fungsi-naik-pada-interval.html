13. fungsi f(x)=x^3-6x^2+9x-7 naik pada interval... tolong gambarkan grafik interval naiknya


Fungsi merupakan suatu relasi yang menghubungkan setiap anggota domain tepat satu pada kodomain. Berdasarkan soal, dapat disimpulkan bahwa fungsi f(x) = [tex]x^{3}-6x^{2} +9x-7[/tex] naik pada interval x < 1 dan x > 3. Gambaran grafik interval naiknya pada x < 1 dan x > 3.

Penjelasan dengan langkah-langkah:

Diketahui:

Fungsi f(x) = [tex]x^{3}-6x^{2} +9x-7[/tex].

Ditanyakan:

Tentukan fungsi f(x) = [tex]x^{3}-6x^{2} +9x-7[/tex] akan naik pada interval!

Jawab:

Ketika sedang mempelajari matematika maka kita akan belajar tentang fungsi. Berdasarkan soal, fungsi f(x) = [tex]x^{3}-6x^{2} +9x-7[/tex] akan naik pada interval:

f(x) = [tex]x^{3}-6x^{2} +9x-7[/tex]

f'(x) = 3x² - 12x + 9 = 0 (dibagi 3)

x² - 4x + 3 = 0

(x -3) (x-1) = 0

x = 3 atau x = 1

Dengan demikian, naik pada interval x < 1 dan x > 3. Gambaran grafik interval naiknya pada x < 1 dan x > 3.

21. Grafik fungsi y=sin x akan naik pada interval


Penjelasan dengan langkah-langkah:

y = sin x

• fungsi naik ➡ f'(x) > 0

• fungsi turun ➡ f'(x) < 0

f'(x) = 0

cos x = 0

x = 90° dan 270°

Maka, diperoleh 3 daerah untuk interval 0° < x < 360°, yaitu 0° < x < 90°,

90° < x < 270°, dan 270° < x < 360°

kita uji satu - satu untuk setiap daerahnya.

• uji daerah 0° < x < 90°, untuk x = 60°

f'(60°) = cos (60°) = ½

karena nilainya f'(x) > 0, berarti merupakan daerah positif.

• uji daerah 90° < x < 270°, untuk x = 120°

f'(120°) = cos (120°) = -½

karena nilainya f'(x) < 0, berarti merupakan daerah negatif.

• uji daerah 270° < x < 360°, untuk x = 300°

f'(300°) = cos (300°) = ½

karena nilainya f'(x) > 0, berarti merupakan daerah positif.

Maka, fungsi akan naik pada interval

0° < x < 90° dan 270° < x < 360°, dan turun pada interval 90° < x < 270°.
