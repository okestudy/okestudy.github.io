7. Ayo berlatih! Batang korek api disusun dengan dengan susunan sebagai berikut. Pola 1 terdapat 6 batang korek api, pola 2 terdapat 9 korek api dan pola 3 terdapat 12 korek api. Jika pola tersebut terus berlanjut, banyak batang korek api pada susunan ke-10 adalah ... batang.


Pembahasan

Batang korek api disusun dengan dengan susunan sebagai berikut.

Pola 1 terdapat 6 batang korek api,

Pola 2 terdapat 9 korek api dan

Pola 3 terdapat 12 korek api.

Jika pola tersebut terus berlanjut, banyak batang korek api pada susunan ke-10 adalah ... batang...

Penyelesaian :

Persoalan diatas merupakan materi yang berkaitan pada Barisan dan Deret Bilangan.

Bilangan yang berada pada suatu barisan dinyatakan dengan nama "SUKU". secara umum suku-suku pada barisan dapat dituliskan sebagai berikut :

U₁ , U₂ , U₃ , U₄ , .... , Un

Dimana : U₁ → nilai suku pertama

               U₂ → nilai suku kedua

               U₃ → nilai suku ketiga

               U₄ → nilai suku keempat

               Un → nilai suku ke - n

Pembahasan Lebih lanjut :

Suatu barisan yang memiliki selisih atau beda antar suku yang berdekatan ditentukan dengan rumus sebagai berikut :

 [tex]\boxed{\bold{U_n = a + (n - 1)b}}[/tex]

Dengan : Un = Nilai suku ke - n

                  a = Nilai suku pertama barisan

                  n = bilangan ke - n

                  b = beda suku barisan


Balik Pada soal :

Pola 1 = 6 batang korek api

Pola 2 = 9 batang korek api

Pola 3 = 12 batang korek api

Jika diurutkan pada pola barisan, maka tiap pola tersebut dituliskan sebagai berikut :

U₁ , U₂ , U₃ = 6 , 9 , 12

Persoalan diatas menanyakan banyak susunan batang korek api pada pola ke -10 yang artinya mencari nilai suku ke - 10 → U₁₀

6 , 9 , 12

Nilai suku pertama ( U₁ ) = 6

       Beda tiap suku ( b ) = U₂ - U₁ atau U₃ - U₂

                                         = 9 - 6 atau 12 - 9

                                         = 3

Sehingga, dengan telah diketahui nilai suku pertama dan beda tiap suku yang berdekatan, maka banyak batang korek api pada pola ke - 10 adalah :

Un = a + (n - 1)b → subtitusi nilai yang telah diketahui

U₁₀ = 6 + (10 - 1)(3)

       = 6 + 9(3)

       = 6 + 27

       = 33 Batang korek api

Jadi, Banyak batang korek api pada susunan ke - 10 adalah 33 batang.
