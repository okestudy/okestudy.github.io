Berikut adalah beberapa kosa kata tentang komputer dan artinya dalam bahasa Indonesia:

app : aplikasi, program, atau software.backup : salinan file dari hard disk yang dibuat jika file hard disk terhapus atau rusak.bit : unit informasi komputer.Bluetooth : sebuah cara untuk membagikan dan ditampilkan.format : graphics card : perangkat komputer untuk memproses gambar.hard disk : perangkat komputer untuk menyimpan data.headphone : perangkat untuk mendengarkan suara.icon : simbol untuk program dan data dalam komputer.internet : jaringan.IP address : kode unik yang membedakan setiap komputer dalam internet.keyboard : perangkat untuk mengetik.MHz (megahertz) : satuan kecepatan Pembahasan

Komputer adalah sebuah rangkaian perangkat elektronik yang dibuat  untuk memperoleh, menyimpan, dan memproses data dan informasi. Tujuan pembuatan komputer adalah untuk membuat aktivitas manusia menjadi lebih mudah, efektif, dan efisien.

Komputer terdiri dari berbagai bagian. Setiap bagian tersebut memiliki fungsi masing-masing. Bagian komputer yang sering ditemui antara lain CPU, monitor, power supply, graphics card, hard disk, dan sebagainya.
