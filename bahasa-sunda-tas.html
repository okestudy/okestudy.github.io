Dhamir  artinya adalah kata ganti. Dhamir dari segi bersambung atau tidaknya terbagi menjadi dua

Dhamir muttasil adalah dhamir yang bersambung dengan kata lainnya. Dhamir mumfasil adalah dhamir yang berdiri sendiri atau berpisah dengan kata lainnya.

 Dhamri berdasarkan wujudnya termagi menjadi dua yaitu

Dhamir dzahir adalah dhamir yang telihat Dhmair mustatir adalah dhamir yang tersembunyi dalm fi'il ( kata kerja ) atau dalam isim ( kata benda)

Dhamir ( kata ganti orang) untuk kepemilikan sesuatu secara keseluruhan ada 14 yaitu

_____________________

Penjelasan:

Kalimah "kamari kuring tas ti imah nini", bisa diwangun ku wangunan ieu:

Kamari: aksara ka, aksara ma, aksara ra + panghulu.Kuring: aksara ka + panyuku, aksara ra + panghulu + panyecek.Tas: aksara ta, aksara sa + pamaéh.Ti: aksara ta + panghulu.Imah: aksara swara i, aksara ma + pangwisad.Nini: aksara na + panghulu, aksara na + panghulu.

_____________________

AKSARA NGALAGENA

AKSARA SWARA

ᮃ (a) ᮅ (u) ᮆ (é) ᮈ (e)ᮄ ( i ) ᮉ (eu) ᮇ (o)

RARANGKEN

➡️ Rarangken luhur:

Panghulu (a -> i). Contohna: ᮒᮤ // ti.

Pamepet (a -> e). Contohna: ᮒᮨ // te

Paneuleung (a -> eu). Contohna: ᮒᮩ // teu.

Panglayar (+ r). Contohna: ᮒᮁ // tar

Panyecek (+ ng). Contohna: ᮒᮀ // tang

➡️ Rarangken sejajar:

Panéléng (a -> é). Contohna: ᮒᮦ // té.

Panolong (a -> o). Contohna: ᮒᮧ // to.

Pamingkal (a -> ya). Contohna: ᮒᮡ // tya.

Pangwisad (+ h). Contohna: ᮒᮂ // tah.

Pamaéh (..a -> ..). Contohna: ᮒ᮪ // t.

➡️ Rarangken handap:

Panyuku (a -> u). Contohna: ᮒᮥ // tu.

Panyakra (a -> ra). Contohna: ᮒᮢ // tra.

Panyiku (a -> la). Contohna: ᮒᮣ // tla.
