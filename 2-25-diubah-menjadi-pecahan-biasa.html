Pecahan biasa ➡️ Desimal

65/100 = 0,65

196/1.000 = 0,196

16/25 = (16 × 4)/(25 × 4) = 64/100 = 0,64

Desimal ➡️ Pecahan biasa

0,45 = 45/100 = (45 : 5)/(100 : 5) = 9/20

Pecahan biasa ➡️ Persen

5/25 = (5 × 4)/(25 × 4) = 20/100 = 20%

Persen ➡️ Pecahan biasa

40% = 40/100 = (40 : 20)/(100 : 20) = 2/5

Desimal ➡️ Persen

0,62 = 62/100 = 62%

Desimal ➡️ Pecahan biasa

0,35 = 35/100 = (35 : 5)/(100 : 5) = 7/20

0,94 = 94/100 = (94 : 2)/(100 : 2) = 47/50

1,75 = 175/100 = (175 : 25)/(100 : 25) = 7/4

Persen ➡️ Pecahan biasa

45% = 45/100 = (45 : 5)/(100 : 5) = 9/20

12% = 12/100 = (12 : 4)/(100 : 4) = 3/25

Pecahan biasa ➡️ Persen

16/25 = (16 × 4)/(25 × 4) = 64/100 = 64%

8/25 = (8 × 4)/(25 × 4) = 32/100 = 32%
