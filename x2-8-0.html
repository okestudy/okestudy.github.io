5. Jika m2 -4 < 0, maka semua nilai x yang memenuhi pertidaksamaan (x2-2x-8) (-x2 + mx-8) <0 adalah


Jawab:

Untuk menentukan semua nilai x yang memenuhi pertidaksamaan (x2-2x-8) (-x2 + mx-8) <0, pertama-tama kita perlu menyelesaikan pertidaksamaan m2 - 4 < 0. Pertidaksamaan tersebut dapat diselesaikan dengan cara mencari nilai m yang memenuhi kondisi tersebut.

Mari kita hitung masing-masing nilai m yang memenuhi pertidaksamaan m2 - 4 < 0.

Jika m = 2, maka m2 = 4, sehingga m2 - 4 = 0, bukan < 0.

Jika m = -2, maka m2 = 4, sehingga m2 - 4 = 0, bukan < 0.

Oleh karena itu, tidak ada nilai m yang memenuhi pertidaksamaan m2 - 4 < 0. Sehingga, tidak ada nilai x yang memenuhi pertidaksamaan (x2-2x-8) (-x2 + mx-8) <0.

12. Persamaan kuadrat yang akar-akarnya x = 5 dan x = -3 adalah . . .    A. X2 + 5x – 15 = 0    B. X2 + 3x + 8 = 0    C. X2 + 5x – 8 = 0    D. X2  + 2x – 15 = 0​


Penjelasan dengan langkah-langkah:

persamaan kuadrat, bentuk umum: ax² + bx + c = 0 dimana a ≠ 0.

Untuk menentukan bentuk persamaan kuadrat yang jika diketahui nilai akar akarnya, bisa menggunakan konsep sebagai berikut:

[tex](x - x_{1})(x - x_{2}) = 0[/tex]

maka:

akar akar persamaan kuadrat [tex] x_1 = 5 [/tex] dan [tex] x_2 = -3 [/tex]

[tex](x - x_{1})(x - x_{2}) = 0 \\ (x - 5)(x + 3) = 0 \\ x(x + 3) - 5( x + 3) = 0 \\ {x}^{2} + 3x - 5x - 15 = 0 \\ {x}^{2} - 2x - 15 = 0[/tex]

Jadi, bentuk persamaan kuadrat tersebut adalah x² - 2x - 15 = 0 .

26. X2+4x-8=0,maka nilai x1.x2 adalah​


Jawaban:

• Persamaan Kuadrat

Diketahui :

x² + 4x - 8 = 0

Ditanya : Nilai dari x1.x2

Jawaban :

Bentuk Umum Persamaan Kuadrat

[tex]ax^{2} + bx + c = 0[/tex]

Dengan :

[tex]x1 + x2 = - \frac{b}{a} \\ x1.x2 = \frac{c}{a} [/tex]

Maka,

x² + 4x - 8 = 0

a = 1

b = 4

c = -8

Sehingga,

[tex]x1.x2 = \frac{c}{a} \\ x1.x2 = \frac{ - 8}{1} = - 8[/tex]

Kesimpulan : Hasil dari nilai x1.x2 adalah - 8

27. Persamaan kuadrat yang tidak mempunyai akar nyata adalah... A. x2 –12x + 36 = 0 B. x2 –14 x – 10 = 0 C. x2 +2x + 8 = 0 D. x2 +8x +8 = 0

A. x² - 12x +36 = 0
a = 1, b = -12, c = 36
D = (-12)² - 4.1.(36)
D = 144-144
D =0 ⇔ D=0 (tidak memenuhi)

B. x² - 14x -10 = 0
a = 1, b = -14, c = -10
D = (-14)² - 4.1.(-10)
D = 196+40
D =236 ⇔ D>0 (tidak memenuhi)

C. x² +2x +8 = 0
a = 1, b = 2, c = 8
D = (2)² - 4.1.(8)
D = 4-32
D =-28 ⇔ D<0 (memenuhi)

D. x² + 8x + 8 = 0
a = 1, b = 8, c = 8
D = (8)² - 4.1.(8)
D = 64 - 32
D =32 ⇔ D>0 (tidak memenuhi)
