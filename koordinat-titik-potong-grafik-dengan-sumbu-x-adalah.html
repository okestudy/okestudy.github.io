Grafik f(x) = x² - 6x + 16 terlampir di Gambar 1.

a. Titik potong dengan sumbu-X :

Periksa nilai diskriminan ( D ) :

a = 1 ; b = –6 ; c = 16

D = b² - 4ac

D = (–6)² - 4.(1).(16) = 36 - 64 = –28 < 0

Karena D < 0, maka grafik f(x) tidak mempunyai titik potong dg sumbu-X, atau grafik berada di atas sumbu-X.

b. Titik potong dengan sumbu-Y :

f(0) = (0)² - 6.(0) + 16

f(0) = 0 - 0 + 16

f(0) = 16

Jadi, titik potong grafik f(x) dengan sumbu-Y :

( 0 , 16 )

c. Persamaan sumbu simetri :

tercapai ketika f'(x) = 0

f'(x) = 2x - 6

2x - 6 = 0

2x = 6

x = 3

Jadi, persamaan sumbu simetri grafik f(x) adalah : x = 3

d. Nilai minimum fungsi :

Diambil dari x = 3 ( persamaan sumbu simetri ) :

f(3) = (3)² - 6.(3) + 16

f(3) = 9 - 18 + 16

f(3) = 7

Jadi, nilai minimum fungsi f(x) adalah = 7

e. Titik balik minimum fungsi :

( 3 , 7 )

27. Koordinat titik potong grafik fungsi kuadrat dengan sumbu x


Koordinat titik potong grafik fungsi kuadrat dengan sumbu x adalah (2 + √2, 0) dan (2 – √2, 0). Bentuk umum fungsi kuadrat adalah f(x) = ax² + bx + c dengan a ≠ 0. Ada beberapa cara untuk menentukan persamaan fungsi kuadrat, yaitu

Jika diketahui titik puncak (xp, yp) dan melalui titik (x, y)

y = a(x – xp)² + yp

Jika diketahui titik potong terhadap sumbu x di titik (x₁, 0) dan (x₂, 0) serta melalui titik (x, y)

y = a(x – x₁)(x – x₂)

Pembahasan

Diketahui

Dalam grafik yang disajikan dalam soal

Titik puncak P(2, –4) ⇒ (xp, yp) = (2, –4) Memotong sumbu y di titik (0, 4) ⇒ (x, y) = (0, 4)

Ditanyakan

Titik potong terhadap sumbu x = … ?

Jawab

y = a(x – xp)² + yp

4 = a(0 – 2)² + (–4)

4 + 4 = a(–2)²

8 = 4a

a = 2

y = a(x – xp)² + yp

y = 2(x – 2)² + (–4)

y = 2(x – 2)² – 4

Fungsi kuadrat memotong sumbu x jika y = 0, maka

2(x – 2)² – 4 = 0

2(x – 2)² = 4

(x – 2)² = 2

(x – 2) = ± √2

x = 2 ± √2

x = (2 + √2) atau x = (2 – √2)

Jadi koordinat titik potong grafik fungsi kuadrat dengan sumbu x adalah (2 + √2, 0) dan (2 – √2, 0)
