2. suatu mesin carnot menggunakan reservoir suhu tinggi 1000k dan reservoir suhu rendah 600k. efisiensi mesin carnot tersebut adalah...

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
T1 = 1000 K
T2 = 600 K

Ditanya :
η = ?

Penyelesaian :
η = T1 - T2 / T1
η = 1000-600 / 1000
η = 400/1000
η = 0,4
η = 40%

-> Jadi, efisiensi mesin carnot tsb adalah 40%

3. Suhu tinggi reservoir sebuah mesin Carnot 727°c dan efisiensi mesin 40%. Agar efisiensi mesin Carnot naik menjadi 60% dengan mempertahankan suhu reservoir rendah tetap, maka berapakah suhu tinggi reservoir mesin Carnot tersebut!


Termodinamika
• Mesin Carnot
η = 40%, T1 = 1000 K
η' = 60%, T1' = ___?

T1' = T1 • (1- η)/(1- η')
T1' = 1000 • (1-0,4)/(1-0,6)
T1' = 1000 • (0,6/0,4)
T1' = 1500 K = 1227 °C ← jwb

4. 3.Sebuah mesin carnot mempunyai resevoir suhu rendah 27 C dan reservoir suhu tinggi 327 C. Hitunglah Besar efisiensi mesin carnot tersebut !4.Sebuah mesin carnot memiliki suhu reservoir suhu panas 727o C dengan efisiensi 30%. Berapakah reservoir suhu rendah yang dimiliki mesin tersebut?5.Sebuah mesin carnot beroperasi pada reservoir panas 487 C dan reservoir dingin 107 C. Jika mesin menyerap kalor 800 Joule dari reservoir panas, hitunglah kalor yang dibuang dari mesin!​


Jawaban:

3. 50%

4. T2 = 700°K

5. Q2 = 400 joule

Penjelasan:

3. η = [1 – (T2/ T1)] x 100%

Keterangan:

η = efisiensi mesin Carnot

T1 = suhu reservoir bersuhu tinggi (K)

T2 = Suhu Reservoir bersuhu rendah (K)

T1 = 327°C = 327° + 273° = 600°K

T2 = 27° C = 27° + 273° = 300°K

η = [1 – (T2/ T1)] x 100%

= [1 - (300°/600°)] x 100%

= [1 - 0,5] x 100%

= 0,5 x 100%

= 50%

4. T1 = 727°C = 727° + 273° = 1.000°K

η = [1 – (T2/ T1)] x 100%

30% = [1 - (T2 / 1.000)] x 100%

30% ÷ 100% = 1 - (T2 / 1.000)

T2 / 1.000 = 1 - 0,3

T2 = 0,7 x 1.000 = 700°K

5. T1 = 487°C = 487° + 273° = 760°K

T2 = 107°C = 107° + 273° = 380°K

η = [1 – (T2/ T1)] x 100%

η = [1 – (Q2/ Q1)] x 100%

Q1 = kalor yang diserap

Q2 = kalor yang dibuang

Q2 / Q1 = T2 /T1

Q2 = T2/T1 x Q1

= 380° / 760° x 800

= 0,5 x 800

= 400 joule


5. suhu rendah suatu reservoir mesin carnot 500 derajat K dan efisiensi 60% agar efisiensi mesin 80% maka suhu tinggi reservoir mesin carnot harus dinaikan sebesar..


Termodinamika.
Kelas XI kurikulum 2013 revisi 2016.

η = 1 - T₂ / T₁ → T₂ = T₁ (1 - η)
η' = 1 - T₂ / T₁'
    = 1 - T₁ (1 - η) / T₁'
T₁' = T₁ (1 - η) / (1 - η')
     = (500 K) (1 - 0,6) / (1 - 0,8) = 1000 K
Harus dinaikkan sebesar 1000 K - 500 K = 500 K
