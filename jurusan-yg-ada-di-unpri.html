14. sebuah terminal bus melayani 3 jurusan. bus bus yg menuju jurusan pertama berangkat setiap 45menit, kejurusan kedua berangkat setiap 60menit, dan jurusan ketiga berangkat setiap 75menit. jika pada pukul 06.00 ada 3 bus yg berangkat menuju ketiga jurusan tersebut secara bersamaan, pda pukul berapakah bus bus berikutnya akn berangkat secaraan menuju jurusan tersebut?


45 menit = 5 x 3²
60 menit = 2² x 5 x 3
75 menit = 5² x 3

maka KPK nya
= 5² x 3² x 2²
= 25 x 9 x 4
= 900 menit (60 menit = 1jam)
= 15 jam

jadi pukul 06.00 + 15.00 = 21.00 berangkat bersama lagi.
