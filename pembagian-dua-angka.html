Diketahui :  

Akan dibentuk bilangan yang terdiri dari empat angka berbeda Bilangan tersebut tidak habis dua Angka pembentuknya terdiri dari 1, 2, 3, 4, 5, 6, 7 ada 7 buah

Ditanyakan :

Bilangan yang terbentuk ada sebanyak

Jawab :

Karena bilangan yang dicari adalah yang tidak habis dibagi dua artinya bilangan tersebut ganjil.

Dan yang bisa membuat bilangan tersebut menjadi bilangan ganjil yaitu angka yang menduduki tempat ke-4 / satuan merupakan bilangan ganjil sehingga yang bisa menempati angka ke-4 / satuan adalah 1 / 3 / 5 / 7

Jadi yang bisa menempati tempat ke-4 / satuan ada 4 buah Untuk angka ke-3 / puluhan yang bisa menduduki tinggal 6 buah angka karena salah satunya sudah dipakai sebagai angka satuan Untuk angka ke-2 / ratusan yang bisa menduduki tinggal 5 buah angka karena ada dua angka yang sudah dipakai sebagai angka satuan dan puluhan Untuk angka ke-1 / ribuan yang bisa menduduki tinggal 4 buah angka karena ada tiga buah angka yang sudah dipakai sebagai angka satuan, puluhan, dan ratusan

Jadi bilangan yang terbentuk ada sebanyak  

[tex]= 4 \times 5 \times 6 \times 4\\= 480[/tex]

d : b = a : c  .... pers IV

Subtitusi a  + c = d - b ke dalam pers I

(a + c) + b + d = 12

d - b + b + d = 12

              2d = 12

                d = [tex]\displaystyle \frac{12}{2}[/tex]

                d = 6

Subtitusi c = b - a dan d = 6 ke dalam pers I

     2b = 12 - 6

       b = [tex]\displaystyle \frac{6}{2}[/tex]

       b = 3

d : b = a : c

[tex]\displaystyle \frac{d}{b} = \frac{a}{b - a}[/tex]

28. Sebuah bilangan terdiri dari empat buah angka yang berbeda. Jumlah keempat angka adalah 12. Angka pertama ditambah angka ketiga sama dengan angka keempat dikurangi angka kedua. Angka kedua sama dengan selisisih dengan angka pertama dan ketiga. Angka ke empat dibagi angka ketiga sama dengan angka pertama dibagi angka kedua. Bilangan tersebut adalah


Bilangan tersebut adalah 2136.

Soal ini merupakan soal USM STAN 2017 dengan soal berbentuk persamaan linear empat (4) variabel.

Pembahasan

Diketahui :

Jumlah keempat angka = 12

Angka pertama ditambah angka ketiga = angka keempat dikurangi angka kedua

Angka kedua = selisisih dengan angka pertama dan ketiga

Angka ke empat dibagi angka ketiga = angka pertama dibagi angka kedua

Ditanya :

Bilangan tersebut ?

Penyelesaian :

Membuat persamaan dari pernyataan

Misal : a = angka I

           b = angka II

           c = angka III

           d = angka IV

a + b + c + d = 12 ... pers I

a  + c = d - b .... pers II

b = c - a ..... pers III

d : c = a : b  .... pers IV

Subtitusi a  + c = d - b ke dalam pers I

(a + c) + b + d = 12

d - b + b + d = 12

              2d = 12

                d = [tex]\displaystyle \frac{12}{2}[/tex]

                d = 6

Subtitusi b = c - a dan d = 6 ke dalam pers I

a + b + c + d = 12

a + c - a  + c + 6 = 12

2c + 6 = 12

     2c = 12 - 6

       c = [tex]\displaystyle \frac{6}{2}[/tex]

       c = 3

Subtitusi b = c - a ke dalam pers IV

d : c = a : b

[tex]\displaystyle \frac{d}{c} = \frac{a}{c - a}[/tex]

d (c - a) = a c   ... pers V

Subtitusi c = 3 dan d = 6 ke dalam pers V

d (c - a) = a c

6 (3 - a) = 3 a

18 - 6a = 3a

-6a - 3a = -18

      -9a = -18

          a = [tex]\displaystyle \frac{-18}{-9}[/tex]

          a = 2

Subtitusi a = 2 dan c = 3 ke dalam pers III

b = c - a

b = 3 - 2

b = 1

Dengan demikian nilai a = 2, b = 1, c = 3 dan d = 6

Jadi bilangan tersebut adalah 2136
