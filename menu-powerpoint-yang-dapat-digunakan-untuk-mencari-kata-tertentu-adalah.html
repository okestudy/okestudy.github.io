20. untuk membuat diagram pada Microsoft Office Powerpoint 2019 menggunakan menu​


Jawaban:

Untuk membuat diagram di Microsoft Office PowerPoint 2019, pertama-tama buka program PowerPoint dan buatlah sebuah slide baru. Kemudian ikuti langkah-langkah berikut:

Klik tab "Insert" di toolbar.Klik "Illustrations" kemudian pilih "SmartArt".Pilih jenis diagram yang ingin Anda buat dari daftar yang tersedia.Setelah memilih jenis diagram, akan muncul jendela baru yang memungkinkan Anda untuk mengatur diagram tersebut. Anda dapat mengubah warna, ukuran, dan bentuk dari setiap elemen diagram sesuai keinginan.Ketika selesai, klik "OK" untuk memasukkan diagram ke dalam slide.

Anda juga dapat membuat diagram menggunakan tab "Insert" di toolbar. Pilih "Chart" kemudian pilih jenis diagram yang ingin Anda buat dari daftar yang tersedia. Setelah memilih jenis diagram, Anda dapat mengatur dan mengedit data yang akan ditampilkan di diagram tersebut melalui jendela "Edit Data" yang muncul.

28. menu menu dalam Microsoft powerpoint


Menu - Menu di Microsoft Powerpoint:FileHomeInsertDrawDesignTransitionsAnimationsSlide ShowReviewViewStoryboardingMENU INSERT
KETERANGAN:

A. File / office buttom
B. Save : untuk mengesave file.
C. Table : untuk membuat table.
D. Picture : untuk memasukkan gambar.
E. Clipart : untuk menambahkan clipart.
F. Shape : untuk menambahkan bidang ruang / obyek.
G. Grafik : untuk membuat grafik.
H. Word art : untuk membuat text artistik.
I. Syimbol : untuk menambahkan syimbul.
J. video :untuk menambahkan video.
K. Audio : untuk menambahkan suara/lagu.
L. Title bar / judul bar.


MENU HOME
KETERANGAN :

A. Save : untuk menyimpan dengan nama yang sama.
B. New slide : untuk menambah halaman baru.
C. Font : untuk mengedit font.
D. Layout : untuk mengatur tata halaman.
E. Paragraph : untuk mengatur paragraph.
F. Drawing : untuk membuat / mengatur obyek.


MENU DESIGN
KETERANGAN :

A. File / office buttom
B. Save : untuk menyimpan file dengan nama yang sama.
C. Undo : untuk mengembalikan ke sebelumnya.
D. Theme : untuk mengganti background halaman.
E. Background style : untuk mengatur backgroun halaman.


MENU ANIMATION
KETERANGAN :

A. Save : untuk menyimpan file dengan nama yang sama.
B. Prevew : untuk melihat hasil.
C. Animation : untuk menambahkan animasi pada huruf/kata/kalimat/paragraf.
D. Advenced animation : untuk menambah animasi lain.
E. Timing : untuk memberi waktu kapan animasi akan di play dan berapa lama durasinya.


MENU TRANSITION
KETERANGAN :

A. Save : untuk menyimapan file dengan nama yang sama.
B. Prevew : untuk melihat hasil translite.
C. File / office button
D. Transition to this slide : untuk menambahkan translite pada layout.
E. Timing : memberi waktu kapan akan ganti slide.


MENU SLIDE SHOW
KETERANGAN :

A. From beginning : Untuk menjalankan slide mulai dari posisi awal hingga akhir slide.
B. From current slide : Untuk menjalankan slide mulai dari slide aktif atau slide yang sedang dipilih hingga akhir slide.
C. Custom slide show : Untuk mengatur sendiri urutan slide presentasi sesuai dengan keinginan.
D. Set up slide show : Untuk mengatur tampilan slide.
E. Use timings : Untuk mengatur lamanya tampilan slide menggunakan waktu tertentu.


MENU REVIEW
KETERANGAN :

A. Spelling : Untuk mengecek pengejaan kalimat.
B. Research : Hampir sama dengan Find, namun ini lebih spesific.
C. Translate : Untuk menerjemahkan kebahasa lain.
D. Language : Untuk memilih bahasa kalimat.
E. Comments : Untuk memberikan catatan pada slide yang Anda buat.
