2. Diketahui tabel harga jumlah permintaan dan jumlah penawaran terhadap barang X sebagai berikut harga..1.Harga barang Rp1.500.00 jumlah permintaan 110 jumlah penawaran 90. <br />2. Harga barang Rp 2.500.00 jumlah permintaan 80 jumlah penawaran 100 <br />3. Harga barang Rp 3.000.00 jumlah permintaan 60 jumlah penawaran 120. Berdasarkan tabel tersebut tentukan pungsi permintaan??​


Fungsi Permintaan Qd = a - bp

Q - Q1 / Q2 - Q1 = P - P1 / P2 - P1

________________________________

Diketahui :

P1 = Rp1.500

Q1 = 110

P2 = Rp2.500

Q2 = 80

P3 = Rp3.000

Q3 = 60

Ditanya :

Qd = ?

Jawab :

Q - Q1 / Q2 - Q1 = P - P1 / P2 - P1

Q - 110 / -30 = P - 1.500 / 1.000

1.000Q - 110.000 = -30P + 45.000

1.000Q = 110.000 + 45.000 - 30P

1.000Q = 155.000 - 30P

Q = 155 - 0,03P

Qd = 155 - 0,03P

Jadi, berdasarkan tabel tersebut, fungsi permintaannya adalah Qd = 155 - 0,03P

Semoga bermanfaat
