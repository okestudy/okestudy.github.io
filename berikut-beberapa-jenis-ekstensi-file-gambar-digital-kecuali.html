11. File ekstensi untuk fotoshop pada saat disimpan adalah dapat berupa file file di bawah ini,kecuali?


Didalam Photoshop, terdapat kurang lebih 21 jenis format file yang dapat dipilih oleh pengguna. Format file tersebut antara lain adalah sebagai berikut

- PSD,PDD (Photoshop Document). Merupakan format penyimpanan asli dari Photoshop

- PSB (Photoshop Big). Digunakan untuk meningkatkan resolusi maksimum

- BMP, RLE, DIB. Digunakan untuk menyimpan gambar tipe bitmap

- GIF. Digunakan untuk menyimpan gambar tipe GIF. Memiliki kombinasi 256 warna. Biasa digunakan untuk logo, line drawing dan icon

- EPS (Encapsutaed Potscript). Digunakan untuk pencetakan atau pertukaran dokumen antar program grafis

- IFF (Interchange File). Digunakan untuk video

- JPEG. Digunakan untuk menyimpan mode warna RGB, CYMK, Grayscale

- JPEG 2000 (JPF,JPX,JP2,J2C,J2K,JPC). Merupakan standar kompresi untuk meningkatkan rasio kualitas dan kompresi

- JPS (JPEG Stereo). Terdiri dari side by side berdasarkan JPEG

- MPO (Multi Picture Format). Digunakan untuk format 3D

- PCX (Personal Computer Exchange). Berberntuk rester.

- Photoshop PDF. Digunakan oleh Adobe Acrobat

- RAW. Digunakan untuk menyimpan warna RGB, CMYK dan Grayscale

- PXR (Pixar). Digunakan untuk pertukaran file dengan Pixar Image Computer

- PNG (Portable Network Graphics). Digunakan untuk transparasi dalam browser

- Portable Bit Map (PBM,PGM,PPM,PNM,PFM,PAM). Digunakan untuk menyimpan informasi dengan kualitas 1 bit hingga 24 bit

- Scitex CT (SCT)

- Targa (TGA,VDA,ICB,VST)

- Tagged Image Format File (TIF,TIFF)

Dengan demikian, jika file ektensi tidak termasuk dalam daftar diatas maka bukan merupakan file ekstensi untuk Photoshop

Mapel : TIK

15. Sebutkan 4 jenis ekstensi dari jenis data/file Video!


Jawaban:

1. MP4

2. MKV

3. WMV

4. AVI

Penjelasan:

1. mp4 adalah ekstensi video berstandar nasional, kelebihannya dengan tingkat kompresi tinggi yang digunakan mp4 dapat berukuran lebih kecil dari ekstensi kebanyakan

2. mkv adalah matroska video karena format file MKV hanya format kontainer umum, dapat menampung beberapa track berbeda yang masing-masing menggunakan format kompresi berbeda. Dengan demikian, tidak mudah bergantung pada satu pemain MKV yang dapat terbuka setiap File MKV yang Anda miliki, kelebihannya adalah mempunyai fitur pencarian cepat yang ada di dalam file, memiliki dukungan ful metadata

3. wmv adalah windows media video, ektensi ini dikhususkan untuk windows, menggunakan format pembawa ASF milik Microsoft, kelebihannya adalah wadah ASF opsional dapat mendukung manajemen hak digital menggunakan kombinasi kriptografi kurva elivtik pertukaran kunci, DES block cipher, blok cipher kustom, RC4 stream cipher dan SHA-1 fungsi hashing

4. avi adalah audio video interleave, kelebihannya: banyak pilihan jenis kompresi yang dapat digunakan, semakin baik jenis kompresinya, semakin baik kualitasnya, gambarnya lebih tajam dibandingkan dengan ASF

19. berikut yang merupakan meta dan file kecuali judul deskripsi keyword ekstensi​


Jawaban:

Metadata adalah informasi terstruktur yang mendeskripsikan, menjelaskan, menemukan, atau setidaknya menjadikan suatu informasi mudah untuk ditemukan kembali, digunakan, atau dikelola. Metadata sering disebut sebagai data tentang data atau informasi tentang informasi. Metadata ini mengandung informasi mengenai isi dari suatu data yang dipakai untuk keperluan manajemen file/data itu nantinya dalam suatu basis data. Jika data tersebut dalam bentuk teks, metadatanya biasanya berupa keterangan mengenai nama ruas (field), panjang field, dan tipe fieldnya: integer, character, date, dll. Untuk jenis data gambar (image), metadata mengandung informasi mengenai siapa pemotretnya, kapan pemotretannya, dan setting kamera pada saat dilakukan pemotretan. Satu lagi untuk jenis data berupa kumpulan file, metadatanya adalah nama-nama file, tipe file, dan nama pengelola (administrator) dari file-file tersebut.
