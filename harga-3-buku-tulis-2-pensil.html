[tex] \text{ - 4y = - 12.000}[/tex]

[tex] \text{y} = \frac{ -12.000}{ - 4} [/tex]

[tex] \text{y} = 3.000[/tex]

Y = 3.000

[tex] \text{3x + y = 18.000}[/tex]

[tex] \text{3x = 15.000}[/tex]

[tex] \text{x} = \frac{15.000}{3} [/tex]

[tex] \text{x} = 5.000 [/tex]

Misalkan, x = buku tulis dan y = pensil, Maka

Diketahui:

2x + 3y = 18.000

x + 2y = 10.500

Ditanya:

Harga 2x + 2y = ?

Jawab;

2x + 3y = 18.000 ...... pers (1)

x + 2y = 10.500 ....... pers (2)

Eliminasi x dari persamaan (1) dan (2)

2x + 3y = 18.000 | × 1 | 2x + 3y = 18.000

x + 2y = 10.500 | × 2 | 2x + 4y = 21.000

_______________-

-y = -3.000

y = 3.000

Substitusikan nilai y ke persamaan (2)

x + 2y = 10.500

x + 2(3.000) = 10.500

x + 6.000 = 10.500

x = 10.500 - 6.000

x = 4.500

Ditemukan bahwa, nilai x = 4.500 dan y = 3.000

Maka,

2x + 2y = 2(4.500) + 2(3.000)

= 9.000 + 6.000

= 15.000

Sehingga, harga dari 2 buku tulis + 2 pensil adalah Rp 15.000,00
