1. Sketsalah grafik fungsi berikut ini. a. y = 6x2 + 5x + 7 b. y = 7x2 – 3x + 2


Jawaban:

a. tidak bisa digambar

b. tidak bisa digambar

Penjelasan dengan langkah-langkah:

Langkah - Langkah :

Soal A

1. Tentukan koordinat koordinat x dan y

Sebelum melangkah jauh, kita harus memastikan persamaan ini memiliki akar penyelesaian.

[tex]y = f(x) = 6 {x}^{2} + 5x + 7[/tex]

Jika y = 0, maka x :

[tex]0 = f(0) = 6 {x}^{2} + 5x + 7 \\ a = 6 \\ b = 5 \\ c = 7[/tex]

Persamaan di atas tidak bisa diselesaikan dengan faktorisasi. Yang terbaik adalah metode ABC

Mari kita cek D (Diskriminan) dahulu :

D =

[tex]{b}^{2} - 4ac = {5}^{2} - 4(6)(7) = - 143[/tex]

Karena nilai D negatif, maka tidak memiliki akar persamaan real. Ini akan menghasilkan bilangan imajiner.

Maka kita tidak bisa melanjutkan operasi ini. TIDAK BISA DIGAMBAR.

Soal B

1. Tentukan koordinat koordinat x dan y

Sebelum melangkah jauh, kita harus memastikan persamaan ini memiliki akar penyelesaian.

[tex]y = f(x) = 7{x}^{2} - 3x + 2[/tex]

Jika y = 0, maka x :

[tex]0 = f(0) = 7 {x}^{2} - 3x + 2 \\ a = 7\\ b = - 3 \\ c = 2[/tex]

Persamaan di atas tidak bisa diselesaikan dengan faktorisasi. Yang terbaik adalah metode ABC

Mari kita cek D (Diskriminan) dahulu :

D =

[tex]{b}^{2} - 4ac = {( - 3)}^{2} - 4(7)(2) = - 47[/tex]

Karena nilai D negatif, maka tidak memiliki akar persamaan real. Ini akan menghasilkan bilangan imajiner.

Maka kita tidak bisa melanjutkan operasi ini. TIDAK BISA DIGAMBAR.


2. Sketsalah grafik fungsi berikut ini a) y= 6x2+5x+7 b) 7x2-3x+2banyak ni poinnya


Sketsa grafik fungsi kuadrat dapat dikerjakan menggunakan beberapa langkah. Pertama tentukan nilai a(koefisien x2) dari fungsi tersebut. Jika a > 0 maka grafik terbuka ke atas. Jika a > 0 maka grafik akan terbuka ke bawah. Lalu tentukan nilai D dengan rumus D = b2 – 4ac.

 Berikut adalah gambaran grafik berdasarkan nilai a dan D :    

  (Gambar 1)


Selanjutnya tentukan titik potong nilai x ( dengan memasukkan y = 0) dan tentukan titik potong nilai y ( dengan memasukkan x = 0). Setelah itu tentukan sumbu simetri dengan rumus x = -b/2a. Lalu tentukan titik puncak dengan memasukkan nilai x yang didapat dari rumus x = -b/2a ke persamaan awal.

Pembahasan  

1. 6x² + 5x + 7    

a > 0 maka grafik terbuka ke atas     D = 52 – 4 . 6 .7 = -143 , D < 0 grafik tidak memotong sumbu x     Saat x = 0 , y = 7     Sumbu simetri x = -5/12 = - 0.4     Titik puncak, saat x  = -0.4,   y = 5.96        Jadi koordinat puncaknya adalah (-0.4, 5.96)        

(Gambar 2)


2. 7x² - 3x + 2    

a > 0 maka grafik terbuka ke atas     D = 32 – 4 . 7 .2 = -47 , D < 0 grafik tidak memotong sumbu x     Saat x = 0 , y = 2     Sumbu simetri x = 3/14 =  0.2     Titik puncak, saat x  = 0.2 ,   y = 1.68       Jadi koorddinat puncaknya adalah (0.2, 1.68)      

        (Gambar 3)

Kesimpulan

Yang perlu ditentukan untuk mensketsa fungsi kuadrat adalah nilai a, D, titik potong x dan y, sumbu simetri dan titik puncak.

16. Tanpa menggambar sketsa grafiknya,tentunkan apakah grafik fungsi kuadrat berikut terbuka ke atas atau ke bawah y=2x²+5x-1​


Grafik Fungsi Kuadrat

Untuk mengetahui bentuk umum grafik fungsi kuadrat, yaitu terbuka ke atas atau terbuka ke bawah, maka lihat koefisien a.

Jika koefisien a > 0, maka grafik tersebut terbuka ke atas.

Sebaliknya, jika koefisien a < 0, maka grafik tersebut terbuka ke bawah.

Karena koefisien a pada persamaan y = 2x² + 5x - 1 lebih dari 0, maka grafik fungsi kuadrat tersebut terbuka ke atas.
