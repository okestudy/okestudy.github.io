penumpang maksimal = 50 orang
kelas I = 60 kg/org, & kelas II = 30 kg/org
beban maksimal = 1800 kg

misal: a = penumpang kls I, & b = penumpang kls II
a x 60 + b x 30 = 1800 --> 60a + 30b = 1800
a + b = 50 --> (dikali 30) : 30a + 30b = 1500

60a + 30b = 1800
30a + 30b = 1500
------------------------------ - (eliminasi)
30a = 300
a = 300/30
a = 10 ---> a + b = 50 --> b = 50-10 = 40

jadi, penumpang kelas I sebanyak 10 org, & penumpang kelas II sebanyak 40 org

21. 1. Sebuah pesawat mempunyai tempat duduk tidak lebih dari 52 penumpang. setiap penumpang kelas utama boleh membawa bagasi 60kg. sedangkan untuk penumpang kelas ekonomi bagasinya dibatasi 20kg. pesawat itu hanya dapat membawa bagasi 1.560kg. Harga tiket kelas utama Rp. 1.600.000.00 dan kelas ekonomi Rp. 850.000.00 jika banyak penumpang kelas utama dimisalkan x dan banyak penumpang kelas ekonomi dimisalkan y. pendapatan maksimum yang diperoleh adalah...


                  Penumpang   Bagasi       tiket
Utama                x                60     1.600.000
Ekonomi            y                20       850.000
Jumlah             52            1.560  

Persamaan Linier
x + y ≤ 52
60x + 20y ≤ 1.560
Ditanya, pendapatan maksimum
1.600.000x + 850.000y =

Titik potong A ; kondisi cuma mengangkut penumpang utama (y=0)
x + y    ≤ 52
x + (0) ≤ 52
     x     ≤ 52
60x + 20y    ≤ 1.560
60x + 20(0) ≤ 1.560
       x          ≤ 1.560/60
           x      ≤ 26
Jadi, titik potong A = (26, 0)

Titik potong B ; kondisi cuma mengangkut penumpang ekonomi (x=0)
x + y    ≤ 52
(0) + y ≤ 52
     y     ≤ 52
60x + 20y    ≤ 1.560
60(0) + 20y ≤ 1.560
             y    ≤ 1.560/20
          y       ≤ 78
Jadi, titik potong B = (0, 52)

Titik potong C ; perpotongan garis A dan B
    x + y     ≤ 52      |x20| 20x + 20y ≤ 1.040
60x + 20y ≤ 1.560 |x1  | 60x + 20y ≤ 1.560
                                    ------------------------ ---
                                       40x         ≤ 520
                                               x     ≤ 520/40
                                                   x ≤ 13
     x + y ≤ 52
(13) + y ≤ 52
          y ≤ 52 - 13
          y ≤ 39
Jadi, titik potong C = (13, 39)

Masukkan fungsi 1.600.000x + 850.000y =
Titik A = (26, 0)   => 1.600.000(26) + 850.000(0)   = 41.600.000
Titik B = (0, 52)   => 1.600.000(0) + 850.000(52)   = 44.200.000
Titik C = (13, 39) => 1.600.000(13) + 850.000(39) =
                           (=) 20.800.000 + 33.150.000       = 53.950.000
---------------------------------------------------------------------------------
Jadi, pendapatan maksimun yang bisa didapat adalah Rp. 53.950.000,-
dengan mengangkut 13 penumpang utama dan 39 penumpang ekonomi

x = tempat duduk kelas bisnis
y = tempat duduk kelas ekonomi

x + y = 48 ---> y = 48 - x

40x + 20y = 1440 ---> kedua ruas dibagi 20
2x + y = 72
2x + 48 - x = 72
x = 72 - 48
x = 24

y = 48 - x
y = 48 - 24
y = 24

jadi jumlah tempat duduk untuk kelas bisnis dan kelas ekonomi masing - masing adalah 24 kursi
