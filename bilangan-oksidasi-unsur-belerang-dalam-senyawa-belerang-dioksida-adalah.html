H₂SO₄= 0

(jumlah atom H x biloks atom H) + (jumlah atom S x biloks atom S) + (jumlah atom O x biloks atom O) = 0

(2 x 1) + (1 x biloks S) + (4x (-2)) = 0

2 + Biloks S - 8 = 0

Biloks S= 8-2

Biloks S = +6

16. Berikut adalah 3 nama senyawa yang mengandung unsur oksigen :1) Belerang dioksida2) Dinitrogen pentaoksida3) Difosfor pentaoksidaBilangan oksidasi unsur belerang, nitrogen, dan fosfor pada senyawa tersebut berturut-turut adalah ....​


Bilangan oksidasi unsur belerang, nitrogen, dan fosfor pada senyawa tersebut berturut-turut adalah +4, +5, dan +5.

PEMBAHASANAturan Biloks

1). Biloks unsur bebas = 0. Unsur bebas adalah unsur yang tidak berikatan dengan unsur lain. Misal Na, K, H₂, S₈.

2). Biloks unsur logam :

→ Golongan IA (Li, Na, K, Rb, Cs, dan Fr) = +1

→ Golongan IIA (Be, Mg, Ca, Sr, Ba, dan Ra) = +2

→ Golongan IB (Ag) = +1

→ Golongan IIB (Zn) = +2

→ Golongan IIIB (Al) = +3

3). Biloks unsur-unsur golongan VII A (F, Cl, Br, I) dalam senyawa biner logam adalah -1.

4). Biloks ion monoatom = jumlah muatannya. Misal biloks Mg²⁺ = +2 dan Cl⁻ = -1.

5). Biloks atom H jika berikatan dengan non logam = +1, sedangkan jika berikatan dengan logam biloks H = -1. Misal pada senyawa hidrida (NaH, LiH, BaH₂, AlH₃, KH, CaH₂, dll) biloks H = -1.

6). Biloks atom O dalam senyawa = -2, kecuali :

Senyawa peroksida (H₂O₂, Na₂O₂, dan BaO₂), biloks O = -1Senyawa superoksida (KO₂, RbO₂), biloks O = -¹/₂Senyawa fluorida (F₂O) biloks O = +2

7). Jumlah biloks dalam suatu senyawa = 0. Misal biloks Fe₂O₃ = 0 dan BaCO₃ = 0

8). Jumlah biloks atom-atom dalam suatu ion poliatomik = jumlah muatannya. Misal biloks ion NH₄⁺ = +1 dan ion Cr₂O₇²⁻ = -2.

※Pertanyaan※

Nama senyawa :

1) Belerang dioksida (SO₂)

2) Dinitrogen pentaoksida (N₂O₅)

3) Difosfor pentaoksida (P₂O₅)

Biloks S, N, dan P pada senyawa tersebut?

※Penyelesaian※

∴ Jumlah biloks suatu senyawa yang tidak bermuatan adalah 0.

1). Biloks S dalam SO₂

⇒ Biloks O dalam senyawa adalah -2.

[tex]\begin{aligned}\rm{S+2~O}&\rm{=0}\\\rm{S+2\times(-2)}&\rm{=0}\\\rm{S-4}&\rm{=0}\\\rm{S}&\rm{=+4}\end{aligned}[/tex]

Biloks S dalam SO₂ adalah +4.

2). Biloks N dalam N₂O₅

⇒ Biloks O dalam senyawa adalah -2.

[tex]\begin{aligned}\rm{2~N+5~O}&\rm{=0}\\\rm{2~N+5\times(-2)}&\rm{=0}\\\rm{2~N-10}&\rm{=0}\\\rm{2~N}&\rm{=10}\\\rm{N}&\rm{=\frac{10}{2}}\\\rm{N}&\rm{=+5}\end{aligned}[/tex]

Biloks N dalam N₂O₅ adalah +5.

3). Biloks P dalam P₂O₅

⇒ Biloks O dalam senyawa adalah -2.

[tex]\begin{aligned}\rm{2~P+5~O}&\rm{=0}\\\rm{2~P+5\times(-2)}&\rm{=0}\\\rm{2~P-10}&\rm{=0}\\\rm{2~P}&\rm{=10}\\\rm{P}&\rm{=\frac{10}{2}}\\\rm{P}&\rm{=+5}\end{aligned}[/tex]

Biloks P dalam P₂O₅ adalah +5.

22. Difosfor pentaoksida,nitrogen dioksida, belerang trioksida. Bilangan oksidasi unsur fosfor,nitrogen,dan belerang pada senyawa tersebut adalah


Difosfor pentaoksida, nitrogen dioksida, belerang trioksida. Bilangan oksidasi unsur fosfor, nitrogen, dan belerang pada senyawa tersebut adalah ...JawabanPendahuluan

Ini merupakan persoalan mengenai penentuan bilangan oksidasi unsur pada suatu senyawa. Ditanyakan bilangan oksidasi unsur fosfor (P), nitrogen (N), dan belerang (S).

Pembahasan

Bilangan oksidasi dapat berupa bilangan negatif, nol, atau positif. Beberapa aturan bilangan oksidasi yang digunakan pada pembahasan soal kali ini meliputi:

[1]. Suatu senyawa netral harus mempunyai penjumlahan bilangan oksidasi unsur-unsurnya sama dengan nol.

Contoh pada H₂O atau Al₂(SO₄)₃.

[2]. Oksigen (O) dalam suatu senyawa mempunyai bilangan oksidasi -2, kecuali pada senyawa peroksida seperti H₂O₂, BaO₂, NaO, bilangan oksidasi O sama dengan -1. Khusus dalam OF₂, bilangan oksidasi O sama dengan +2 karena F lebih elektronegatif.

Mengingatkan kembali awalan penting yang digunakan dalam penamaan senyawa non logam, yakni: mono = 1; di = 2; tri = 3; tetra = 4; penta = 6.

Penyelesaian

(1). Difosfor pentaoksida P₂O₅

2[Biloks P] + 5[biloks O] = 0

2[Biloks P] + 5(-2) = 0

2[Biloks P] = 10

∴ Biloks P = +5

(2). Nitrogen dioksida NO₂

Biloks N + 2[biloks O] = 0

Biloks N + 2(-2) = 0

∴ Biloks N = +4

(3). Belerang trioksida SO₃

[Biloks S] + 3[biloks O] = 0

[Biloks S] + 3(-2) = 0

∴ Biloks S = +6

Kesimpulan

Dari uraian pembahasan dan langkah pengerjaan di atas, diperoleh kesimpulan sebagai berikut:

[tex]\boxed{P_2O_5\rightarrow bilangan~oksidasi~P = +5}[/tex]

[tex]\boxed{NO_2\rightarrow bilangan~oksidasi~N = +4}[/tex]

[tex]\boxed{SO_3\rightarrow bilangan~oksidasi~S = +6}[/tex]

30. Difosfor pentaoksida,nitrogen dioksida, belerang trioksida. Bilangan oksidasi unsur fosfor,nitrogen,dan belerang pada senyawa tersebut afalah


Jawaban:

 

Bilangan oksidasi unsur fosfor dalam Difosfor pentaoksida (P2O5) = +5

Bilangan oksidasi nitrogen dalam Nitrogen dioksida (NO2) = +4

Bilangan oksidasi belerang dalam Belerang trioksida (SO3) = +6

 

Pembahasan:

 

Bilangan oksidasi (biloks) adalah jumlah elektron suatu atom yang dilepaskan atau diterima atom dalam senyawa, dimana senyawa tersebut terbentuk melalui ikatan ionik.

 

Bilangan oksidasi unsur fosfor, nitrogen, dan belerang pada senyawa tersebut aealah

 

1.    Difosfor pentaoksida (P2O5)

 

Perhitungannya: Total biloks senyawa dengan muatan netral P2O5 adalah 0. Biloks O adalah -2, sehingga:

 

2 biloks P + 5 Biloks O = 0

2 biloks P + 5 (-2) = 0

 

biloks P = ½ (0 – (-10)

              = ½ x 10

              = +5

 

2.    Nitrogen dioksida (NO2)

 

Perhitungannya: Total biloks senyawa dengan muatan netral NO2 adalah 0. Biloks O adalah -2, sehingga:

 

biloks N + 2 Biloks O = 0

biloks N + 2 (-2) = 0

 

biloks N = 0 – (-4)

              = +4

 

3.    Belerang trioksida (SO3)

 

Perhitungannya: Total biloks senyawa dengan muatan netral SO3 adalah 0. Biloks O adalah -2, sehingga:

 

biloks S + 3 Biloks O = 0

biloks S + 3 (-2) = 0

 

biloks S = 0 – (-6)

              = +6
