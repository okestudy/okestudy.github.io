KELEBIHANNYA :

1. Ketersediaan source code

2. Berjalan pada banyak platform

3. Kompatibel dengan Microsoft Office dan format office lainnya.

4. Format OpenDocument yang terbuka dan fleksibel

5. Ekspor ke PDF dan format lain dengan mudah

6. Macro dalam berbagai bahasa pemrograman

7. Akses database yang luas dan transparan

Jawaban:

Untuk memulai OpenOffice.org Calc, ikuti langkah-langkah ini:

Buka OpenOffice.org dari menu Start atau melalui ikon di desktop.Pilih "Calc" dari jendela "Pilih Komponen" atau dari menu "File" > "New" > "Spreadsheet".

Lu akan diarahkan ke halaman baru untuk mulai membuat spreadsheet Anda.

Setelah memulai OpenOffice.org Calc, dapat melakukan hal-hal seperti:

Menambahkan data ke sel-sel spreadsheetMenggunakan rumus dan fungsi untuk menghitung dataMenyimpan dan membuka file spreadsheet dalam format yang berbedaMenyesuaikan tampilan dan format spreadsheet, seperti font, warna, dan layoutMengimpor dan mengekspor data dari dan ke format file yang berbedaMembuat grafik dan diagram untuk memvisualisasikan data secara lebih mudah.
