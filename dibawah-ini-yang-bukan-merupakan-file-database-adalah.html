7. konsep sistem database: pengertian.model.elemen, organisasi,file, sistem dbms,relasi database​


Jawaban:

Konsep sistem database adalah suatu konsep yang menjelaskan bagaimana sebuah sistem database dapat digunakan untuk mengelola, mengakses, dan menyimpan data secara efektif dan efisien.

Pengertian sistem database adalah suatu sistem yang memungkinkan pengguna untuk mengakses dan mengelola data secara efektif dan efisien. Sistem ini biasanya terdiri dari sekumpulan perangkat lunak yang bekerja sama untuk menyimpan, mengelola, dan mengakses data.

Model sistem database adalah suatu representasi dari struktur data yang tersimpan dalam suatu sistem database. Ada beberapa jenis model sistem database, yang paling umum digunakan adalah model relasional, model hierarchi, dan model jaringan.

Elemen-elemen sistem database adalah komponen-komponen yang terdapat dalam sebuah sistem database, seperti data, tabel, atribut, dan relasi.

Organisasi data dalam sistem database adalah cara data disusun dan disimpan dalam suatu sistem database, sehingga mudah untuk diakses dan dikelola. Ada beberapa jenis organisasi data, seperti organisasi data secara sekuensial, organisasi data secara index, dan organisasi data secara pointer.

File dalam sistem database adalah suatu unit terkecil dari data yang disimpan dalam sebuah sistem database. File biasanya terdiri dari sekumpulan record yang saling terkait, yang masing-masing record terdiri dari sekumpulan field.

Sistem DBMS (Database Management System) adalah suatu perangkat lunak yang digunakan untuk mengelola data dalam suatu sistem database. Sistem DBMS menyediakan sejumlah fitur yang memungkinkan pengguna untuk mengakses, mengelola, dan menyimpan data dengan mudah dan efisien.

Relasi database adalah hubungan antara tabel-tabel yang terdapat dalam sebuah sistem database. Relasi ini berguna untuk memastikan integritas data dalam sistem database, sehingga data yang disimpan tidak terdapat duplikasi dan tidak terjadi data yang hilang.

9. Untuk melakukan file koneksi database MySQL diawali dengan penggunaan file​


Apa yang Anda Butuhkan?

Sebelum memulai langkah pada tutorial ini, pastikan bahwa Anda:

Memiliki akses ke kontrol panel hosting

Anda bisa melakukan koneksi database PHP jika memiliki akses ke kontrol panel hosting Anda. Berikut langkah-langkah yang bisa Anda ikuti dengan mudah, terlebih lagi PHP sudah menyediakan fungsi-fungsi untuk  membuat koneksi database php.

Cara Melakukan Koneksi PHP ke MySQL

Buat Database MySQL

Langkah ini diperlukan ketika Anda belum memiliki database MySQL. Pertama-tama, silakan buat database MySQL melalui cPanel dengan mengikuti tutorial berikut ini. Dalam tutorial tersebut database berhasil dibuat menggunakan nama niagahos_namadatabase. Nama niagahos_user sendiri berfungsi sebagai user dari database. Jika ingin menggunakan nama database Anda sendiri, silakan sesuaikan dengan nama database yang Anda inginkan.

PENTING: Setelah Anda berhasil membuat database MySQL, jangan lupa catat detail database seperti nama, username dan password dari database.

Buat File Koneksi PHP untuk Dihubungkan ke Database MySQL

Setelah Anda membuat atau sudah memiliki database MySQL langkah selanjutnya ialah membuat koneksi php. Di sini koneksi ke database akan berjalan dengan nama niagahos_namadatabase dan user niagahos_user.  Selanjutnya yang tidak boleh Anda lewatkan pada cara membuat koneksi php ialah memastikan Anda telah menyimpan password dari user database Anda, pada contoh ini menggunakan password passworddatabase.

Hal lain yang cukup penting, pastikan hostname yang Anda gunakan sudah benar. Kebanyakan hosting menggunakan hostname bernama localhost, sama halnya jika web hosting Anda berada di layanan Niagahoster. Apabila Anda tidak menggunakan web hosting Indonesia di Niagahoster, silakan tanyakan  ke layanan hosting Anda mengenai hostname yang Anda gunakan.

Silakan buat file PHP pada direktori utama website maupun direktori yang Anda inginkan. Dalam contoh ini kami meletakkannya pada direktori public_html dengan nama file koneksidatabase.php. Nantinya Anda dapat mengaksesnya melalui http://namadomain.com/koneksidatabase.php.

Masukkan script koneksi ke dalam file koneksidatabase.php. Berikut ini adalah contoh kode standar yang dapat digunakan untuk melakukan koneksi PHP ke MySQL.

<?php

$servername = "localhost";

$database = "niagahos_namadatabase";

$username = "niagahos_user";

$password = "passworddatabase";

// untuk tulisan bercetak tebal silakan sesuaikan dengan detail database Anda

// membuat koneksi

$conn = mysqli_connect($servername, $username, $password, $database);

// mengecek koneksi

if (!$conn) {

   die("Koneksi gagal: " . mysqli_connect_error());

}

echo "Koneksi berhasil";

mysqli_close($conn);

?>

Metode utama pada script ini menggunakan function mysqli_connect(). Ini adalah fungsi internal PHP dan Anda dapat membaca penjelasannya di sini. Di MySQLi, huruf i  adalah improved yang berarti versi update dari MySQL.

Pada baris kode di awal, Anda akan melihat beberapa deklarasi variabel berikut value-nya. Biasanya diperlukan 4 variabel agar dapat terhubung ke database, yaitu: $servername, $database, $username, dan $password. Di bagian kode, telah ditentukan detail database sebagai value dari variable tersebut. Di dalam function mysqli_connect() Anda dapat melihat variabel tersebut digunakan.

Function mysqli_connect() akan mencoba untuk terhubung ke database sesuai value dari variabel. Jika koneksi tidak berhasil, pesan seperti di bawah ini akan muncul.

if ($conn->connect_error) {

die("Connection failed: " . $conn->connect_error);

}

Function die() dieksekusi pada bagian ini, dimana akan menghentikan script dan memberikan pesan yang telah di-set. Karenanya, secara default akan menampilkan Connection failed: dan pesan error akan membantu untuk menentukan masalahnya.

Jika koneksi berhasil, kode selanjutnya akan dieksekusi.

echo "Connected successfully";

Kode ini akan menampilkan pesan sukses sebagai ganti pesan gagal pada kondisi sebelumnya.

Bagian terakhir adalah kode berikut ini.

mysqli_close($conn);

Kode di atas akan menutup koneksi ke database dengan cara manual. Jika tidak Anda tentukan, koneksi akan tertutup dengan sendirinya setelah script berakhir. Nah, Anda dapat menggunakan baris kode ini untuk menutup koneksi.

Anda dapat menjalankan kode di atas dengan mengakses script melalui domain Anda.

17. ekstensi file untuk program database adalah


Ekstensi file untuk program database adalah .dbf yaitu File database. Selain itu, ekstensi lain untuk files database adalah .pdb (program database file), .wdb (microsoft work database file), .db (file database), .mdb (microsoft Access Database File).

Pembahasan

Database atau disebut juga basis data yaitu kumpulan data yang berhubungan dengan suatu objek, topik atau tujuan khusus tertentu, biasanya berupa kumpulan dari tabel.

Berikut ini jenis-jenis aplikasi untuk database, yaitu:

Dbase, yaitu aplikasi database yang dikembangkan oleh perusahaan Ashton Tate, untuk platform Apple dan Unix kemudian digunakan dibawah DOS. Ekstensi filenya “dbf” masih dipakai untuk Arcview dan Foxpro.IBM DB2, yaitu aplikasi database pertama menggunakan bahasa SQL. Microsoft SQL Server, yaitu aplikasi database buatan Microsoft, yang berguna untuk database skala besar dan terpusat ke server.My SQL, yaitu aplikasi database yang bersifat gratis (freeware). Digunakan untuk aplikasi berbasis web dan dikombinasikan dengan bahasa pemrograman berbasis web, yaitu PHP.Oracle, yaitu aplikasi database client-server yang digunakan untuk penyimpanan dan pengelolaan data yang sangat besar dan banyak.Microsoft Access, yaitu aplikasi database buatan Microsoft, untuk penggunaan skala kecil dengan file yang dihasilkan berbentuk file accdb (access database).

18. Proses menyimpan ulang isi file pada database dan memberikan hasil berupa file dalam format .sql disebut


Menyimpan ulang isi sebuah database dan menghasilkan file dalam format .sql merupakan proses Export atau ekspor database.

Proses export database ini sendiri dapat dilakukan melalui beberapa cara.

Cara pertama melalui command line

Buka command prompt dengan menekan tombol windows + r, atau mengetikan cmdLalu masuk ke directory mysql/binMasuk ke database mysql dengan perintah mysql -u username -p passwordLalu lakukan export dengan perintah mysqldump -u username -p password db_name  > db_name.sql

Cara kedua menggunakan phpMyAdmin sebagai tool administrasi MySQL

Masuk ke phpMyAdmin, pada umumnya dengan mengakses alamat http://localhost/phpmyadminMasuk ke database yang akan di eksporPilih menu ExportSetelah melakukan pengaturan seperlunya pilih Go untuk melakukan ekspor

28. Jelaskan cara membuat file database!


Adapun detail cara membuat database dan tabel baru pada Ms Access bisa di simak penjelasan di bawah ini :

Buka Ms Access sobat Start -> All Programs -> Microsoft Office -> Microsoft Access. Maka akan muncul menu Blank Database sobat ganti nama databasenya pada form File Name, klik tombol browse jika mau menyimpannya /merubah tempat penyimpanan di drive lain “biasanya untuk penyimpanan default di C:\Users\nama id computer\Documents\” dan klik tombol Create

Tunggu beberapa saat hingga muncul tabel baru kosong, disini sobat bisa langsung menambahkan tabel baru dimenu Create -> Table atau bisa juga langsung memperbaiki struktur tabel pertama yang masih kosong. Kemudian pada bagian Table Tools -> View -> Design View. maka akan muncul pesan Save As tabel name dimana sobat harus memberikan nama pada tabel terlebih dulu.

 

Selanjutnya adalah membuat struktur tabel yang terdiri dari beberapa bagian

– Field Name : untuk memasukkan nama field yang akan dibuat

– Data Type : untuk menentukan tipe data, untuk pengertian dan jenis-jenis tipe data sudah saya jelaskan di artikel sebelumnya yang bisa sobat baca disini

– Description : untuk memasukkan keterangan jika di anggap perlu “ndak harus di isi”.

– Field Size : untuk mengatur ukuran field dari masing2 field, saya sarankan untuk memberikan size secukupnya karena nanti berpengaruh pada size tabel yang di buat.

Selain itu sobat juga harus memberikan primary key di masing – masing tabel, primary key di ambil dari salah satu field dalam tabel yang mempunyai sifat unik misal ; NIM, kode_karyawan, dll. untuk membuat primary key pada tabel pilih field yang ingin dijadikan primary key pada menu bar Design, klik tombol menu Primary Key

Jika sudah menyusun struktur tabel klik shorcut Save, selanjutnya adalah mengisi tabel dengan cara memilih menu View -> Datasheet View dan isikan data sesuai field yang sudah sobat buat.

Terakhir klik save, dari tutorial ini berarti sobat sudah bisa membuat database dan tabel baru dengan microsoft access.
