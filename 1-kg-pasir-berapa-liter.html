Gula = x
Minyak = y
2kg gula + 1l minyak = Rp 31.500
(2x + y = 31.500) .... (¡)
1kg gula + 3l minyak = Rp 44.500
(x + 3y = 44.500) .... (¡¡)

Dit:
Harga 2kg gula dan 4liter minyak adalah...

Jawab:
Eliminasi persamaan (¡) dan (¡¡):
2x + y = 31.500 | (×1)
x + 3y = 44.500 | (×2)
----------------------------------

2x + y = 31.500
2x + 6y = 89.000
_______________-
-5y = -57.500
y = 11.500

Substitusi (y=11.500):
2x + y = 31.500
2x + 11.500 = 31.500
2x = 31.500 - 11.500
2x = 20.000
x = 20.000 / 2
x = 10.000

2kg gula + 4l minyak:
= 2x + 4y
= 2(10.000) + 4(11.500)
= 20.000 + 46.000
= 66.000

Maka, harga 2kg gula dan 4liter minyak adalah Rp 66.000
