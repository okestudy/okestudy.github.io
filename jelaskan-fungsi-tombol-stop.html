11. Sebutkan tombol-tombol yang ada pada toolgroup font dan jelaskan fungsinya


Jawaban:

Toolgroup font adalah kumpulan tombol pada program pengolah kata atau aplikasi desain grafis yang berfungsi untuk mengatur teks seperti jenis huruf, ukuran, warna, dan efek. Berikut ini adalah beberapa tombol yang umumnya ada pada toolgroup font dan fungsinya:

1. Tombol jenis huruf (font): Tombol ini digunakan untuk memilih jenis huruf atau font yang akan digunakan dalam dokumen atau desain grafis.

2. Tombol ukuran huruf: Tombol ini digunakan untuk mengatur ukuran huruf dalam dokumen atau desain grafis.

3. Tombol cetak tebal (bold): Tombol ini digunakan untuk membuat teks menjadi lebih tebal atau bold.

4. Tombol miring (italic): Tombol ini digunakan untuk membuat teks menjadi miring atau italic.

5. Tombol garis bawah (underline): Tombol ini digunakan untuk memberi garis bawah pada teks.

6. Tombol garis tengah (strikethrough): Tombol ini digunakan untuk memberi garis tengah pada teks.

7. Tombol warna teks (font color): Tombol ini digunakan untuk mengatur warna teks.

8. Tombol efek bayangan (shadow): Tombol ini digunakan untuk memberi efek bayangan pada teks.

9. Tombol efek luaran (outline): Tombol ini digunakan untuk memberi efek luaran pada teks.

10. Tombol efek bersinar (glow): Tombol ini digunakan untuk memberi efek bersinar pada teks.

11. Tombol efek refleksi (reflection): Tombol ini digunakan untuk memberi efek refleksi pada teks.

Dengan menggunakan tombol-tombol di atas, pengguna dapat mengatur tampilan teks yang lebih menarik dan sesuai dengan kebutuhan dokumen atau desain grafis yang dibuat.

17. Jelaskan fungsi tombol-tombol yang ada pada kotak CPU!


Jawaban:

Umumnya tombol-tombol yg ada pada kotak CPU yaitu:

1. tombol power

2. tombol reset

Penjelasan:

1. Tombol power berfungsi untuk menghidupkan/ menyalakan komputer.

2. Tombol reset berfungsi untuk menghidupkan ulang/ menyalakan ulang komputer.

Bila dikaitkan dengan proses boot, tombol power dipakai untuk melakukan cold boot dan tombol reset dipakai untuk melakukan warm boot.

22. Jelaskan fungsi dari tombol tombol central box


Menu perintah pada control box dapat dilihat di pojok kanan atas program, yaitu:
1. Minimize: menu untuk meminimalkan tampilan program sehingga hanya aktif di taskbar.
2. Maximize: menu untuk memaksimalkan tampilan program sehingga memenuhi layar, serta mengaktifkan menu Restore.
3. Restore: menu untuk menampilkan program secara biasa (tidak sampai memenuhi layar) dan mengaktifkan menu Maximize.

28. Sebutkan dan jelaskan fungsi tombol-tombol pada keyboard yang anda ketahui


1. Fungsi Tombol Esc Tombol Escape untuk membatalkan perintah yang sudah dilakukkan. Jika kita menekan tombol ini maka perintah atau dialog yang kita masukkan akan dibatalkan.
2. Fungsi Tombol Tab Tombol Tab berfungsi jika ditekan akan memindahkan cursor ke arah kanan dengan satu tabulansi.
3. Fungsi Tombol Caps Lock  Jika menekan tombol ini maka lampu indikator caps lock pada keyboard menyala, hal ini menandakan bahwa anda menggunakan mode semua huruf besar atau semua huruf kapital, jika anda menekan lagi dan lampu indikator mati maka dalam mode seperti biasa atau semua huruf kecil.
4. Fungsi Tombol Shift Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).
5. Fungsi Tombol Ctrl Khusus untuk tombol ini fungsinya itu untuk menyisipkan perintah-perintah pintas. Contohnya untuk mencopy  file tinggal tekan ctrl+c.
6. Fungsi Tombol Alt Fungsinya sama seperti tombol Ctrl yaitu untuk menyisipkan perintah secara pintas. Contohnya alt+F4, itu untuk menutup program yang sedang dijalankan.
7. Fungsi Tombol Enter Sebagai perintah "ok" yang diberikan oleh program pada komputer dan untuk membuat paragraf baru.
8. Fungsi Tombol Back Space  Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.
9. Tombol Delete Tombol ini sama seperti tombol backspace, bedanya itu jika tombol delete menghapus satu huruf di sebelah kanan kursor.
