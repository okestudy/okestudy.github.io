2. Tentukan ∆H reaksi berikut jika diketahui energi ikatan ikatannya!


Diketahui data energi ikatan :

C = C = 614 kJ/mol

C - C = 348 kJ/mol

C - H = 413 kJ/mol

C - Cl = 328 kJ/mol

Cl - Cl = 244 kJ/mol

Maka nilai ΔH reaksi dari :

H   H                            H   Cl

Ι     Ι                             Ι     Ι

C = C  +  Cl - Cl ⇒ H - C - C - H

Ι     Ι                             Ι     Ι

H   H                            Cl   H

adalah sebesar -146 kJ/mol.

Penyelesaian Soal :

ΔH reaksi = ΣEI zat pereaksi – ΣEI zat hasil reaksi

                = {(4 × C–H) + C=C + Cl–Cl}  - {(4 × C–H) + C–C + (2 × C–Cl)}

                = {(4 × 413) + 614 + 244}  - {(4 × 413) + 348 +(2 × 328)}

                = {1652 + 614 + 244}  - {1652 + 348 + 656}

                = 2510 kJ/mol  - 2656 kJ/mol

                = -146 kJ/mol

∴ Kesimpulan perubahan entalpi reaksi tersebut adalah -146 kJ/mol.

3. 1 Diketahui energi Ikatan sebagai berikut. ​


Jawaban:

ΔH = -90 kJ

Penjelasan:

Diketahui :
(C—H) = 50 kJ / mol
(C = C) = 30 kJ / mol
(C—C) = 25 kJ / mol
(H—H) = 5 kJ / mol

Pada persamaan :
C2H6 + H2 → C2H6

Sepertinya ada kesalahan soal, jadi saya ijin mengoreksi :
C2H4 + H2 → C2H6

Ditanya :
ΔH = ... ??

Dijawab :
Untuk gambar ikatannya, sudah terlampir pada foto yang saya bagikan

ΔH = ΣΔH(reaktan) - ΣΔH(produk)
ΔH = (4C—H + C = C + H—H) - (6C—H + C—C)
ΔH = (C = C + H—H) - (2C—H + C—C)
ΔH = (30 + 5) - (2(50) + 25)
ΔH = 35 - 125
ΔH = -90 kJ

4. hitung lah energi ikatan N-H dalam NH3 jika diketahui energi ikatan N-H=437,6 kj/mol.energi ikatan N- - - N =949,2 kj/mol dan energi pembentukan NH3=-46,2 kj/mol


Hitunglah energi ikatan rata-rata N - H dalam NH₃ jika diketahui:H - H = 437,6 kJN ≡ N = 949,2 kJΔH°f NH₃ = - 46,2 kJ/mol (entalpi pembentukan standar)Jawaban  Pendahuluan  

Persoalan ini terkait hukum Hess yang melibatkan data energi ikatan rata-rata antaratom. Diminta untuk menghitung energi ikatan rata-rata N-H dalam molekul NH₃.  

Pembahasan  

Energi ikatan adalah energi yang diperlukan untuk memutuskan 1 mol ikatan antaratom dalam keadaan berwujud gas.

Data energi ikatan rata-rata dalam kJ/mol.

N ≡ N = 949,2

H - H = 437,6

Ditanyakan energi ikatan rata-rata N - H.

Perhatikan reaksi pembentukan standar NH₃ berikut ini. Dalam keadaan standar koefisien NH₃ yang dibentuk adalah 1.

[tex]\boxed{\frac{1}{2}N_2_{(g)} + \frac{3}{2}H_2_{(g)}\rightarrow NH_3_{(g)}~~\Delta H^o_f=-46,2~kJ/mol}[/tex]

Agar bilangan koefisien bulat dan memudahkan dalam perhitungan, semua koefisien dan ΔH dikalikan 2.

[tex]\boxed{N_2_{(g)} + 3H_2_{(g)}\rightarrow 2NH_3_{(g)}~~\Delta H^o_f=-92,4~kJ/mol}[/tex]

Dibuat dalam bentuk struktur ikatan antaratomnya seperti yang tampak pada gambar lampiran. Dari skema gambar terdapat 6 ikatan N - H di ruas kanan. Di ruas kiri ada 1 ikatan N ≡ N dan 3 ikatan H - H.  

Langkah berikutnya adalah penerapan hukum Hess untuk energi ikatan. Di sini kita menyatakan simbol (N - H), (N ≡ N), dan (H - H) sebagai energi ikatan rata-rata antaratom.

Ingat, [tex]\boxed{\Delta H=\Sigma energi~pemutusan~(kiri)~-~\Sigma energi~pembentukan~(kanan)}[/tex]

• Pemutusan ikatan ⇒ ruas kiri

• Pembentukan ikatan ⇒ ruas kanan  

Setiap energi ikatan antaratom dikalikan banyaknya ikatan dan koefisien.

ΔH = [(N ≡ N) + 3(H - H)] - [6(N - H)]

-92,4 = [(949,2) + 3(437,6)] - 6(N - H)  

-92,4 = 2.262 - 6(N - H)  

6(N - H) = 2.354,4

[tex](N-H)=\frac{2.354,4}{6} =392,4~kJ/mol[/tex]

Kesimpulan

Berdasarkan langkah pengerjaan di atas, diperoleh energi ikatan rata-rata N - H dalam molekul NH₃ sebesar 392,4 kJ/mol.

13. Diketahui energi ikatan rata-rata: C-C = 80 kkal/mol ​


Jawaban:

Besarnya ΔH reaksi [tex]C_{2}H_{5}[/tex] - C ≡ N + 2[tex]H_{2}[/tex] → [tex]C_{2}H_{5}[/tex] - [tex]CH_{2}[/tex] - [tex]NH_{2}[/tex] adalah -30 kkal/mol. Besarnya ΔH reaksi tersebut dihitung dengan menggunakan data energi ikatan rata-rata.

Penjelasan:

Reaksi : [tex]C_{2}H_{5}[/tex] - C ≡ N + 2[tex]H_{2}[/tex] → [tex]C_{2}H_{5}[/tex] - [tex]CH_{2}[/tex] - [tex]NH_{2}[/tex]

Struktur dari reaksi tersebut adalah:

     H   H                                       H   H    H   H

      Ι     Ι                                         Ι     Ι     Ι     Ι

H - C - C - C ≡ N + 2(H - H) → H - C - C - C - N

      Ι     Ι                                         Ι     Ι     Ι     Ι

     H   H                                        H   H   H   H

ΔH = ∑EI reaktan - ∑EI produk (EI = energi ikatan)

∑EI reaktan = ((2 x EI C-C) + (5 x EI C-H) + (EI C≡N) + (2 x EI H-H))

                   = ((2 x 80) + (5 x 95) + (210) + (2 x 104)

                   = 160 + 475 + 210 + 208

                   = 1053 kkal/mol

∑EI produk =((2 x EI C-C) + (7 x EIC-H) + (EI C-N) + (2 x EIN-H)

                  = ((2x 80) + (7 x 95) + 70 + (2 x 94))

                  = (160 + 665 + 70 + 188)

                  = 1083 kkal/mol

ΔH = ∑EI reaktan - ∑EI produk

      = 1053 kkal/mol - 1083 kkal/mol

      = -30 kkal/mol

28. Diketahui energi ikatan dari beberapa unsur sebagai barikut​


Jawaban:

Diketahui energi ikatan beberapa unsur berikut :

C - H : 413 kJ

C - C : 348 kJ

C - F : 485 kJ

F - F : 155 kJ

H - F : 147 kJ

Jika 116 gram 2-metil propana (Mr : 58) difluorinasi sesuai persamaan:

CH3 CH3

I I

CH3 - CH - CH3 + F2 ⇒ CH3 - CF - CH3 + HF

maka jumlah mol 2-metil-propana adalah:

n = gr/Mr = 116/58 = 2 mol

perubahan entalpi reaksi adalah:

ΔH = ∑(Energi ikatan reaktan) - ∑ (Energi ikatan produk)ΔH = 3(C-C) + 10(C-H) + (F-F) - 3(C-C) - (C-F) - 9(C-H) - (H-F)

ΔH = (C-H) + (F-F) - (C-F) - (H-F)

ΔH = 413 + 155 - 485 - 147

ΔH = -64 kJ

ΔH = -64 kJ/2 mol

ΔH = -32 kJ/mol

Maka jumlah kalor yang dilepaskan adalah 32 kJ/mol.

30. diketahui energi ikatan rata - rata untuk :


Diketahui energi ikatan rata :

C - C : 348 kJ/mol

C = C : 641 kJ/mol

C - H : 413 kJ/mol

C - Cl : 328 kJ/mol

H - Cl : 431 kJ/mol

Maka perubahan entalpi reaksi : CH₂ = CH₂ + HCl ⇒ CH₃ - CH₂ - Cl adalah -17 kJ/mol.

Penyelesaian Soal :

Maka untuk memudahkan dalam menghitung, bukalah rantai rantai pada reaksi menjadi :

H    H                                H  H

Ι      Ι                                  Ι    Ι

C = C   +   H - Cl   ⇒  H - C - C - Cl

Ι      Ι                                  Ι     Ι

H    H                                H   H

ΔH reaksi = ΣEI zat pereaksi – ΣEI zat hasil reaksi

                = {(4 × C–H) + C=C + H-Cl} – {(5 × C–H) + C-C + C-Cl}

                = {(4 × 413) + 641 + 431} – {(5 × 413) + 348 + 328}

                = {1652 + 641 + 431} – {2065 + 348 + 328}

                = 2724 kJ/mol – 2741 kJ/mol

                = -17 kJ/mol

∴ Kesimpulan perubahan entalpi reaksi tersebut adalah -17 kJ/mol.
