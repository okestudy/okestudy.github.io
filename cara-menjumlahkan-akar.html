2. bagaimana cara untuk akar di kalikan atau dijumlahkan,di pangkatkan,di akarkan ?


akar bisa dijumlahkan kalo sama, contoh akar 2 + akar 2 = 2 akar 2
akar dikalikan langsung saja bisa dikerjakan, contoh akar 3 x akar 2 = akar 6
akar dipangkatkan sama denga dikalikan akar itu sendiri contoh akar 2 pangkat3 = akar 2 x akar 2 x akar 2 = akar 8 atau bisa ditulis juga 2 akar 2
akar diakarkan, maka akan masuk dalam akar, akar (akar2) = akar pangkat 4 dari 2

16. Tentukan persamaan kuadrat baru apabila akar-akarnya adalah 5 dan negatif 3 dengan cara rumus jumlah dan hasil kali akar-akar nya​


Menentukan Persamaan Kuadrat Jika Diketahui Akar-Akarnya

Misal, s dan t akar-akar dari suatu persamaan kuadrat, maka persamaan kuadratnya adalah

x² - (s + t)x + (s × t) = 0

==================================

Diketahui akar-akar suatu persamaan kuadrat adalah 5 dan -3.

Jumlah akar-akarnya

5 + (-3) = 2

Hasil kali akar-akarnya

5 × (-3) = -15

Persamaan Kuadrat:

x² - (5 + (-3))x + (5 × (-3)) = 0

x² - 2x + (-15) = 0

x² - 2x - 15 = 0

Semoga membantu.

25. Jumlah kebalikan akar akar persamaa 3x²-9x+4=0 Beserta caranya


Jika persamaan kuadrat [tex]a x^{2} + bx + c = 0[/tex] maka persamaan kuadrat yang kebalikannya adalah akar-akar prsamaan tersbut adalah [tex]c x^{2} + bx + c = 0[/tex]
jadi akar-persamaan yang baru yang akar-akarnya kebalikan dari akar-akar persamaan kuadrat [tex]3 x^{2} - 9x + 4[/tex] adalah :
[tex]4 x^{2} - 9x + 3 =0[/tex] dan jumlah akar akarnya adalah :
- (-9/4)=9/4
Jumlah kebalikan:
1/x₁ + 1/x₂ = (x₁ + x₂)/x₁x₂
= (9/3) / (4/3)
= 3 / (4/3)
= 9/4

29. Buatlah persamaan kuadrat jika diketahui akar-akarnya -⅔ dan -⅓ dengan menggunakan cara faktor dan rumus jumlah dan hasil kali akar-akar nya


(3x + 2)(3x + 1) = 0
jadi persamaanya adalah

9x^2 + 9 x + 2 = 0

cara 1 : cara faktor

akar-akarnya : x1 = -2/3     dan     x2 = -1/3

persamaan kuadratnya adalah:

(x - x1) (x - x2) = 0

(x - (-2/3)) (x - (-1/3)) = 0

(x + 2/3) (x + 1/3) = 0

[tex]x^{2}[/tex] + 1/3 x + 2/3 x + 2/9 = 0

[tex]x^{2}[/tex] + x + 2/9 = 0


Cara 2 : Hasil jumlah (HJ) dan Hasil Kali (HK) akar-akarnya

akar-akarnya adalah -2/3 dan -1/3

HJ = -2/3 + (-1/3)                              HK = (-2/3) (-1/3)

    = -2/3 - 1/3                                         = 2/9

    = -3/3 = -1

Persamaan kuadratnya adalah:

[tex]x^{2}[/tex] - (HJ) x + HK = 0

[tex]x^{2}[/tex] - (-1) x + 2/9 = 0

[tex]x^{2}[/tex] + x + 2/9 = 0


jadi, persamaan kuadratnya adalah [tex]x^{2}[/tex] + x + [tex]\frac{2}{9}[/tex] = 0
