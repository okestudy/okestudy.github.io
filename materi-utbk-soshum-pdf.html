20. UTBK SOSHUM 2019 Nilai x yang memenuhi |x + 1| + 2x <3 adalah


☑ Pertidaksamaan Nilai Mutlak

-

Nilai x yang memenuhi |x + 1|+ 2x < 2 adalah x < ⅔

PEMBAHASAN :

|x + 1| + 2x < 3

|x + 1| < 3 - 2x

Dengan menerapkan sifat :

|x| < r maka -r < x < r

Maka :

x + 1 > -(3 - 2x)

x + 1 > -3 + 2x

x - 2x > -3 - 1

-x > -4

x < 4 { TM }

x + 1 < 3 - 2x

x + 2x < 3 - 1

3x < 2

x < ⅔

Maka nilai x yang memenuhi pertidaksamaan adalah x < ⅔

-AL

23. sbmptn / utbk harus dilakukan semua murid di Indonesia tidak?siapa saja yang harus melakukan sbmptn / utbk?untuk apa sbmptn / utbk?​


Sbmptn/utbk tidak harus dilakukan oleh semua siswa(i) Indonesia. Sbmptn hanya diikuti oleh siswa(i) yg berada di jenjang SMA dan gagal di Snmptn, Snmpn, dan Span-ptkin atau diikuti oleh siswa(i) Indonesia yang berada di jenjang SMA tetapi tidak masuk kuota siswa eligible. Seperti namanya, Sbmptn (Seleksi bersama masuk perguruan tinggi negeri) maka sbmptn adalah tes atau seleksi untuk masuk ke perguruan tinggi negeri.
