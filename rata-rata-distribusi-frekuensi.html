11. tentukan simpangan rata" dari data tabel distribusi frekuensi dibawahdata: 2,4,6,10frekuensi 4,2,3,1​


Answer+Explain

Diketahui:

Data = 2(4), 4(2), 6(3), 10(1)

Ditanya: Rata-rata?

Jawab:

Rata-rata = mean

Mean = jmlh data : banyak data

[tex] = \frac{jmlh \: data}{banyak \: data} \\ = \frac{(2 \times 4) + (4 \times 2) + (6 \times 3) + (10 \times 1)}{4 + 2 + 3 + 1} \\ = \frac{8 + 8 + 18 + 10}{10} \\ = \frac{44}{10} \\ = 4.4[/tex]

Jadi, rata2 nya adalah 4,4

12. Buat data asli atau data fiktip (bukan yang sebenarnya) minimal 50 hasil belaja. Buat tabel distribusi frekuensi , kemudian hitung rata-ratanya


Buat data asli atau data fiktip (bukan yang sebenarnya) minimal 50 hasil belajar. Buat tabel distribusi frekuensi, kemudian hitung rata-ratanya

Berikut 50 data hasil belajar:

11 12 13 14 15 16 17 18 20 21

22 25 28 31 32 33  35 36 39 40

42 43 44 45 46 48 49 50 52 55

57 58 62 64 65 71 72 75 76 78

81 85 86 88 90 92 94 95 96 99

Tabel distribusi frekuensinya adalah:

Nilai      Frekuensi

11-23            11

24-36           7

37-49           9

50-62          6

63-75           5

76-88           6

89-101          6

Nilai rata-rata dari data di atas adalah 50.54

Penjelasan dengan langkah-langkah

Step 1

Tentukan berapa banyak kategori kelas dengan rumus:

K = 1 + 3.3 log (n)

Dimana n adalah banyaknya data -> n = 50

K = 1 + 3.3 log (50)

  = 6.606601 => dibulatkan menjadi 7 kelas

Step 2

Untuk mencari interval diperlukan panjang kelas dengan rumus:

I = jangkauan / jumlah kelas

 = (data tertinggi – data terendah) / jumlah kelas

 = (99-11) / 7

 = 12.57 -> dibulatkan menjadi 13

Jadi pada kolom “Nilai” dibuat range data sebanyak 13 data

Step 3

Buat tabel distribusi frekuensi:

Nilai      Frekuensi

11-23            11

24-36           7

37-49           9

50-62          6

63-75           5

76-88           6

89-101          6

Step 4

Mencari nilai rata-rata:

Rumus rata-rata data adalah:

X = ( n1·x1 + n2·x2 + … + nk·xk ) / ( n1 + n2 + … + nk )

Dimana xi adalah nilai tengah data ke-I yaitu:

Xi = (Ta + Tb) / 2

Dimana Ta adalah batas atas di kelompok tersebut dan Tb adalah batas bawah di kelompok tersebut dan ni adalah jumlah data ke-i.

Jadi nilai tengah dari setiap kelompok:

X1 = (11 + 23) / 2 = 17

X2 = (24 + 36) / 2 = 30

X3 = (37 + 49) / 2 = 43

X4 = (50 + 62) / 2 = 56

X5 = (63 + 75) / 2 = 69

X6 = (76 + 88) / 2 = 82

X7 = (89 + 101) / 2 = 95

Selanjutnya, nilai rata-rata

= (17x11 + 30x7 + 43x9 + 56x6 + 69x5 + 82x6 + 95x6) / (11+7+9+6+5+6+6)

= (187 + 210 + 387 + 336 + 345 + 492 + 570) / 50

= 2527 / 50

= 50.54

Maka nilai rata-rata dari data di atas adalah 50.54

Pembahasan

Distribusi frekuensi merupakan rangkaian data atau nilai yang masing-masing memiliki frekuensinya. Data ini terdiri dari data tunggal dan data kelompok. Untuk data tunggal dapat disajikan dalam bentuk tabel, diagram garis, diagram batang, atau diagram lingkaran. Untuk data kelompok dapat disajikan dalam bentuk histogram, ogif, dan tabel distribusi frekuensi.
