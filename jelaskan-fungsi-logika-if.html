Fungsi OR ✔  

Fungsi OR adalah fungsi bawaan di Excel yang dikategorikan sebagai Fungsi Logika. Fungsi OR adalah fungsi logis untuk menguji beberapa kondisi pada saat bersamaan. Rumus OR akan menghasilkan TRUE jika beberapa argument bernilai benar dan FALSE jika semua argument salah.  

Rumus fungsi OR:  

=OR(logical1; logical2; ...)

Fungsi AND ✔  

AND merupakan rumus fungsi logika pada Excel yang berguna untuk menghasilkan nilai TRUE, jika semua argument yang diuji bernilai benar. Jika FALSE, maka semua atau salah satu argument bernilai salah.  

Rumus fungsi AND:  

=AND (logical1; logical2; ...)

Fungsi NOT ✔  

Kebalikan dari rumus logika pada Excel AND, rumus NOT berguna untuk menghasilkan TRUE jika kondisi yang diuji salah dan FALSE jika yang diuji benar.  

Rumus fungsi NOT:  

=NOT(logical)

13. Bisakah fungsi IF digabungkan dengan fungsi logika (AND, NOT atau OR)? Bila “Bisa” tuliskan contohnya, dan bila “Tidak” jelaskan alasannya!


Bisa jika dalam soal/data ada yg berhubungan dengan "jika", lalu ada "dan" atau "atau". Karena terkadang suatu data diolah dengan cara menentukan logika (IF) yang isinya merupakan gabungan data (AND) atau salah satu dari gabungan data tsb (OR) (Kalau NOT belum pernah dicoba)

Misal:
Nilai Akhir (ada di kolom A):
Sel A2 isinya 60
Sel A3 isinya 80
Nilai Kehadiran (ada di kolom B):
Sel B2 isinya 60
Sel B3 isinya 95
Ditanya: Keterangan Lulus/Tidak Lulus, jika:
- Jika nilai akhir dibawah 75, dan nilai kehadiran dibawah 80, maka Tidak Lulus
- Jika nilai akhir diatas 75, dan nilai kehadiran diatas 80, maka Lulus
Maka rumusnya (gabungan IF dengan AND) (ditulis di sel C2. Patokan nilai akhir A2 & patokan nilai kehadiran B2):
=IF(AND(A2<=75;B2<=80);"TIDAK LULUS";"LULUS")
Maka, hasilnya Tidak Lulus.
Kalau kurang dimengerti, bisa lihat hasilnya di lampiran/gbr

14. Jelaskan kegunaan fungsi logika dari: IF; AND; OR; dan NOT(masing-masing contohnya)


Jawaban:

Semoga membantu

Penjelasan:

Logika IF : memberikan pernyataan suatu kondisi.

Contoh : "=IF(B5>75,"Lulus",Gagal")"

Logika AND : memberikan nilai TRUE jika semua inputannya bernilai benar. Jika salah satu inputannya salah, maka menghasilkan False

Contoh = B5 = 80 maka "=AND(B5>1, B5<85)"

Logika OR : Memberikan nilai TRUE jika salah satu inputannya bernilai benar. Jika kedua inputan salah, maka bernilai False.  

Contoh = B1 = 25 dan B2 = 10 "=OR(B1>10, B2>15)"

Logika NOT : membalikkan suatu pernyataan dengan output TRUE atau FALSE.

Contoh = "=NOT(50 > 70)" Maka akan bernilai TRUE(kebalikan dari False)


15. Bisakah fungsi IF digabungkan dengan fungsi logika (AND, NOT atau OR)? Bila “Bisa” tuliskan contohnya, dan bila “Tidak” jelaskan alasannya!


Bisa, contohnya:
Fungsi IF digabungkan dengan fungsi logika AND
→ =IF(AND(B2>6=6,C2>=7),”Lulus”,”Remidi”)

Fungsi IF digabungkan dengan fungsi logika NOT
→ =IF(NOT(E2<65),”LULUS”,”TIDAK LULUS”)

Fungsi IF digabungkan dengan fungsi logika OR
→ =IF(OR(B2=”D3″,B2=”S1″),”DITERIMA”,”TOLAK”)

17. Jelaskan kegunaan fungsi logika dari: IF; AND; OR; dan NOT(masing-masing contohnya)


Logika IF : memberikan pernyataan suatu kondisi.
Contoh : "=IF(B5>75,"Lulus",Gagal")"

Logika NOT : membalikkan suatu pernyataan dengan output TRUE atau FALSE.
Contoh = "=NOT(50 > 70)" Maka akan bernilai TRUE(kebalikan dari False)

Logika AND : memberikan nilai TRUE jika semua inputannya bernilai benar. Jika salah satu inputannya salah, maka menghasilkan False
Contoh = B5 = 80 maka "=AND(B5>1, B5<85)"

Logika OR : Memberikan nilai TRUE jika salah satu inputannya bernilai benar. Jika kedua inputan salah, maka bernilai False. 
Contoh = B1 = 25 dan B2 = 10 "=OR(B1>10, B2>15)"
