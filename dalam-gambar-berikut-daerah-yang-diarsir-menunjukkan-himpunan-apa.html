10. Daerah daerah yg diarsir pada gambar berikut menunjukkan himpunan penyelesaian dari suatu pertidaksamaan kuadrat dua variabel. Tentukan pertidaksamaan kuadrat dua variabel itu​


b.

Titik puncak = (xp,yp) = (2,-4)

y = a(x - xp)² + yp

y = a(x - 2)² - 4

Melalui (0,0)

0 = a(0 - 2)² - 4

a = 1

Persamaan kuadrat → y = (x - 1)² - 4

Uji salahsatu titik pd daerah arsiran (2,0)

(2 - 1)² - 4 = 1 - 4 = -3

-3 < 0 → -3 < y

Daerah arsiran → (x - 1)² - 4 ≤ y

atau y > x² - 2x - 3

c.

P(1,4)

y = a(x - 1)² + 4

Melalui (0,3)

3 = a(0 - 1)² + 4

a = -1

y = -(x - 1)² + 4

y = -x² + 2x + 3

Uji (0,4)

-0² + 2,0 + 3 = 3

3 < 4 → 3 < y

Daerah arsiran → y ≥ -x² + 2x + 3

d.

(-2,0) dan (1,0)

y = a(x + 2)(x - 1)

Melalui (0,-4)

-4 = a(0 + 2)(0 - 1)

a = 2

y = 2(x + 2)(x - 1)

Uji (2,0)

2(2 + 2)(2 - 1) = 8

8 > 0 → 8 > y

Daerah arsiran → y ≤ 2(x + 2)(x - 1)

atau y ≤ 2x² + 2x - 4

25. Daerah daerah yang diarsir pada gambar berikut menunjukkan himpunan penyelesaian


Penjelasan dengan langkah-langkah:

b.

Titik puncak = (xp,yp) = (2,-4)

y = a(x - xp)² + yp

y = a(x - 2)² - 4

Melalui (0,0)

0 = a(0 - 2)² - 4

a = 1

Persamaan kuadrat → y = (x - 1)² - 4

Uji salahsatu titik pd daerah arsiran (2,0)

(2 - 1)² - 4 = 1 - 4 = -3

-3 < 0 → -3 < y

Daerah arsiran → (x - 1)² - 4 ≤ y

atau y > x² - 2x - 3

c.

P(1,4)

y = a(x - 1)² + 4

Melalui (0,3)

3 = a(0 - 1)² + 4

a = -1

y = -(x - 1)² + 4

y = -x² + 2x + 3

Uji (0,4)

-0² + 2,0 + 3 = 3

3 < 4 → 3 < y

Daerah arsiran → y ≥ -x² + 2x + 3

d.

(-2,0) dan (1,0)

y = a(x + 2)(x - 1)

Melalui (0,-4)

-4 = a(0 + 2)(0 - 1)

a = 2

y = 2(x + 2)(x - 1)

Uji (2,0)

2(2 + 2)(2 - 1) = 8

8 > 0 → 8 > y

Daerah arsiran → y ≤ 2(x + 2)(x - 1)

atau y ≤ 2x² + 2x - 4
