18. sebutkan dan jelaskan tujuh tab ribbon yg terdapat pada program microsoft excel 2007


-Ribbon Home
terdiri dari:
Clipboard: Copy, Paste, Cut, dan Format
Font: Jenis font, ukuran font, warna huruf, Bold, Italic, Underline, warna background cell, dan Border Cell
Alignment: Top alignment, Middle alignment, Bottom alignment, Align text left, Align text center, Align text right, Decrease indent, Increase indent, Warp text dan Merge & Center
Number: Combo box format cell, Accounting number format, Percent style, Comma style, Increase decimal, dan Decrease decimal
Styles: Conditional formatting, Format as table, dan Cell styles untk mempercantik tampilan workshet yg kita bikin
Cell: Insert, Delete dan Format untuk melakukan operasi insert cell, selete cell dan format cell
Editing: Sum, Sort, Find & Select, Fill

-Ribbon Insert
terdiri dari:
Table: Pivot Table, Table
Illustrations: Picture, Clip Art, Shapes, SmartArt
Charts: Column, Line, Pie, Bar, Area, Scatter, Other Chart
Link: Hyperlink untuk menginsert hyperlink (menu bar Insert -> Hyperlink pada Excel 2002)
Text: Text box, Header & Footer, Wordart, Sugnature Line, Object dan Symbol

-Ribbon Page Layout
terdiri dari:
Themes: Themes, Colors, Fonts dan Effects
Page Setup: Margin, Orientation, Size, Print Area, Breaks, Background, Print Tittles
Scale to Fit: Width, Height, Scale
Sheet Options: Gridlies, Heading
Arrange: Bring to Front, Send to Back, Selection Pane, Align, Group dan Rotate

-Ribbon Formulas
terdiri dari:
Function Library: Insert Function, Auto Sum, Recently Used, Financial, Logical, Text, Date & Time, Lookup and References, Math and Trig, dan More Functions
Defined Names: Name Manager, Define Name, Use in Formula, Create from Selection
Formula Auditing: Trace Precedents, Trace Dependents, Remove Arrows, Show Fromula, Error Checking, dan Evaluate Formula
Calculation: Calculation Options

-Ribbon Data
terdiri dari:
Get External Data: From Access, From Web, From Text, dan From Other Sources, Existing Connections.
Connections: Refresh All, Connections, Properties, dan Edit Links.
Sort Filter: Sort, Filter
Data Tools: Text to Columns, Remove Duplicates, Data Validation, Consolidate, dan What If AnalysisOutline: Group, Ungroup dan Subtotal

-Ribbon Review
terdiri dari:
Proofing: Spelling, Research, Thesaurus, Translate
Comments: New Comment, Delete: Previous, Next, Show/Hide Comment, Show All Comments dan Show Link
Changes: Protect Sheet, Protect Workbook, Share Workbook, Protect and Share Workbook, Allow Users to Edit Ranges dan Track Changes

-Ribbon View
terdiri dari:
Workbook view: Normal, Page Layout, Page Brake Preview, Custom View dan Full Screen
Show/Hide: Ruler, Gridlines, Formula Bar, Headings, dan Message Bar
Zoom: Zoon, 100%, dan Zoom to Selection
Window: New Window, Arrange All, Freeze Panes, Slip, Hide, dan Unhide, Save Workspace, Switch Windows
Macros: View Macros, Record Macro

Microsoft Excel merupakan aplikasi untuk mengolah data secara otomatis yang dapat berupa perhitungan dasar, rumus, pemakaian fungsi-fungsi, pengolahan data dan tabel, pembuatan grafik dan menajemen data.

29. Sebutkan nama menu menu tab ribbon yang terdapat di program Microsoft Excel 2007


Pada Microsoft Excel 2007, menu tersusun dalam tab-tab menu yang ditampilkan dalam bentuk ikon-ikon yang dikelompokkan ke dalam ribbon.

Ribbon terbagi menjadi beberapa tab. Pada tiap-tiap tab, ikon dibagi menjadi beberapa kelompok menurut kriteria tertentu. Pada beberapa kelompok, di sudut kiri bawah terdapat ikon yang berfungsi menampilkan kriteria lebih lanjut. Ribbon juga menampilkan tab secara default, yaitu tab home, insert, page layout, formulas, data, re-view, dan view. Jika terdapat add ins, maka aplikasi juga akan menampilkan tab add ins. Selain tab-tab tersebut, juga terdapat tab yang lain yang lebih umum berkaitan dengan objek yang disisipkanTab Home, Tab Insert, Tab Page Layout, Tab Data, Tab Formula, Tab Review dan Tab View
