29. Apabila anda ingin menetikkan huruf lain, maka harus menekan kombinasi beberapa tombol secara bersamaan, merupakan karakteristik dari jenis keyboard jenis….. *.​


Keyboard Chord

Penjelasan:

Apabila anda ingin mengetikkan huruf lain,maka harus menekan kombinasi beberapa tombol secara bersamaan, merupakan karakteristik dari jenis keyboard?

Keyboard adalah alat untuk mengetikan huruf, angka, dan simbol ke dalam komputer. Keyboard yang sering kita gunakan adalah jenis keyboard QWERTY. Pada jenis keyboard ini semua jenis huruf, angka, dan simbol memiliki tombol masing-masing.

Namun jaman dulu ada jenis keyboard bernama Chord. Pada keyboard ini hanya bisa kita temukan 4-5 tombol saja. Oleh karena itu pada keyboard ini membutuhkan kombinasi penekanan tombol untuk mengetikan huruf. Keyboard Chord tidak populer di masanya, karena menyebabkan tangan kelelahan ketika harus menekan tombol bersamaan saat mengetik.
