pp/q x ar/s = a(p/q + r/s)

Sifat Pembagian Bilangan Berpangkat Pecahan

Pada operasi pembagian bilangan berpangkat pecahan, berlaku sifat-sifat sebagai berikut, yaitu:

ap/q : ar/s = a(p/q – r/s)

(ap/q)r = a(p/q) x r = a(pr/q) atau

* sifat bilangan pecahan perkalian
1. sifat tertutup ~ contohnya : (2/5) x (8/11) = 16/55
2. sifat komutatif ~ contohnya : p x q = q x p
3. sifat asosiatif atau pengelompokkan ~ contohnya : 2/5 × (–2/7 × 3/5) = [2/5 ×(–2/7)] × 3/5 = –12/175

* sifat bilangan pecahan perkalian dan penjumlahan
1. sifat distributif ~ contohnya : 6/3 x [5/3 + (–4/3)] = 6/3 × 1/3 = 6/9 
    (6/3 × 5/3) + [6/3 × (–4/3)] = 30/9 – 24/9 = 6/9
    Jadi, 6/3 × [5/3 + (–4/3)] = (6/3 × 5/3) + [6/3 × (–4/3)] = 2/9

* sifat bilangan pecahan perkalian dan pengurangan
1. sifat distributif ~ contohnya : 4/5 × [6/5 – (–5/5)] = 4/5 × 11/5 = 44/25
    (4/5 × 6/5) – [4/5 × (–5/5)] = 24/25 – (–20/25) = 44/25
    Jadi, 4/5 × [6/5 – (–5/5)] = (4/5 × 6/5) – [4/5 × (–5/5)] = 44/25

22. sebutkan bentuk dan sifat sifat pertidaksamaan pecahan ! tolong bantuannya


Sifat pertidaksamaan pecahan.
Dapat dikatakan mirip dengan pertidaksamaan polinomial pada umumnya:
Misal, ketika:
[tex]\displaystyle \frac{f(x)}{g(x)}\leq 0[/tex]
Akan memiliki daerah penyelesaian yang sama dengan
[tex]f(x)g(x)\leq 0[/tex]
Hanya saja untuk kasus pecahan,
[tex]g(x)\neq0[/tex] karena akan menyebabkan penyebut bernilai 0.

Bentuknya bervariasi,
Ada faktor normal,
 Misal:
[tex]\displaystyle \frac{x-1}{x-2}\leq 0[/tex]
Akan memiliki daerah yang sama dengan:
[tex](x-1)(x-2)\leq0 \\ 1\leq x\leq 2[/tex]
Dengan catatan: Penyebut tidak boleh sama dengan 0
Untuk kasus ini,
[tex]x\neq 2[/tex]
Sehingga, HP tersebut adalah [tex]1\leq x<2[/tex]

Faktor kuadratis
Umumnya dapat dilakukan dengan pemfaktoran:
Bentuk 1, Jika D>0, dapat difaktorkan biasa.
Bentuk 2, Jika D=0, merupakan pembuat 0 dan definit
Bentuk 3, Jika D<0, merupakan definit
Misalnya diberikan:
[tex]x^2(x-3)<0[/tex]
Maka, himpunan penyelesaiannya adalah:
x < 3, dan [tex]x\neq 0[/tex] karena 0 tidak memenuhi persamaan.
Jika definit:
[tex]\displaystyle \frac{x^2+3x+1}{x^2-4}\leq 0 \\\\ \frac{\oplus}{(x+2)(x-2)}\leq 0[/tex]
-2 < x < 2 karena [tex]x\neq \{-2,2\}[/tex]

Untuk pertidaksamaan, Tidak berlaku perkalian silang karena akan menimbulkan Mathematical Fallacy
Misal (Yang benar, tanpa perkalian silang.):
[tex]\displaystyle \frac{x+1}{x-2}\geq 1 \\\\ \frac{x+1}{x-2}-1\geq0 \\\\ \frac{x+1-(x-2)}{x-2}\ge 0 \\\\ \frac{3}{x-2}\ge0[/tex]
HP = {x | x > 2}

Penjumlahan Pecahan
Pada dasarnya, dua bilangan pecahan dapat dijumlahkan jika penyebutnya sama. Jika penyebutnya tidak sama, maka harus disamakan terlebih dahulu. Oleh karena itu, sebelum menjumlahkan pecahan kita harus memperhatikan apakah penyebut pecahannya sama atau tidak.
1. Penyebutnya Sama
Jika penyebut pecahannya sama, maka hanya pembilangnya yang dijumlahkan sedangkan penyebutnya tetap. Untuk lebih jelasnya, perhatikan contoh.
a + b = a + b
c c c

5). 5 + 2 = 5 + 2 = 7 = 1⅙ 6 6 6 6
2. Penyebutnya Tidak Sama
Jika penyebut pecahannya tidak sama, maka penyebutnya harus disamakan terlebih dahulu. Untuk menyemakan penyebut kita bisa mengalikan penyebutnya atau dengan cara menentukan KPKnya.

Pengurangan Pecahan
Sama seperti penjumlahan, dua bilangan pecahan dapat dikurangkan jika penyebutnya sama. Jika penyebutnya tidak sama, maka harus disamakan terlebih dahulu. Oleh karena itu, sebelum mengurangkan pecahan kita harus memperhatikan apakah penyebut pecahannya sama atau tidak.
1. Penyebutnya Sama
Jika penyebut pecahannya sama, maka hanya pembilangnya yang dikurangkan sedangkan penyebutnya tetap. Untuk lebih jelasnya, perhatikan contoh.
a − b = a − b
c c c

2. Penyebutnya Tidak Sama

c d c x d

Perkalian Pecahan
Perkalian pecahan dapat dilakukan secara langsung tanpa harus melihat apakah penyebutnya sama atau tidak. Dengan kata lain ita tidak harus menyamakan penyebutnya.
a x c = a x c
b d b x d

Sifat Perkalian Pecahan
1. Komutatif
a/b x c/d = c/d x a/b
2. Asosiatif
(a/b x c/d) x e/f = a/b x (c/d x e/f)
3. Distributif
a/b x (c/d ± e/f) = (a x c)/(b x d) ± (a x e)/(b x f)
