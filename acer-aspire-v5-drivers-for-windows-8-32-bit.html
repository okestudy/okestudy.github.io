13. Apa bedanya Windows 32 bit dengan Windows 64 bit? Dan apa kelebihan dan kekurangan keduanya?


Perbedaan arsitektur prosesor 32-bit dan 64-bit sangat identik pada pemrosesan data pada CPU. Jika pada laptop/komputer yang mempunyai arsitektur 32-bit, maka pada saat data diproses dalam sekali siklus pemrosesan hanya sebesar 32-bit. Sedangkan yang menggunakan arsitektur prosesor 64-bit, maka pada saat memproses data yang dilakukan dalam sekali siklus pemrosesan akan membawa jumlah data sebesar 64-bit. Jadi 64-bit dapat memuat lebih banyak perintah premrosesan daripada 32-bit.

14. maksud dari 64 bit/32 bit di windows itu apa?


Windows 32-bit dan 64-bit pada dasarnya mengacu pada teknologi pemrosesan data pada processor tentang bagaimana menangani informasi. Processor 64 bit akan mampu mereferensikan pengalamatan data yang lebih banyak apabila dibandingkan dengan processor 32 bit yang akan berefek pada peningkatan performance dari kinerja keseluruhan perangkat komputer.

22. kelebiha dan kekurangan file so pada windows dan linux yang 32 bit dan 64 bit


Pada dasarnya perbedaan system 32 bit dengan 64 bit terletak pada processor-nya, jika processor 64 bit keluaran Intel diberi kode EM64T (Intel Dual Core atau Core2Duo) dan keluaran AMD(Advanced Micro Devices ) diberi kode 64 diakhir seri (Athlon 64), kalo komputer sobat generasi Pentium 3 kebawah sudah pasti systemnya 32bit. Untuk OS mulai XP sampai sekarang sudah ada 32 bit dan 64 bit, tetapi dari XP kebawah seperti winme, win98 dst masih menggunakan system 32 bit.

Pada teorinya system 64 bit itu lebih cepat dalam memproses suatu data dibandingkan dengan system 32 bit. Bahkan dalam menangani memory RAM pun 32 bit hanya mampu 4Gb tapi pada komputer kadang terbaca hanya 3,25Gb saja sedangkan untuk system 64 bit bisa lebih dari 4Gb.Untuk saat ini lebih banyak software atau perangkat lunak yang support system 32 bit sedangkan untuk 64 bit masih jarang, tetapi banyak pula software 32 bit bisa di install di 64 bit, kecuali software driver dan anti virus harus sesuai dengan versi OS yang diinstal di komputer atau laptop.

Setelah mengetahui Kelebihan, Kekurangan, Perbedaan System 32 bit dan 64 Bit kita bisa menentukan ingin menggunakan system 32 Bit atau 64 Bit dengan catatan hardware atau perangkat keras sudah support 64 Bit. 32-bit, dalam arsitektur komputer adalah sebuah kata sifat yang digunakan untuk menjelaskan bahwa terdapat sebuah bilangan bulat (integer) yang memiliki panjang 32 bit. Istilah ini juga merujuk kepada istilah yang menjelaskan arsitektur mikroprosesor yang dibuat berdasarkan register prosesornya, bus alamatnya, atau bus data yang digunakannya berukuran 32 bit. Nilai bilangan integer yang dapat disimpan dalam 32 bit adalah berkisar antara 0 hingga 4294967295 (unsigned integer), atau dari -2147483648 hingga 2147483647 (signed integer). Prosesor yang menggunakan bus alamat memori 32-bit, mampu mengalamatkan memory secara langsung hingga 4 Gigabyte.
