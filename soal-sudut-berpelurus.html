Contoh Soal Sudut Berpelurus dan Jawaban Lengkap

Tentukan nilai x, pelurus ∠BCD dan besar ∠BCD apabila besar ∠ACD = 2x° dan ∠BCD = (5x + 5)º?

Pembahasan.

Besar sudut berpelurus biasanya mencapai 180º. Dengan besar sudut ini, kita dapat menyelesaikan contoh soal sudut saling berpelurus dengan cara berikut:

∠ACD + ∠BCD = 180º

2x° + (5x + 5)º = 180º

7x° + 5° = 180º

7x° = 175°

x° = 25°

Besar ∠BCD = (5x + 5)º

= (5 . 25 + 5)º

= (125 + 5)º

= 130º

Pelurus ∠BCD = ∠ACD

= 2x°

= 2 . 25°

= 50°

Contoh Soal Sudut Berpelurus dan Jawaban Lengkap

Tentukan besar nilai xº, yº dan zº?

Pembahasan.

Contoh soal sudut berpelurus ini dapat diselesaikan dengan metode berikut ini:

70° + 11x° = 180° (berpelurus)

11x° = 180° – 70°

11x° = 110°

x° = 10°

Setelah nilai x ditemukan, lalu menentukan nilai y dengan cara di bawah ini:

60° + 2x° + 4y° = 180° (berpelurus)

60° + 2.10° + 4y° = 180°

60° + 20° + 4y° = 180°

80° + 4y° = 180°

4y° = 180° – 80°

4y° = 100°

y° = 25°

Kemudian mencari nilai z dengan langkah langkah yaitu:

60° + 11x° + 2z° = 180° (jumlah sudut pada segitiga)

60° + 11.10° + 2z° = 180°

60° + 110° + 2z° = 180°

170° + 2z° = 180°

2z° = 180° – 170°

2z° = 10°

z° = 5°

Jadi nilai x, y, dan z ialah 10°, 25° dan 5°.

Contoh Soal Sudut Berpelurus dan Jawaban Lengkap

Tentukan besar nilai nilai x, besar ∠PQT, besar ∠RQS, pelurus ∠PQT, pelurus ∠SQT, dan pelurus ∠RQS?

Pembahasan.

Contoh soal sudut berpelurus ini dapat dihitung menggunakan cara seperti berikut:

∠PQT + ∠SQT + ∠RQS = 180º (sudut berpelurus)

4xº + 40º + (2x +20)º = 180º

6xº + 60º = 180º

6xº = 180º – 60º

6xº = 120º

xº = 20º

∠PQT = 4xº

= 4 . 20º

= 80º

∠RQS = (2x + 20)º

= (2 . 20 + 20)º

= (40 + 20)º

= 60º

Pelurus ∠PQT = 180º – ∠PQT

= 180º – 80º

= 100º

Pelurus ∠SQT = 180º – ∠SQT

= 180º – 40º

= 140º

Pelurus ∠RQS = 180º – ∠RQS

= 180º – 60º

= 120º

Contoh Soal Sudut Berpelurus Beserta Pembahasannya

Pada gambar di atas terdapat ∠ACD = 2x° dan ∠BCD = (3x + 30)º. Hitunglah nilai x, besar ∠BCD dan pelurus ∠BCD?

Pembahasan.

Sudut saling berpelurus biasanya memiliki besar sudut 180º. Contoh soal sudut saling berpelurus tersebut dapat diselesaikan menggunakan cara di bawah ini:

∠ACD + ∠BCD = 180º

2x° + (3x + 30)º = 180º

5x° + 30° = 180º

5x° = 150°

x° = 30°

∠BCD = (3x + 30)º

= (3.30 + 30)º

= (90 + 30)º

= 120º

Pelurus ∠BCD = ∠ACD

= 2x°

= 2.30°

= 60°

Hitunglah nilai xº, yº dan zº?

6z° = 42°
