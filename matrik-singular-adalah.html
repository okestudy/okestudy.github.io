2. Apa yang di maksud dengan matrik singular


Jawaban:

Matrik singular adalah matrik yang memiliki determinan (nilai absolut) yang sama dengan nol. Ini berarti bahwa matriksnya tidak dapat diputar atau diinvers.

Penjelasan dengan langkah-langkah:

Sebagai contoh, matriks berikut adalah matriks singular:

[1 0 0]

[0 0 0]

[0 0 0]

Karena determinan dari matriks di atas adalah 0, itu adalah matriks singular. Matriks ini tidak memiliki solusi dan tidak dapat diinvers. Jadi, tidak ada cara untuk menyelesaikan persamaan linear yang terkait dengan matriks ini.

5. matrik A adalah matrik 2x2 matrik B adalah matrik yang di peroleh dengan menukarkan baris pertama dengan baris kedua pada matrik A. apa hubungan antara det A dan det B. tolong jelaskan !!!


Kelas : XII (3 SMA)
Materi : Matriks
Kata Kunci : matriks, ordo, determinan

Pembahasan :
Matriks adalah jajaran bilangan berbentuk persegi panjang atau persegi yang tersusun dalam baris dan kolom yang terletak di dalam kurung.

Bilangan di dalam kurung dinamakan elemen dari matriks.
Elemen horisontal dinamakan baris dan elemen vertikal dinamakan kolom.

Suatu matriks diberi lambang dengan huruf kapital.
Contoh : A, B, dan lain-lain.
Sedangkan elemen matriks diberi lambang dengan huruf kecil berindeks.
Contoh : a₁₂, b₂₁, dan lain-lain.

Ordo suatu matriks adalah bilangan asli yang menyatakan banyaknya baris dan kolom dari matriks tersebut. Lambang dari ordo matriks adalah A(m x n).
Contoh : A(2 x 3) artinya matriks berordo 2 x 3, A(3 x 3) artinya matriks berordo 3, dan lain-lain.

Determinan suatu matriks adalah selisih hasil kali elemen-elemen pada diagonal utama dengan hasil kali elemen-elemen pada diagonal samping.
Contoh lambang dari determinan matirks A adalah |A| atau det A.

Mari kita lihat soal tersebut.
Matriks A merupakan matriks 2 x 2 atau berordo 2. Matriks B merupakan matriks yang diperoleh dengan menukarkan baris pertama dengan baris kedua pada matriks A. Apakah hubungan determinan A dengan B?

Jawab :
Misalkan matriks A adalah
[tex]\left|\begin{array}{cc}a&b\\c&d\end{array}\right][/tex].

det A
[tex]=\left|\begin{array}{cc}a&b\\c&d\end{array}\right|[/tex]
⇔ det A = ad - bc

Matriks B diperoleh dengan menukarkan baris pertama dengan baris kedua, diperoleh
[tex]\left[{\begin{array}{cc}c&d\\a&b\end{array}\right] [/tex]

det B
[tex]= \left|\begin{array}{cc}c&d\\a&b\end{array}\right| [/tex]
⇔ det B = bc - ad

Kedua persamaan kita tulis, diperoleh
ad - bc = -(bc - ad)
det A = -det B.

Jadi, hubungan antara determinan A dengan determinan B adalah det A lawan dari det B.

17. Tentukan nilai dari x agar matrik dibawah ini menjadi matriks singular x x+1-4 x+1


Jawaban:

Matriks singular = determinan 0

determinan = x(x+1) - (x+1)(-4) = 0

x^2 +x+4x+4 = 0

x^2 +5x+4 = 0

(x+4)(x+1) = 0

x = -4 atau x = -1

matriks

-

Diberikan matriks,

A = [ x x + 1 ; -4 x + 1]

Agar A adalah matriks singular ( tidak memiliki invers ) , maka :

0 = | A |

0 = x (x + 1) - (x + 1) (-4)

0 = x² + x + 4x + 4

0 = x² + 5x + 4

0 = (x + 1) (x + 4)
